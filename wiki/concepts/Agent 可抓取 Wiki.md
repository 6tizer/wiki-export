## 定义

Agent 可抓取 Wiki 是一种面向 AI Agent 读取和遍历而设计的知识组织形式，通常以结构稳定的 Markdown 页面、索引页和清晰的链接关系为基础，让 Agent 能直接检索、追踪和组合知识。

## 关键要点

- 它强调知识页面要对机器友好，而不仅仅是对人类读者友好。

- 典型产物包括 index 页面、概念页、摘要页以及可持续维护的链接结构。

- 在本文里，graphify 的 `--wiki` 模式就是将原始资料目录编译为可供 Agent 抓取的 Wiki 形态。
