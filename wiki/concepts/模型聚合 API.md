## 定义

模型聚合 API 是一种把多个模型供应商的调用方式、鉴权、计费与路由能力封装到统一接口中的接入层，方便开发者通过单一端点调用不同模型。

## 核心要点

- 把原本分散在不同厂商的 API 接口抽象成统一调用面。

- 降低多模型接入、切换和运维成本，适合需要灵活选型的应用场景。

- 常与统一计费、监控、限流和兼容协议能力一起出现。
