## 定义

Agent 网络是由多个代表个人或团队的 Agent 构成的协作网络，负责在组织内部自动完成信息发现、同步、协调与分发。

## 关键要点

- 每个 Agent 代表一个人的上下文、偏好与工作边界

- 信息流从点对点的人际沟通转为 Agent 之间的结构化交换

- 适合处理变更通知、依赖识别、知识同步与跨团队协作

- 网络价值来自默认开放的知识库与可执行的项目上下文
