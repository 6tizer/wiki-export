Agent Swarms（智能体集群）是指多个自主 AI Agent 协同工作的去中心化 Agent 网络，是 ai16z/elizaOS 生态的核心愿景，也是 Agent 编排框架的进化方向。

## 核心概念

Agent Swarms 不同于单个 Agent 的自主执行，其核心在于：

- **多 Agent 协作**：每个 Agent 专注特定领域，共同完成超出单个 Agent 能力的复杂任务

- **去中心化执行**：无需中心化协调者，Agent 间通过协议通信和协作

- **动态组合**：根据任务需求动态组合不同能力的 Agent

## ai16z 的 Agent Swarms 愿景

ai16z DAO 将 Agent Swarms 定位为下一代加密 AI 的突破口：

- 通过代币经济激励（手续费回购 $ai16z）驱动生态增长

- Agent 间形成去中心化的"自主代理经济"

- 目标：将 $ai16z 打造成 Agent 经济的基础结算货币

## 技术实现（Swarms 框架）

kyegomez 的 Swarms 框架提供了具体的 Swarms 编排模式：

- SequentialWorkflow（顺序执行）

- ConcurrentWorkflow（并发执行）

- MixtureOfAgents（专家混合路由）

- AutoSwarm（自动任务分解）
