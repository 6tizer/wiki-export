## 定义

Memory Sparse Attention 是 EverMind 提出的长记忆注意力架构，把记忆访问内化到模型注意力机制中，而不是依赖外挂检索。

## 关键要点

- 通过稀疏注意力降低超长上下文计算成本

- 目标是在长记忆场景中兼顾精度与效率

- 适合多文档、多跳推理与 Agent 长期记忆任务
