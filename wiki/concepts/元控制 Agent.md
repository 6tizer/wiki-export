## 定义

元控制 Agent 是多 Agent 系统中的上层调度者，负责监控整体效果、收集反馈并动态调整其他 Agent 的参数与策略。

## 关键要点

- 关注系统级优化，而非单个任务执行

- 常承担质量监控与策略回路职责

- 是多 Agent 自适应运行的关键角色
