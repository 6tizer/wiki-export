## 定义

角色分工式 Agent 工作流是把规划、编码、审查、测试和发布等任务拆给不同 Agent 角色处理的工程组织方式。

## 核心要点

- 通过角色边界减少单个 Agent 的认知混杂

- 适合复杂软件工程和多阶段审核流程

- 常以 slash commands、子 Agent 或固定工序来实现
