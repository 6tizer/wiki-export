## 定义

Markdown 转公众号工作流，是指先以 Markdown 编写正文，再经过预览、格式转换、图片处理与草稿投递，最终生成适合微信公众号发布内容的一整套写作与分发流程。

## 关键要点

- 内容源保持为 Markdown，便于版本管理、复用与 AI 协作

- 工作流通常包含 inspect、preview、convert、upload、draft 等环节

- 可以在 API 模式与 AI 模式之间切换，满足“速度优先”与“样式优先”两类需求

- 本质上是把写作工具链产品化，而不是只做一次性排版脚本

## 典型步骤

1. 用 Markdown 编写文章

1. 检查标题、摘要、作者等 metadata

1. 预览转换结果或生成 AI request

1. 上传图片并推送到微信草稿箱
