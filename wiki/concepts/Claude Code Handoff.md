## 定义

Claude Code Handoff 是一种为跨 Session 接力而设计的交接机制，通过结构化交接文档把当前任务状态、已做修改和下一步建议传递给新的会话。

## 关键要点

- 缓解每个新 Session 都从零开始的上下文断裂问题

- 适合长任务、多终端和多人协作场景

- 可通过 Skill 或规范模板把交接格式标准化
