## 定义

Agent NPC 是指在游戏或互动世界中由 AI Agent 驱动的非玩家角色，用来提供行为、反馈、对抗或陪伴，而不是替代玩家本身。

## 关键要点

- 它的价值在于增强世界活性与互动张力，而不是把游戏自动化到失去参与感。

- 如果 Agent 直接代替玩家执行核心动作，往往会削弱情绪回报和成就感。

- 优秀的 Agent NPC 需要有足够丰富、不可预测的行为模式，才能接近真人互动体验。
