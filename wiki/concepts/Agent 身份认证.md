## 定义

Agent 身份认证是指为智能体建立可验证的身份标识，并把该身份与开发者、拥有者或运营主体绑定起来，使外部系统能够确认“这个 Agent 是谁、代表谁、被谁授权”。

## 关键要点

- 它解决的不是能力问题，而是交易前的信任问题。

- 在企业场景中，身份认证通常要和权限控制、审计记录、责任归属一起工作。

- 对 A2A 场景而言，身份认证是 Agent 之间建立互信与发起高价值操作的前提。

- 文章中讨论的重点，是把人类社会已有的认证逻辑迁移到 Agent 世界。
