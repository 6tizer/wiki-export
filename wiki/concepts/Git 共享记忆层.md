## 定义

Git 共享记忆层指把 Agent 在执行任务中沉淀出的有效操作经验、修复办法和脚本改动提交到 Git，让后续 Agent 可以直接复用这些经验，而不必从零重新摸索。

## 关键要点

- 它把一次性的运行经验转化为可继承的长期记忆。

- 共享对象不只是结果，还包括对复杂界面和异常情况的处理方式。

- 这种机制适合多 Agent 或多轮执行场景，能降低重复试错成本。
