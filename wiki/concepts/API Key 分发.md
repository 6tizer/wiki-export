## 定义

API Key 分发是指由统一中间层生成、管理并下发调用密钥，把上游模型服务的接入凭证与下游应用的使用凭证解耦。

## 关键要点

- 统一管理不同模型服务的接入方式与密钥生命周期。

- 让多人、多应用共享同一套分发与权限控制策略。

- 常与账号管理、计费、并发控制和审计能力一起出现。
