## 定义

[SOUL.md](http://soul.md/) 是 OpenClaw 的**核心个性化配置文件**，用户通过它定义 Agent 的个人偏好、背景信息、行为规范等，是 OpenClaw 九层 System Prompt 架构中用户层的核心载体。

## 作用

- 存储用户身份、偏好、工作习惯等信息

- 让 OpenClaw Agent 产生个性化响应

- 配置 Agent 的目标、约束和工作方式
