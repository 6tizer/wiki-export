## 定义

Git 组织协议指的是把 PR、Review、Merge、Diff 等 Git 机制从代码协作扩展为组织协作的基础协议，用统一的状态与历史记录承载信息流动。

## 关键要点

- 每次协作都以可审查、可追溯、可回滚的变更存在

- 不再依赖某个人记得通知谁或同步谁

- 适合连接项目库、知识库与 Agent 的自动化执行流程

- 本质上是把组织协作从广播式沟通改造成结构化变更流
