## 定义

[USER.md](http://user.md/) 是 Hermes Agent 文档体系中的用户画像文件，用来保存用户的角色、偏好、沟通风格与长期协作约束，帮助 Agent 在跨任务场景中持续保持“为谁工作”的稳定认知。

## 关键要点

- 它回答的是“当前 Agent 服务的用户是谁”，与描述 Agent 自身身份的 [SOUL.md](concepts/SOUL.md.md) 分工不同

- 适合承载稳定的用户背景与沟通偏好，不适合塞入会快速过时的任务细节

- 与 [MEMORY.md](concepts/MEMORY.md.md) 分层后，能把静态画像和动态经验分开管理，减少记忆污染

- 在 Hermes 的文件体系里，[USER.md](http://user.md/) 常与 [SKILL.md](concepts/SKILL.md.md)、[渐进式披露](concepts/渐进式披露.md) 一起使用，支撑更稳的长期协作体验
