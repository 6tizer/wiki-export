## 定义

OpenAI 兼容 API 代理 是一种把非 OpenAI 模型或账号能力封装成 OpenAI 接口协议的适配层，方便现有 AI 工具直接接入。

## 核心要点

- 通过统一 `/v1/*` 接口降低工具接入成本

- 常见于模型中转、自托管代理和订阅能力复用场景

- 价值在于协议兼容，而不是改变底层模型本身
