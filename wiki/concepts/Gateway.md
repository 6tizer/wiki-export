## 定义

Gateway 是 OpenClaw 的消息与执行中枢，负责接收外部渠道消息、路由到对应 Agent，并协调工具调用与响应返回。

## 关键要点

- 一个 Gateway 可以连接多个消息渠道和多个 Agent

- 它让单进程托管多角色、多群组或多工作流成为可能

- 在多 Agent 场景中，Gateway 是消息入口与会话路由的核心枢纽
