## 定义

Docs Canvas Skill 是 Cursor 用来教 Agent 生成特定类型画布的技能范式，示例场景是为代码仓库自动生成可交互的架构图。

## 关键要点

- 说明 Canvas 能力可以被封装成可复用的 Skill，而不只是一次性演示功能

- 让 Agent 在特定任务里按固定模式生成更稳定的界面输出

- 适合把团队常见工作流沉淀为可重复调用的画布模板

- 官方示例是 Docs Canvas Skill，可为仓库生成交互式架构视图
