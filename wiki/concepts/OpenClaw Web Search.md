## 定义

OpenClaw Web Search 是 OpenClaw 的网页搜索与正文抓取能力，用来让 Agent 主动搜索互联网并提取可读网页内容。它既可以在通过 Ollama 启动 OpenClaw 时自动启用，也可以作为独立插件安装到本地环境中。

## 关键要点

- 它把 OpenClaw 从“只会基于上下文回答”扩展成“能主动联网检索并抓取页面”的执行型 Agent

- 本地模型和云模型都可以借助这项能力补齐实时信息获取能力

- 对应插件名为 `@ollama/openclaw-web-search`，文档中也称为 Web search and fetch
