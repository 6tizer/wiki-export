## 定义

Codex Pets 是 OpenAI Codex 编辑器中的虚拟宠物功能，用户可通过 `/pet` 命令唤醒一只像素风电子宠物，悬浮于 macOS 桌面。宠物会实时反映 Codex 工作状态（运行中、等待输入、待 Review），并显示简短进度提示，实质上是一个拟人化的状态指示器。

别名：电子宠物、Codex 桌面宠物

## 关键要点

- 通过 `/pet` 命令或 Settings > Appearance > Pets 面板唤醒

- 快捷键 Cmd+K 呼出命令菜单，执行 Wake Pet / Tuck Away Pet

- 官方预设八种不同宠物形象

- 安装 `hatch-pet` skill 后，可用 `$hatch-pet` 命令根据最近项目内容自动生成定制宠物

- 灵感可能来自 Claude Code 源码泄露的 Buddy System（内置电子宠物系统）

- 本质是 Codex 的状态栏可视化，但以游戏化方式呈现，提升开发者体验
