Agent Mindshare 是由 [Cookie.fun](http://cookie.fun/) 平台开创的 AI Agent 代币注意力份额指标，衡量某个项目在加密社区中被讨论、引用、关注的比例，是比市値更灵敏的叙事先行指标。

## 核心逻辑

- Mindshare = 该项目在 AI Agent 社区中被提及/讨论的份额

- **先行性**：Mindshare 变化通常早于市値变化反映叙事转向

- 案例：AIXBT Mindshare 从 9.81% 跳升至 16.1%，领先市値翻倍；Fartcoin Mindshare 先跌则市値后跌

## 实际应用

- **叙事判断**：Mindshare 持续上升 → 叙事热度在上升

- **风险预警**：Mindshare 下滑但市値未动 → 可能的叙事松动信号

- **赛道对比**：跨项目 Mindshare 对比，找出被低估的项目

## 数据来源

[Cookie.fun](http://cookie.fun/)（@cookiedotfun）提供实时 AI Agent Mindshare 追踪数据和 Swarm API 接入

## 各项目 Mindshare 快照（2025 元旦）

- **AIXBT**：16.1%（+8%）——AI 市场分析终端，换 NFT PFP 引爆叙事

- **Virtuals**：8.97%（+4.3%）——Agent 生态操作系统

- **AI16Z**：8.79%（+4.1%）——Eliza 框架 DAO

- **Fartcoin**：8.09%（-13%）——纯 Meme 叙事开始走弱
