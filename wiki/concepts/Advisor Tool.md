## 定义

Advisor Tool 是 Anthropic 提供的一种工具能力，用于在任务运行过程中把更强模型作为顾问按需调用。

## 关键要点

- 它通常与 Advisor Strategy 搭配理解

- 主模型在遇到高难决策点时，可以把共享上下文交给更强模型获取指导

- 文章指出该能力需要在 API 请求层显式配置，而非 Claude Code 默认可用
