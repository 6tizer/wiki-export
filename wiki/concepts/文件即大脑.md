## 定义

文件即大脑是 OpenClaw 的核心设计哲学，指 Agent 的身份、规则与记忆不再隐藏在一次性 prompt 或黑盒服务中，而是拆分为一组可读、可编辑、可版本控制的 Markdown 文件。

## 关键要点

- Agent 的“大脑”以 workspace 文件结构长期存在，支持审计、回滚与多人协作。

- 通过分层加载不同文件，可以把稳定人格、操作规则、用户上下文与长期记忆拆开治理。

- 这种设计让 Agent 更像可维护的软件系统，而不是每次都从零开始的一次性对话机器人。
