## 定义

KAIROS 是 Anthropic 内部代号，即 Claude Code 的亲生龙虾项目，在 Claude Code 源码泳露事件中被外界发现。具备心跳机制、autoDream、主动推送、PR 订阅等 OpenClaw 全功能对标的自主 Agent 特性。

## 核心功能

- **心跳机制**：每隔几秒收到一个“心跳” Prompt，决定是主动动手还是继续安静待机

- **24 小时后台运行**：贷与 OpenClaw 心跳机制相同，能修代码 bug、回消息、更新文件

- **autoDream**：晚上运行记忆整合，将白天学到的东西整合压缩为结构化 [MEMORY.md](http://memory.md/)

- **主动推送**：直接向手机或电脑发通知，无需用户开终端

- **PR 订阅**：监控 GitHub，代码变动自动响应

## 与 OpenClaw 的关系

KAIROS 被认为是 Anthropic 对 OpenClaw 的全方位对标，官方能直接访问模型权重，架构可做得更极致。
