## 定义

结构化网页抓取是指抓取网页后直接输出 Markdown、JSON、摘要或链接列表等更适合 Agent 消费的结果，而不是原始 HTML。

## 关键要点

- 核心价值是降低下游解析成本

- 对动态页面与批量站点抓取尤其重要

- 常被包装成 API 或 Skill 供 Agent 调用
