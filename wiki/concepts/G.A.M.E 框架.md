## 定义

G.A.M.E（Generative Autonomous Multi-modal Entity）是 Virtuals Protocol 的核心 AI Agent 框架，支持**无代码链上 Agent 创建**和代币化变现。

## 关键特点

- Virtuals 生态专属，链上集成深度强

- 无代码可视化界面创建 Agent

- Agent 与代币化经济模型绑定

- 适合 DeFi Agent 项目开发

- 与 ERC-8183 共同构建去中心化 Agent 经济体系

## 对比同类

- **G.A.M.E**：Virtuals 生态专属，适合 DeFi Agent

- **ElizaOS**：通用性更强，开源生态更开放
