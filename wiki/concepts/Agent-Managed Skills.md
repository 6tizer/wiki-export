## 定义

Agent-Managed Skills 是 Hermes 把重复成功的工作流沉淀为可复用 Skill 的机制。它让 Agent 不只“记住经验”，还能把经验整理成触发条件、操作步骤、注意事项和验证方法。

## 关键要点

- 它对应的是程序性知识，而不只是事实记忆

- 目标是把非平凡工作流固化为可复用 SOP，减少重复推理成本

- 高质量 Skill 通常需要清晰的触发条件、可执行步骤、注意事项和验证方法

- 当某类经验反复出现在记忆里时，更适合升级为 Skill，而不是继续散落在记忆文件中
