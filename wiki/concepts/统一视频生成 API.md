## 定义

统一视频生成 API 是一种把文生视频、图生视频与不同模型厂商的接口差异封装到单一接入层的模式，让开发者用同一套调用方式接入多种视频模型。

## 关键要点

- 把视频生成纳入与文本、图像、音频并列的统一 API 体系

- 通过路由封装，减少开发者分别适配不同厂商接口的成本

- 适合在上层工作流里快速切换模型、比较效果与控制成本
