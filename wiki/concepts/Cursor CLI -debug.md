## 定义

Cursor CLI /debug 是 Cursor CLI 中面向疑难问题排查的调试命令，核心目标是先定位根因，再生成修复方案。

## 关键要点

- 适合处理难复现、难理解或上下文复杂的 bug。

- 它把“找原因”单独提升为明确模式，而不是把调试混在普通编码对话里。

- 对终端场景尤其有价值，因为很多线上问题、日志问题和环境问题天然发生在 CLI 中。
