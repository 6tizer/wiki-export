## 定义

Agent 治理是指围绕组织内 Agent 的权限边界、审批流程、运行审计、配置可见性与风险控制所建立的一整套管理机制，确保 Agent 能在企业环境中被安全、可控地部署与复用。

## 关键要点

- 它关注的不是 Agent 能不能做事，而是 Agent 在组织里如何被安全地授权、监督与暂停。

- 治理能力通常包括工具接入范围、数据访问权限、敏感操作审批、运行记录和管理员控制台视图。

- 当 Agent 从个人玩具变成团队基础设施后，治理往往比模型能力更决定能否真正落地。
