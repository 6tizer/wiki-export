## 定义

XML Tags 是一种用结构化标签划分提示上下文、任务、约束和输出格式的方法，用来降低复杂 Prompt 的歧义。

## 关键要点

- 通过显式分段，模型更容易区分背景信息、执行目标与边界条件

- 它适合组织包含多段材料、约束和格式要求的长 Prompt

- XML Tags 的价值不在于“特殊语法”，而在于提升输入结构清晰度
