## 定义

Tool Maker 指 Agent 在平台未预置现成能力时，能够围绕具体任务现场设计、组合并补充新的工具能力，而不是只消费既有工具列表。

## 关键要点

- 与固定 API 清单不同，它强调按任务需求临场补能力

- 文中案例里 Agent 先后补出了原推直链与 og:image 预览能力

- 这种能力让 Agent 从“会用工具”升级为“会给自己造工具”

- 它体现了 Agent 平台在执行层面的开放性与适应性
