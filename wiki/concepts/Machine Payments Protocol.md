## 定义

Machine Payments Protocol，简称 MPP，是 Stripe 与 Tempo 推出的机器支付协议，目标是让 AI Agent 以编程方式完成授权、微支付与结算。

## 关键要点

- 让支付像 HTTP 请求一样可编程

- 用 Session 管理消费上限与持续支付

- 适合高频微支付与 Agent 工作流场景
