## 定义

Skill File 是一种用结构化文本把任务边界、执行步骤、判断标准与调用约束写清楚的技能载体，用来让 Agent 在特定任务上减少即兴发挥、提高可复用性与稳定性。

## 关键要点

- 它把原本隐含在 prompt 里的经验外显成可维护的操作规范。

- Skill File 常和 harness、resolver、工具调用一起工作：前者负责运行时，后者负责把正确任务送到正确技能。

- 它的核心价值不是让模型更聪明，而是让模型在已知任务上更少偏航。

- 对 Coding Agent 而言，Skill File 也是团队经验沉淀和行为标准化的关键接口。
