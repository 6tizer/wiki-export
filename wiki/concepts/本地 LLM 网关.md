## 定义

本地 LLM 网关是部署在本机或本地网络中的模型代理层，用统一 API 屏蔽上游模型差异，并负责路由、鉴权、回退和流量治理。

## 关键要点

- 将多个模型提供商抽象成单一入口

- 方便 IDE、CLI 和 Agent 工具共用配置

- 常见能力包括回退、负载均衡、成本控制和日志观测
