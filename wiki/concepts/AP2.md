## 定义

AP2 是 Google Cloud 推进的 Agent Payments Protocol 路线，试图通过授权书与可验证凭证建立 Agent 支付中的信任机制。

## 关键要点

- 关注支付授权与身份可信度

- 与 MPP、x402 共同构成 Agent 支付协议竞赛

- 更强调企业级信任与可验证性
