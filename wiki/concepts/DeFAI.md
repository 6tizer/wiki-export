DeFAI（DeFi + AI）是去中心化金融与 AI Agent 深度融合的新范式，指 AI Agent 作为 DeFi 协议的自主用户，自主执行链上交易、管理流动性、执行投资策略的应用场景。

## 核心逻辑

"AI 需要 Crypto 实现最终自主性，Crypto 需要 AI 实现大规模采用"——两者内在互补：

- **AI 侧**：自主 Agent 需要无需许可的价値传输层（去中心化账户/结算）

- **Crypto 侧**：AI Agent 可将复杂 DeFi 操作抽象化，显著降低普通用户使用门槛

## 代表应用

- **Agentic DeFi**：AI Agent 自主管理流动性和投资组合

- **Corporate Audit AI**：AI 自动分析 SEC 文件，提供透明审计

- **ai-hedge-fund**：多 Agent 模拟不同投资风格进行协作决策

- **Arma Agents**：7 个月 TVL 从 20 万 → 1120 万，累计代理交易量 3.24 亿

## 协议侧改造需求

DeFi 协议迎接 AI Agent 需要：OpenAPI 规范文档 + 无状态 API + WebSocket + 批处理端点 + 异常检测
