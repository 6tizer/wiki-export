## 定义

Vibedeploy 是 Vibe Coding 向部署环节的自然延伸，指通过 CLI 命令或 AI Agent 以对话/提示的方式直接完成服务编排、凭证管理和产品上线，无需手动操作多个仪表盘或逐一注册第三方服务。该术语由 Stripe 在推广 Stripe Projects 时提出。

别名：Vibe Deploy、Vibe Deploying

## 关键要点

- 核心理念：从「想法到部署」的时间极度缩短，开发者（或 Agent）在 CLI 中一站式完成服务供应、凭证同步和计费管理

- 由 Stripe Projects 平台落地：支持 Vercel、Netlify、ElevenLabs、Twilio、Auth0、Algolia、Sentry 等十余家服务商

- 与 Vibe Coding 的区别：Vibe Coding 侧重代码生成阶段，Vibedeploy 侧重基础设施编排与部署阶段

- Agent 友好：Stripe Projects 允许 AI Agent 以编程方式完成整个供应流程，是 Agentic Commerce 的基础设施
