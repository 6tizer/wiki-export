## 定义

提示注入（Prompt Injection）是指攻击者通过网页内容、文档、工具返回值或其它外部输入，把恶意指令嵌入 Agent 的上下文中，从而诱导模型偏离原始任务、暴露隐藏提示词、泄露敏感信息或执行越权操作。

## 关键要点

- 它既是安全攻击手法，也是理解 Agent 边界的重要研究入口

- 在 System Prompt 提取场景中，提示注入常被用来诱导模型暴露上层规则

- 对 Agent 产品来说，提示注入意味着提示本身不能被视为完整安全边界

- 与传统注入攻击不同，Prompt Injection 利用的是模型对自然语言指令的服从倾向，而不是程序语法解析缺陷

- 只要 Agent 仍能直接读取高价值密钥或会话信息，提示注入就可能把这些能力转化为实际损害，因此安全设计需侧重最小权限和能力隔离
