## 定义

外部记忆是存储在上下文窗口之外、可被 Agent 持续读取和更新的记忆层，例如 workspace 文档、[memory.md](http://memory.md/)、[soul.md](http://soul.md/) 等，用于保存长期偏好、流程经验与协作共识。

## 关键要点

- 与 in-context memory 互补，负责跨会话持久化

- 可以沉淀人与 Agent 的长期互动、调教记录与最佳实践

- 在协作平台中不仅服务单个 Agent，也可以成为团队共享记忆

- 是 Agent 可 fork、可迁移、可复用的重要资产

- 能承载渐进式披露所需的入口信息
