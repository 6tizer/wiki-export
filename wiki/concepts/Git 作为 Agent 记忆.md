## 定义

Git 作为 Agent 记忆，是指把提交历史、差异记录与回滚机制直接当作 Agent 的长期记忆载体，让系统能够追踪试过什么、哪些有效、哪些失败。

## 关键要点

- 用版本历史替代额外记忆数据库

- 让实验结果具备可追溯性与可归因性

- 适合代码、研究和持续优化场景
