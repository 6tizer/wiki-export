## 定义

网站 CLI 化是把网页或桌面应用封装成结构化命令行接口的做法，让 Agent 以统一命令范式访问原本没有标准 API 的服务。

## 关键要点

- 核心价值是统一调用方式而不是替代浏览器

- 适合 Agent 需要稳定读取、触发和串联多平台能力的场景

- 让“没有 API 的网站”也能被纳入自动化工具链
