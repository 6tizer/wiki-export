## 定义

Agent 数据采集 API 是专门为智能体提供网页、站点或搜索结果抓取能力的接口层。

## 关键要点

- 让模型把重点放在决策与理解，而不是底层抓取细节

- 往往内置反爬、异步抓取和结构化输出能力

- 是 Agent 真正具备“上网能力”的关键基础设施
