## 定义

链上交易 Skill 是把行情筛选、合约查询、风险检查和交易执行封装成可调用技能的 Agent 能力单元。

## 关键要点

- 把扫链、调研和下单流程标准化成可复用操作

- 适合高频、低时延和步骤固定的交易场景

- 常通过 API Key 直连真实钱包或交易平台
