## 定义

SearxNG 是一个可自托管的开源聚合元搜索引擎，用于整合多个搜索源并输出原始搜索结果。

## 关键要点

- 核心价值在于零 API Key、可私有部署和对搜索源的高度可配置性。

- 它适合作为 Agent 的底层检索层，让主模型自己判断是否继续读取全文。

- 在本地 Agent 体系里，SearxNG 常被视为低成本、低耦合的“搜索外脑”。
