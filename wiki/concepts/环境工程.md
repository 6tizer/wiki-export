**定义**：环境工程（Environment Engineering）是 AI Agent 领域的新兴概念，指构建和优化 Agent 所运行的现实世界模拟环境，包括工具的自动配置、任务边界的智能定义和 Agent 自主适应的机制。

**与 Harness 工程的差异**

- Harness 工程：人工构建和维护周围 Agent 的控制层，短期有效

- 环境工程：构建能让 Agent 自主适应和学习的世界模拟，长期目标

**关键特性**

- Agent 可以在环境中自主学习和适应，无需持续人工介入

- 显著提升开发效率（杠杆率），将复杂开发过程简化

- 商业场景中环境工程是长期目标，不能作为短期捷径
