## 定义

CLI Harness 是把复杂软件能力封装为结构化命令行接口的一层适配机制，使 Agent 能以稳定、可审计的命令方式调用真实软件功能。

## 关键要点

- 将 GUI 行为映射到可编排的 CLI 子命令

- 通常支持 JSON 输出，便于 Agent 消费

- 比截图点击式自动化更稳定、更省上下文
