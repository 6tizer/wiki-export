## 定义

Routines 是 Claude Code Desktop 新增的云端自动化任务能力，把 prompt、仓库访问权限和 Connectors 打包成可重复运行的任务，无需本地机器持续在线。

## 关键要点

- 支持定时调度、GitHub 事件触发、API 调用三类触发方式

- 运行在 Anthropic 云端基础设施，不需要自建 cron job、服务器或常驻电脑

- 适合 issue triage、PR review digest、dependency update check、flaky test tracker 等开发自动化场景

- 将本地 CLI 能力扩展为持续运行的云端 Coding Agent 工作流
