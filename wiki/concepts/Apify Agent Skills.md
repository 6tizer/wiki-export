## 定义

Apify Agent Skills 是云端爬虫平台 Apify 推出的一套预制技能包，专门教 AI Coding Agent 如何做结构化数据采集，支持 Claude Code、Cursor、Codex、Gemini CLI 等主流 AI 编程工具。

## 关键要点

- 核心技能 apify-ultimate-scraper 是万能爬虫，支持 Instagram、Facebook、TikTok、YouTube、Google Maps 等平台

- 返回结构化数据（CSV/JSON），而非文本摘要

- 12 个技能覆盖竞品分析、品牌口碑监测、电商数据采集、KOL 发现等场景

- 按结果付费，有免费额度

- 与 Apify MCP Server 互补：Skills 适合固定场景，MCP 适合灵活场景
