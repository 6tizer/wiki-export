## 定义

Cursor CLI /config 是 Cursor CLI 内置的配置面板入口，用于查看和修改默认行为、运行偏好与相关设置。

## 关键要点

- 它把配置管理从外部文件或隐藏参数，转为 CLI 内可见、可操作的界面。

- 配合 /update-cli-config 技能，配置本身也可以交给 agent 协助完成。

- 这说明 CLI 不只是执行入口，也在演化为可持续使用的运行环境。
