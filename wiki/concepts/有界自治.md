## 定义

有界自治（Bounded Autonomy）是一种 Agent 安全模型，核心原则是：Agent 的自主权必须受到身份、授权范围、预算上限和审计链路的明确约束。与完全自主或完全受控相对，有界自治在「让 Agent 独立行动」和「保持人类监督」之间取得平衡。

别名：Bounded Autonomy

## 关键要点

- **独立身份**：Agent 不共享人类凭证，拥有专属的 OAuth/OIDC 身份和作用域受限的 API Token

- **最小权限**：每个 Agent 只获得完成任务所需的最小权限集

- **预算控制**：通过 Stripe Projects 等机制设置消费上限（如默认 $100/月），防止失控消费

- **审批节点**：关键操作（如首次订阅、大额支出）需人类确认，日常操作可自动执行

- **可撤销性**：Agent 的权限和预算可随时由人类撤销或调整

- 被 Michael Guo 总结为："Autonomy without boundaries is a risk. Autonomy with identity, consent, least privilege, and spend controls is how agents become production-grade infrastructure."

## 与 Tizer 工作流的关联

有界自治是 Harness 工程的核心设计原则之一，适用于 OpenClaw 中 Agent 的权限模型设计——确保 Agent 在执行任务时有明确的行动边界。
