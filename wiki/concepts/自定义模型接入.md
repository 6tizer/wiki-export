## 定义

自定义模型接入，是指在 Claude Code 这类宿主工具中，不直接使用原生默认模型，而是通过 provider、别名或适配层，把第三方模型接入同一套命令行工作流。

## 关键要点

- 文中 Kimi K2.6 以 `kimi-for-coding` 的身份进入 Claude Code CLI，属于典型的自定义模型接入。

- 这种方式保留了既有工作流入口，但体验会受到宿主与非原生模型之间适配层的影响。

- 差异通常体现在流式输出流畅度、长会话记忆、自主修复能力，以及工具链一体化程度。
