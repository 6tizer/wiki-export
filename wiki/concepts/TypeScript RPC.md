## 定义

TypeScript RPC 是把工具能力暴露为 TypeScript 函数调用的通信方式，让模型直接调用函数而不是手写 HTTP 请求。

## 关键要点

- 减少 prompt 中描述 API 结构的成本

- 降低工具调用格式出错率

- 适合 Agent 场景中的高频、结构化工具调用
