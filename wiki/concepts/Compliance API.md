## 定义

Compliance API 是面向管理员与企业治理场景的接口能力，用于查看 Agent 的配置、变更历史与运行记录，并支持在需要时对 Agent 进行暂停或审计。

## 关键要点

- 它为企业提供 Agent 的可观测性与审计链路，而不只是让 Agent 更会干活。

- 在组织规模化部署 Agent 时，配置历史、行为记录和应急暂停能力是合规落地的重要基础设施。

- 它通常与权限控制、审批机制和管理控制台一起构成完整的 Agent 治理能力。
