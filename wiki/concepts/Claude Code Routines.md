## 定义

Claude Code Routines 是 Claude Code Desktop 提供的**云端自动化任务能力**，可以把 Prompt、仓库访问权限和 Connectors 打包成可重复运行的任务，在 Anthropic 的云端按计划或事件触发执行。

## 关键要点

- 不依赖本地电脑持续在线，适合替代必须常驻本机的 Scheduled Tasks

- 支持三类触发方式：定时调度、GitHub 事件触发、API 调用触发

- 面向开发团队的常见自动化场景，例如 Issue triage、PR review digest、依赖更新检查、Flaky test 跟踪

- 本质上是把 Coding Agent 从“交互式编程助手”扩展为“可托管运行的自动化执行器”
