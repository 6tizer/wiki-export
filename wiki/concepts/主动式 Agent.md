## 定义

主动式 Agent 是一种不等待用户逐条下指令，而是能够主动抓取信息、判断优先级，并在后台连续完成多步动作的 Agent 形态。

## 关键要点

- **从被动响应转向主动执行**：不再局限于一问一答，而是围绕目标持续推进任务。

- **能力提升伴随成本跃迁**：相比普通对话式应用，主动式 Agent 往往会显著放大 token 消耗与工具调用次数。

- **体验与成本强绑定**：更高的主动性通常意味着更强的产品感知，但也更容易触发单位经济失衡。

- **适合高价值任务场景**：只有当任务价值足够高时，主动式执行带来的额外成本才可能被商业化覆盖。
