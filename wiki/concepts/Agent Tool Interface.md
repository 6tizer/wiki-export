## 定义

Agent Tool Interface 是一类专门为 Agent 调用设计的工具接口形态，强调低上下文负担、可组合性、可调试性和稳定执行。

## 关键要点

- CLI、MCP、结构化 API 都可以成为其实现形式

- 好的接口会显著降低 Agent 使用工具的成本和出错率

- 是软件“面向 AI 用户”重构时的关键抽象
