## 定义

Agent 云设备是指为 Agent 提供的独立可操作计算终端，包括云电脑、云手机及其配套工作环境，使 Agent 能像人类一样直接操作桌面系统、移动应用、文件与网页。

## 核心要点

- **独立执行环境**：Agent 在后台运行任务时不占用主对话线程，可持续完成数据采集、文件处理、页面操作等任务。

- **双端形态**：云电脑提供桌面级生产力与代码执行能力；云手机提供移动端 App 操作、滑动点击与实时推流能力。

- **人机接管机制**：在关键操作节点保留人工接管入口，适合高风险或需要确认的任务。

- **运行基座价值**：它不是单一工具，而是让 Agent 从 API 调用者升级为数字环境操作者的底层基础设施。
