## 定义

画布式 Agent 编排是一种把多个 Agent 的状态、任务分工与结果呈现在同一可视化画布中的交互范式，使用户能像管理工作台一样组织多 Agent 协作。

## 关键要点

- 它把传统隐藏在提示词或后台流程中的编排逻辑前置到界面层。

- 用户可以在同一屏幕里观察多个 Agent 的并行工作，从而降低多线程协作的切换成本。

- 这种模式通常比纯聊天式交互更强调空间布局、任务可见性和人工调度。
