## 定义

模块化内容排版是一种将排版样式预定义为固定模块（而非依赖 prompt 即时生成）的方法，由 AI Agent 根据内容语义自动选择最合适的模块组合。与 prompt 驱动排版每次输出不一致不同，固定模块确保同样内容每次输出一致。

## 关键要点

- 将排版拆分为多个功能模块（如开头吸睛、观点表达、数据信息图、证据支撑、品牌表达、转化成交等大类）

- Agent 先分析内容语义，再自动匹配最少但最准确的模块

- 模块化方法比纯 prompt 方法更稳定——prompt 每次输出不同，模块每次一致

- 核心理念：「用对三五个模块比堆十几个效果好」

- 模块通过 JSON 驱动，支持 Agent 原生接入（如 Claude Code、Codex）
