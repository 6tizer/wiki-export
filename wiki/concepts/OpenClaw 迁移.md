## 定义

OpenClaw 迁移是指将 OpenClaw 既有的配置、记忆、Skills、API 密钥与工作区说明文件迁移到 Hermes Agent 的过程，使用户能在不完全重建工作区的前提下切换到底层能力更强的新 Agent 栈。

## 关键要点

- 迁移对象不仅是模型配置，还包括记忆文件、Skills、消息平台设置与工作区上下文。

- 这类迁移机制能显著降低老用户尝试新 Agent 框架的切换成本。

- 对 Hermes 而言，迁移能力也是其争夺 OpenClaw 生态用户的重要入口。

- 在实际工作流里，迁移意味着把历史知识资产一并带走，而不是只替换对话界面。
