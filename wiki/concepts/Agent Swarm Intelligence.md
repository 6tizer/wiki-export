## 定义

Agent Swarm Intelligence 是一种多 Agent 群体协作思路，通过自动派生专门化子 Agent 并行处理子任务，再汇总结果完成复杂目标。

## 关键要点

- 重点不在单个 Agent 更强，而在协作结构更有效

- 更适合可以拆解并并行推进的问题

- 需要额外的协调、通信和冲突管理机制
