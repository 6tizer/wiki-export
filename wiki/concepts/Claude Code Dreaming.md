## 定义

Claude Code Dreaming 是 Claude Code 长期记忆体系中的后台整理机制，利用空闲时段跨会话回顾、合并和剪枝记忆，类似“睡眠时巩固记忆”的工程实现。

## 关键要点

- 不面向即时回答，而是面向长期记忆质量维护

- 通过后台异步运行避免打断主工作流

- 体现了 Agent 记忆从“存下来”走向“持续整理”的演进方向
