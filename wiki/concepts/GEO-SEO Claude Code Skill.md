## 定义

GEO-SEO Claude Code Skill 是一个运行在 Claude Code 上的开源审计工具集，用于同时评估网站在生成式搜索和传统搜索中的可见度表现。

## 关键要点

- 以 GEO 为主线，同时覆盖技术 SEO 检查

- 支持并行 Agent 审计和评分输出

- 面向内容引用率、结构化数据和 AI 可见度优化
