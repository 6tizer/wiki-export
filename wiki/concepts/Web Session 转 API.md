## 定义

Web Session 转 API 是把网页端登录态转换为可编程接口访问能力的做法，使原本只能在官方前端使用的服务被统一接入本地代理或自动化工作流。

## 关键要点

- 本质是把交互式会话抽象成可复用凭证

- 常用于把网页产品接入统一 API 网关

- 价值在于兼容旧生态，但也伴随更高的失效与风控风险
