## 定义

Tool Registry 是对 Agent 可调用工具或 API 的结构化目录，用于描述可用能力及其调用入口，支撑运行时选择与编排。

## 关键要点

- 让 Agent 不必预先写死所有可用工具

- 适合作为动态工具发现的上游能力目录

- 常与函数描述、元数据和调用约束一起使用
