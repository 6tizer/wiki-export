## 定义

Holdout Set 是与优化集分离的一组保留评测样本，用来检验 Agent 或 Harness 的改进是否真正具备泛化能力，而不是只对已知样本过拟合。

## 关键要点

- 角色类似模型训练里的验证集

- 能帮助区分“真实进步”和“对测试题背答案”

- 对持续迭代的 Agent 系统尤其重要
