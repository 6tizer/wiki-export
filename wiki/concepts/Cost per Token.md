## 定义

Cost per Token 指系统每产出一个 Token 所对应的综合成本，是衡量推理系统商业可行性和资源利用效率的核心经济指标。

## 关键要点

- 它把 GPU 空等、通信开销和基础设施成本都折算到 Token 产出上。

- 同样的硬件条件下，单位时间产出的有效 Token 越多，Cost per Token 越低。

- 该指标迫使网络、计算、存储等团队围绕共同经济目标协同优化。

- 在推理成为主战场后，它比单纯硬件单价更能解释系统效率。
