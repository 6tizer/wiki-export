## 定义

Cursor CLI /statusline 是 Cursor CLI 的状态栏定制能力，用于把会话状态、运行时信号和用户关心的指标显示在终端界面中。

## 关键要点

- 它提升了 agent 运行过程的可观测性。

- 用户可以把抽象的运行状态转成稳定可见的界面信号，减少“现在系统在做什么”的不确定感。

- 这类能力对于长时间运行的 Coding Agent 和终端自动化尤其重要。
