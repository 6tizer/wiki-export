## 定义

Large Memory Model（LMM，大记忆模型）是一种全新的 AI 架构，与 LLM 形成对比。LLM 将世界文本压缩为权重，在 prompt 时给出回答；而 LMM 反其道行之——捕获用户所见、所谈、所到之处的上下文，在合适的时刻自动浮现正确的信息，**无需 prompt**。

别名：LMM

## 关键要点

- LMM 专为人类记忆的工作方式设计，不依赖 RAG 或向量搜索，属于全新范式

- 核心理念是 **persistent memory**（持久记忆），赋予每个应用持久化的上下文记忆能力

- 与 LLM 的核心区别：LLM 是「压缩→检索」，LMM 是「捕获→自动浮现」

- 由 Engramme 公司提出并开发
