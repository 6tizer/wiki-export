## 定义

阿里百炼 Coding Plan 是阿里云百炼平台提供的 **Anthropic API 协议兼容方案**，让国内用户通过阿里帐号和支付宝付款，在 Claude Code 等工具中调用国产大模型（通义千问、Kimi、GLM 等），无需海外信用卡和代理。

## 关键优势

- 国内直连，无需代理

- 支付宝付款，无需海外信用卡

- 兼容 Anthropic API 协议，配置方式与官方完全一致

## 支持模型

Kimi-k2.5、qwen3.5-plus、glm-5、MiniMax-M2.5 等。

## 注意事项

- Coding Plan 费用约 200 元/月（2026-04 时价）

- 最新 Claude Code 版本可能存在兴容性问题，如遇到 API Error 400 可尝试关闭 Thinking mode

- 也可直接使用 Kimi/智谱 GLM 的 API Key 替代
