## 定义

Agent 原生视频编辑，是指把视频创作流程设计成更贴近大模型能力分布的形式，让 Agent 直接通过编写或修改 HTML、CSS、JavaScript 等熟悉媒介来生成、调整并渲染视频。

## 关键要点

- 不要求 Agent 学习传统时间轴剪辑软件，而是复用其更擅长的 Web 代码表达

- 能把“改标题大小、加转场、换暗色背景”这类要求直接映射为代码修改

- 适合与 [Claude Code](entities/Claude Code.md) 之类的 Coding Agent 协同工作

- 其核心目标不是替代所有剪辑工具，而是为自动化内容生产建立更可编程的接口
