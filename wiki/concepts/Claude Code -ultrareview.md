## 定义

Claude Code /ultrareview 是 Claude Code 提供的深度代码审查命令，用于在生成或修改代码后，发起一轮更严格、更系统的审查流程，以发现 bug、设计缺陷与实现风险。

## 关键要点

- 它把代码审查从一次普通问答，提升为带有明确审查角色和审查目标的专门流程

- 它适合放在交付前、演示前或合并前，作为质量闸门使用

- 在 Agent 编程工作流中，它体现了“先生成，再复核”的 Builder-Reviewer 分工思想
