Agentic DeFi 是 AI Agent 作为 DeFi 协议自主参与者的新范式，Agent 可以自主管理流动性、执行交易策略、参与链上治理，而无需人类的每次操作授权。

## 核心特征

- **自主执行**：Agent 24/7 监控链上状态，自动执行预定策略

- **高频调用**：Agent 每分钟可发起数百次 API 请求，与人类用户的行为模式完全不同

- **链上结算**：无国界、无许可的加密货币作为 Agent 间的结算媒介

## Agent-Ready DeFi 协议设计要求

- OpenAPI/Swagger 规范文档（机器可读）

- 无状态端点 + WebSocket 实时推送

- 批处理端点 + 动态频率限制

- 异常检测（防止 Agent 高频操作被利用于闪贷攻击）

- 事件 Webhook + 机器可解析交易日志

## 现状数据（2025-2026）

- 68% 新 DeFi 协议已集成 AI Agent 自主管理流动性

- Arma Agents 7 个月 TVL 从 20 万 → 1120 万美元

- Base 链上 AI Agent 管理稳定币超过 2000 万美元
