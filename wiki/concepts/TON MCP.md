## 定义

TON MCP 是 TON 生态中面向 Agent 的统一能力接口层，用来把不同服务抽象成可组合、可调用的标准化能力，让 Agent 可以跨服务编排任务流。

## 关键要点

- 为 Agent 提供统一的服务调用接口

- 让不同服务之间更容易完成组合与协同

- 强调 Protocol 层在 Agent 执行链路中的连接作用

- 在文章中承担从对话到外部服务调用的连接层角色
