## 定义

混合模型策略是一种在 AI Agent 系统中**用前沿模型做规划、用本地免费模型做执行**的武装配置方法，旨在大幅降低 API 成本的同时保留前沿智能的推理质量。

## 核心逻辑

- 执行层任务（写代码、格式化输出、批量处理）占 AI Token 消耗的约 **90%**——这些交给唔费本地模型

- 规划层任务（目标设定、任务拆解、复杂推理）占约 **10%**——保留前沿商业模型（如 Anthropic Claude）

- 整体 API 账单可降至原来的 1/10

## 典型实现

Alex Finn 的方案：3 台 Mac Studio 跑 Qwen 3.5 + OpenClaw，4 个 Agent 并行开发同一 SaaS 产品；前沿模型处理高难度规划，本地模型处理具体执行。
