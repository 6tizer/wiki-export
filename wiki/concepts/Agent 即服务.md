## 定义

Agent 即服务（Agent-as-a-Service）是一种商业模式，指将 AI Agent 能力打包为可交付的服务产品，由服务商为客户定制、部署和维护 Agent，客户无需具备技术背景即可使用。

## 关键要点

- 核心价值主张是**降低 Agent 使用门槛**：客户只需描述业务问题，服务商负责 Agent 设计、部署和运维

- 典型交付模式：前期 AI 审计（识别痛点） → 定制 Agent → 托管部署 → 按月收费维护

- 与传统 SaaS 的区别在于高度定制化，Agent 可以针对每个客户的具体流程（邮件、CRM、报表等）量身打造

- 受益于托管基础设施（如 Claude Managed Agents）的出现，技术门槛从「组建开发团队」降至「描述需求」

- 目标客户通常是非技术型企业（律所、会计事务所、房产中介、诊所等），他们有明确痛点但缺乏 AI 技术能力
