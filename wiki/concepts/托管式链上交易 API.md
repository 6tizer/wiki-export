## 定义

托管式链上交易 API 是一种由平台代为管理交易执行能力、用户通过 API Key 发起买卖指令的链上接口模式，目的是降低私钥直接暴露到脚本中的风险。

## 关键要点

- 用 API Key 替代私钥直连，提高接入便利性

- 适合自动化交易、Agent 工具链和多链接口统一

- 风险从私钥保管转移为平台托管与权限泄露风险
