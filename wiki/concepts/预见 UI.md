## 定义

预见 UI 是 FloatBoat 团队提出的 AI 产品交互理念，强调系统在用户明确发起操作之前，先理解情境并给出更贴近任务阶段的界面与行动建议。

## 关键要点

- 文章将其与 Chat-as-OS 并列为团队较早系统化提出的核心理念。

- 核心不是增加复杂界面，而是让 Agent 在正确时间点给出更合适的交互组织方式。

- 它与 FloatIM 中 Agent 主动理解用户时间截面、给出具体帮助建议的体验方向一致。

- 可视为从静态 UI 走向情境感知式 Agent 交互的重要过渡概念。
