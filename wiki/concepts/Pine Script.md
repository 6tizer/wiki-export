## 定义

Pine Script 是 TradingView 生态中的专用脚本语言，用于编写指标、策略和图表分析逻辑，也是 AI 在 TradingView 场景下进行指标定制与技术分析自动化的重要接口。

## 关键要点

- 可用于编写自定义指标、策略脚本和图表标注逻辑

- 在这篇文章的语境里，AI 不只是读取图表，还可以辅助生成、编译和调试 Pine Script

- 它让“自然语言描述交易逻辑 → 生成脚本 → 回到图表验证”的闭环更容易形成

- 与实时行情读取、截图分析和 UI 自动化结合后，Pine Script 成为交易研究工作流里的可编程层
