## 定义

JSON Prompt 是把图像描述拆成结构化字段的提示词表达方式，用更明确的层级来约束生成模型对主体、光线、镜头和氛围的理解。

## 关键要点

- 与自然语言 prompt 相比，更强调结构化和可编辑性

- 有助于提高图像风格复现和局部修改的一致性

- 特别适合需要“复刻”或“变体生成”的场景
