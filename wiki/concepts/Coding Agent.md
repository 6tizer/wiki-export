## 定义

Coding Agent 是以代码库、终端与工程工作流为主要执行环境的智能体，能够理解任务、编写代码、运行测试、调试错误，并持续迭代完成较长链路的软件工程任务。

## 关键要点

- 它不只是代码补全工具，而是可以承担实现、验证、修复与维护的一整段闭环

- 价值在于把软件开发拆成可委派、可监督、可回放的长程任务

- 在这篇访谈里，Notion 将其视为通往更通用智能的重要路径，因为 Agent 可以自举自己的工具和能力
