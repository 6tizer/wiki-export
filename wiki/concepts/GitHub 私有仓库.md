**定义**

GitHub 私有仓库是 GitHub 提供的非公开代码与文件托管空间，可用于存放个人笔记、配置文件和版本历史，只有被授权的人可访问。

### 关键要点

- 文章将私有仓库作为 Obsidian 笔记的云端“家”，强调免费且适合纯文本笔记。

- 私有仓库能提供远程备份与跨设备分发能力，同时保留完整提交历史。

- 与普通网盘相比，它的核心优势不只是存储，而是与 Git 的版本控制能力天然结合。

- 文章也提醒私有仓库不应被当作敏感密钥或大型媒体文件的长期保险箱。
