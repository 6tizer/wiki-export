## 定义

云浏览器自动化是把浏览器运行环境放到云端，由 Agent 或脚本远程调度执行网页访问、表单操作、登录态维持和多步骤界面交互的一类能力。它把原本依赖本地桌面的浏览器控制，抽象成可复用的远程执行层，适合 Claude Code 这类终端型 Agent 直接调用。

## 关键要点

- 云端执行能减少本地机器的资源占用和界面卡顿，更适合长时间任务与并发任务

- 这类能力通常会和代理、反检测、验证码处理、持久化 profile 等机制结合，提升复杂网页任务的成功率

- 接入 Coding Agent 后，网页操作可以成为自动化工作流的一部分，而不再依赖人工截图或手动接管
