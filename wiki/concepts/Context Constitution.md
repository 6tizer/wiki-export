## 定义

Context Constitution 是 Letta 提出的一组上下文治理原则，用来约束 Agent 如何把 context 作为稀缺资源进行管理，并在 token 空间中形成身份、记忆与持续学习能力。

## 关键要点

- 它强调 context 不只是提示词容器，而是 Agent 身份、连续性与长期记忆的载体。

- 它关注上下文压缩、记忆组织、反思与长期改进，而不是一次性检索命中。

- 它要求 harness 提供足够的上下文读写与治理能力，原则与基础设施是绑定的。

- 它可以被视为 memory-native agent 的操作原则文档。
