## 定义

Codex GitHub Action 是把本地已验证的 Codex 工作流逻辑搬进 CI 流水线的自动化执行层。

## 关键要点

- 目标是复用本地的 [AGENTS.md](http://agents.md/)、skills 与 scripts，而不是为 CI 重写一套规则

- 让本地开发与持续集成中的 Agent 行为保持一致

- 体现了 Agent 工程从个人环境走向团队流程的关键过渡
