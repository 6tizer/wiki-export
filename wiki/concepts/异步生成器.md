## 定义

异步生成器是一种一边执行一边逐步产出结果的控制流抽象，特别适合表示 Agent 那种长时运行、可流式输出、可中断且需要持续与外部系统交互的主循环。

## 关键要点

- 相比一次性返回结果的普通函数，异步生成器更适合承载长任务中的逐步产出、暂停与恢复。

- 在 Agent 场景里，它天然支持流式 UI、取消执行、背压控制和多个调用方复用同一执行循环。

- 这篇文章把 Claude Code 的核心 loop 解释为“异步生成器而不是 while loop”，强调抽象选择本身就是产品能力的一部分。
