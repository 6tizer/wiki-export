## 定义

Scheduled Run 是将已保存的 Agent 绑定到固定时间表上自动执行的调度机制，让一次跑通的任务从临时操作变成稳定、周期性的流程。

## 关键要点

- 文中案例将 Agent 设为每天早上 8 点自动运行

- 支持把结果持续输出到 Notion 与 Gmail 等渠道

- 价值不在单次执行，而在可预测、可复用、可积累

- 它解决了信息差获取中“今天能找到，明天未必稳定找到”的问题
