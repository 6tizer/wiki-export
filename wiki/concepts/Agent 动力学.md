## 定义

Agent 动力学是用于理解多 Agent 在共享环境中如何分工、协作、竞争、形成群体印象与组织文化的一种分析视角。

## 关键要点

- 关注 agent 数量、角色分工、共享记忆、Token 成本与结果质量之间的非线性关系

- 认为多 Agent 组织会逐渐形成类似企业文化的群体记忆和行为模式

- 用户的管理方式会影响组织形态：协作式会促进共创，竞争式会诱发赛马与“办公室政治”

- 这不是纯技术问题，也与管理学、组织学、社会学密切相关

- 可用于指导 agent-native team 的规模、分工与治理设计
