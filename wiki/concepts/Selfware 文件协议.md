## 定义

Selfware 文件协议是 FloatBoat 团队为 Agent 时代提出的开源文件协议，用于支持 Agent 之间的上下文与数据交换。

## 关键要点

- 文章将其定位为 Agent 专属文件协议，而非传统软件时代的封闭数据格式。

- 强调“文件即软件、可自包含、自进化、数据归用户所有”的理念。

- 目标是降低平台对数据与协议的垄断，使 Agent 协作中的上下文流转更自然。

- 可视为 Agent 原生基础设施中连接存储、记忆与协作的一层。
