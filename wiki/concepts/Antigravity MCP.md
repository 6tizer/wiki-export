## 定义

Antigravity MCP 是一种把 MCP 服务组织为可接入 OpenClaw 的工具层方案，用来为本地 Agent 增加标准化外部能力。

## 关键要点

- 它的意义在于把搜索、抓取等能力变成协议化、可插拔的模块。

- 与单个技能文件相比，MCP 方案更强调标准接口和持续复用。

- 适合本地优先的 Agent 体系扩展外部工具时使用。
