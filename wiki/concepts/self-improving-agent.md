## 定义

self-improving-agent 是一种让 Agent 自动总结使用模式、提炼新技能并持续优化自身工作流的自改进能力模式。

## 关键要点

- 把复盘、抽象和技能沉淀纳入 Agent 自身循环

- 适合长期运行的编程或知识工作场景

- 代表 Agent 从执行工具走向持续学习系统的路径
