## 定义

Codex 插件系统 是围绕 Codex 构建的可扩展能力层，用技能包形式把平台知识、操作规范和执行脚本注入到编程 Agent 中。

## 核心要点

- 用插件把通用编程 Agent 变成领域专用 Agent

- 适合封装平台规则、构建脚本和调试路径

- 是 Codex 从单一产品向生态型 Agent 平台演化的关键接口
