## 定义

Tool Wrapper 模式是把某个库、框架或 API 的核心用法封装进 Skill，让 Agent 在需要时按需加载“领域专家”能力。

## 关键要点

- 适合快速变化或细节繁多的工具文档场景

- 能减少把整份说明塞进系统提示词的上下文浪费

- 本质是在 Skill 层封装专家知识而不是封装执行代码
