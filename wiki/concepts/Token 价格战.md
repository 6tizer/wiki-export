**定义**：Token 价格战是指 LLM API 供应商之间通过持续降低推理 token 价格来争夺市场份额的竞争现象。由 DeepSeek 率先发起，随后引发阿里云、字节跳动等大厂跟进。

**关键要点**

- 2024 年 8 月：DeepSeek V3 降价，引发第一轮 Token 价格战，阿里云、字节等大厂跟进

- DeepSeek R1 发布后：通过夜间空闲时段额外打折，价格战进一步升温

- 2026 年 4 月：DeepSeek V4 输入输出 2.5 折 + 缓存命中再打 1 折，将价格战推向新阶段

- 核心驱动力：底层架构和算法创新持续压低推理成本，成本优势转化为市场竞争力

- 对海外市场冲击力大：以美元计价后小数点位数极多

**受益群体**

- 学生和 AI 初学者

- 小型企业

- 高频 API 调用的 Agent 系统
