## 定义

Cursor Canvas 是 Cursor 让 Agent 直接生成可交互画布的能力，可在对话中产出仪表盘、数据界面和自定义操作面板，而不止返回纯文本。

## 关键要点

- 输出形态从纯文本升级为可操作界面

- 适合调试、数据分析、PR 审查、库学习等信息密度高的任务

- 画布可与 Cursor 现有终端、浏览器、源码控制等工具并列存在

- 使用 React 组件体系渲染，支持表格、图表、diff、待办等组件
