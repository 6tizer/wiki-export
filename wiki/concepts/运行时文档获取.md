## 定义

运行时文档获取是指在 Agent 执行编码任务时，实时拉取最新 API 文档或规范，而不是依赖训练快照中的静态知识。

## 核心要点

- 用最新文档降低过时 API 导致的幻觉

- 适合代码生成、SDK 接入和接口迁移场景

- 通常与 CLI 检索工具和社区维护文档库结合使用
