## 定义

Idea File 是 Karpathy 提出的轻量规格载体，用自然语言描述目标、约束与方向，让 Agent 在各自环境中按需重建实现，而不是直接复用同一份代码。

## 关键要点

- 它把“分享什么”从代码成品切换为意图与设计思路。

- 同一份 Idea File 可以被不同 Agent 按本地工具链、数据和工作流重新实现。

- 这种形式更适合快速传播早期想法，也更便于后续补充讨论和迭代。
