## 定义

**女娲.skill** 是一个 Claude Code 元技能（meta-skill），用于从任意人物的语料（文章、语录、访谈等）中蒸馏出其认知操作系统，生成可调用的 `.skill` 插件包。它是众多"人物.skill"的生成器。

别名：nuwa-skill（GitHub 仓库名：`alchaincyf/nuwa-skill`，6623 Stars）

## 核心功能

- 输入：任意人物的公开语料

- 输出：包含心智模型、决策启发式、表达 DNA 的 `.skill` 文件

- 使用方式：在 Claude Code 中运行 `/create-[person]` 命令生成 skill

- 已生成的 skill 示例：芒格.skill、乔布斯.skill、费曼.skill、马斯克.skill 等

## 关键特点

- 不是语录合集，而是**可运行的思维框架**

- 支持通过 `/slug` 命令调用生成的 skill

- 是整个"人格蒸馏"生态的核心工具
