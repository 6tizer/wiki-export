## 定义

GitHub 假 Star 是指通过批量账号、僵尸账号或商业刷量服务人为制造的仓库 Star 数据，用来伪装项目热度、影响推荐分发或增强外部背书。

## 关键要点

- 常见手段包括批量注册空白账号、购买高质量老账号、模拟真人节奏点星，以及通过 API 批量下单。

- 它会扭曲 GitHub Trending、项目热度判断和投资筛选逻辑，让 Star 不再是可靠的单一信号。

- 在 AI/LLM 项目中，这类造假更突出，容易误导用户、媒体和投资机构。

- 假 Star 只有短期拉升效果，长期可能拖累真实热度与项目可信度。
