## 定义

TON Pay SDK 是 TON 生态中的支付集成组件，用来让 Agent 在执行任务时完成费用计量、支付与结算，是 Agent 从“能做事”走向“能收费”的关键基础设施。

## 关键要点

- 为 Agent 执行结果提供支付与结算通道

- 让每次执行都能被计量并完成资金流转

- 与 TON Wallet 共同构成 Telegram 生态中的支付层

- 在文章中承担 Agent 经济活动的交易基础设施角色
