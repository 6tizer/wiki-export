## 定义

**GBrain** 是由 YC 总裁 Garry Tan 开源的生产级 AI Agent 记忆管理系统，基于 MIT 协议发布。它是 Garry Tan 本人真实运行的配置，专为管理海量个人知识文件而设计，支持 OpenClaw / Hermes Agent 等框架实现对大规模 Markdown 文件的「完美全量召回」（total recall）。

## 核心特性

- 管理 **10,000+ Markdown 文件**、3,000+ 人物档案、13 年日历数据、5,800 条苹果笔记

- 与 OpenClaw / Hermes Agent 深度集成，支持基于全量上下文的智能检索

- MIT 协议开源，可直接用于构建个人 mini-AGI
