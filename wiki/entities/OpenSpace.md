## 定义

OpenSpace 是港大 HKUDS 团队开源的 Agent 技能进化引擎，通过 MCP 接入现有 Agent，把技能从静态提示词变成可修复、可衍生、可共享的动态资产。

## 关键要点

- 不替代 Agent 框架，而是作为可插拔的进化层存在

- 支持 FIX、DERIVED、CAPTURED 三类技能进化

- 同时强调技能复用、跨 Agent 共享与 Token 成本优化
