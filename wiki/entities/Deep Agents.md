## 定义

Deep Agents 是 LangChain 正在构建的开源 Agent 框架，核心主张是让 Harness、记忆与执行层保持开放、可迁移和可自部署。

## 关键要点

- 强调 **开源** 与 **模型无关**

- 使用开放标准，如 `agents.md` 和 `skills`

- 支持通过 Mongo、Postgres、Redis 等插件接入长期记忆

- 目标是避免把 Agent 的记忆与执行逻辑锁死在单一厂商 API 后面
