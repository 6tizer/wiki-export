## 定义

ClawHost 是一个基于 Kubernetes 的 OpenClaw 多租户托管与管理平台，用来批量分发和维护独立 Agent 实例。

## 关键要点

- 它把个人使用的 OpenClaw 扩展成面向团队和 SaaS 的托管能力

- 每个 Bot 可运行在独立 Pod 中，并通过 API 和面板统一管理生命周期

- 其价值在于把 OpenClaw 从个人工具推向平台化运营场景
