## 定义

bb-browser 是一个通过 **Chrome DevTools Protocol** 直接操控真实 Chrome 登录态的浏览器自动化工具，把网站能力封装成可复用的 CLI 与 MCP 接口。

## 关键要点

- 复用真实浏览器的 Cookie、鉴权 Token 与登录环境

- 通过网站适配器把网页操作结构化为命令行能力

- 适合给 Agent 提供高确定性的网页数据抓取与执行入口
