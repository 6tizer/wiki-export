## 定义

Hermes WebUI 是为 Hermes Agent 提供浏览器交互界面的开源前端项目，目标是在保留 CLI 能力的同时，补齐会话管理、工具可视化与移动端访问体验。

## 关键要点

- 强调与 Hermes CLI 的 1:1 功能对等

- 让 Tool Call、子 Agent 和会话状态变得可视化

- 本质上是在为长期运行型 Agent 增加一个更低门槛的操作台
