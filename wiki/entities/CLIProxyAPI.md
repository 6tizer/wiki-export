## 定义

CLIProxyAPI 是一个把多个网页账号 Session 或 API Key 聚合到统一接口后的本地代理项目，用来为上层应用提供 OpenAI 兼容 API。

## 关键要点

- 核心能力是账号池轮询与统一出口

- 既可承接免费额度，也可混入付费 Key 做兜底

- 它把“多账号折腾”抽象成了一个可编排的 API 网关
