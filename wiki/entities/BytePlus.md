## 定义

字节跳动面向企业的云服务与 API 平台。就本文语境而言，它是 Seedance 2.0 面向海外企业开放调用的主要分发入口。

## 核心要点

- 支持企业客户以预付费 API 方式调用 Seedance 2.0

- 覆盖 100 多个国家的企业市场，但当前不包含美国

- 说明视频生成模型已经从论文展示走向可交付的商业接口
