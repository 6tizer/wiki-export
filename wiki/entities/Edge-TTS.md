## 定义

Edge-TTS 是微软提供的免费文本转语音（TTS）服务，基于 Azure Cognitive Services 的 Edge 浏览器语音引擎，支持多语言、多音色的高质量语音合成。开发者通常通过开源 Python 包 `edge-tts` 调用该服务，无需 API Key。

## 关键要点

- 免费使用，无需注册 Azure 账号或申请 API Key

- 支持 300+ 种语音，覆盖中文、英文、日文等多语言

- 语音质量接近商用 TTS 水平，延迟低

- 常被集成到开源项目中作为默认语音合成方案（如 Pixelle-Video、MoneyPrinterTurbo 等视频生成工具）

- Python 包 `edge-tts` 提供简洁的异步 API，易于集成
