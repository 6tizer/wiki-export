## 定义

React Flow 是一个基于 React 的节点式流程图与可视化编辑库，常用于搭建工作流编辑器、流程编排界面和节点拖拽交互。

## 关键要点

- 适合构建节点、连线、缩放、拖拽等可视化编排界面

- 常被用于 Agent 工作流、自动化流程与低代码产品前端

- 在 AI Hedge Fund 中，它被用于让用户拖拽组合不同 Agent 节点，设计投资委员会流程
