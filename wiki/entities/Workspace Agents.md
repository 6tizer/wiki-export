## 定义

Workspace Agents 是 OpenAI 在 ChatGPT 中推出的组织级 Agent 产品形态，面向团队共享使用，能够跨工具执行复杂任务和长流程工作，并支持后台运行、定时触发与多成员协作。

## 关键要点

- 它是 GPTs 向组织级 Agent 平台演化后的代表形态，重点不再只是单次问答，而是持续执行任务。

- 底层由 Codex 驱动，因此兼具代码执行、工具调用与跨步骤处理复杂任务的能力。

- 它强调“共享”而非个人专属：一个 Agent 配置完成后，可以被整个团队复用，也能接入 Slack 等协作入口。

- 它把模板化业务流程沉淀为可运行的 Agent，例如产品反馈路由、销售跟进、周报生成、软件审批与第三方风控。

- 对企业落地而言，权限控制、审批、日志与合规可见性和模型能力同样重要。
