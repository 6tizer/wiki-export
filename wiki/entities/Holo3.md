## 定义

Holo3 是 H Company 推出的 Computer Use 模型系列，面向 GUI 自动化与桌面任务执行，强调高基准分、较低成本与部分开源能力。

## 关键要点

- 同时覆盖旗舰 API 版本与开源轻量版本

- 目标是让模型直接看屏幕、点按钮、操作软件

- 代表了 Computer Use 从闭源能力向开源生态外溢的趋势
