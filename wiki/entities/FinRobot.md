**定义**

FinRobot 是 AI4Finance Foundation 推出的开源 AI Agent 平台，面向金融分析场景，整合 LLM、强化学习与量化分析能力。

### 关键要点

- 超越 FinGPT 的单模型方案，统一多种 AI 技术（LLM + RL + 量化分析）为全栈智能金融方案。

- 核心理念是「AI Agent for Finance」：Agent 能感知环境、做出决策、执行行动，而非简单的模型推理。

- 覆盖投研自动化、算法交易策略、风险评估等场景。

- 开源：[GitHub](https://github.com/AI4Finance-Foundation/FinRobot) ｜ 论文：[arXiv 2405.14767](https://arxiv.org/abs/2405.14767)

### 与知识库其他条目的关系

- 与 FinGPT 同出一脉（AI4Finance 基金会），是其 Agent 化升级

- 多 Agent 金融平台思路与 [StockClaw](entities/StockClaw.md) 可对比

- 与 [Kronos](entities/Kronos.md) 的关系：FinRobot 是上层 Agent 编排，Kronos 是底层预测引擎，二者可组合
