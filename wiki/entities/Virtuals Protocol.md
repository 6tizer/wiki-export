## 定义

Virtuals Protocol 是一个**将 AI Agent 代币化**的区块链平台，让每个 Agent 都能拥有自己的链上经济模型和独立的代币。

## 关键特点

- AI Agent 代币化：每个 Agent 拥有独立的链上经济模型

- G.A.M.E 框架是其核心技术底座

- 与 ERC-8183 共同构建 Agent 经济基础设施

- 具备独角兽级别的市场潜力

- 拥有丰富的早期生态积累
