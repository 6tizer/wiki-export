## 定义

VideoAgent 是面向视频理解、编辑与重制的多模态 Agent 框架，强调通过自然语言对现有视频进行分析、规划与再创作。

## 关键要点

- 将视频理解、编辑和 remake 整合在统一 Agent 工作流中

- 采用意图分解、图式工作流和自我纠正机制

- 适合 meme、解说、音乐视频与跨语言改编等创意场景

- 可结合本地模型与外部 API 形成混合执行方案
