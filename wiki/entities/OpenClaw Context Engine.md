## 定义

OpenClaw Context Engine 是 OpenClaw 在新版本中引入的上下文管理能力，用于在长任务或流式场景中持续维护工作上下文，而不是每轮从零开始。

## 关键要点

- 适合实时数据监控和连续任务处理

- 能减少频繁重建上下文的成本

- 是 OpenClaw 从对话工具走向持续执行 Agent 的关键能力之一
