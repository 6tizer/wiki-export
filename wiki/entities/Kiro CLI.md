## 定义

Kiro CLI 是 AWS 推出的终端 AI 编程 Agent，面向命令行环境提供代码生成、代码审查、Bug 修复、文档检索与自动化执行能力。

## 关键要点

- 以 CLI 形态运行，适合直接嵌入开发者现有终端与脚本环境。

- 2.0 版本的关键升级是引入 **Headless 模式**、正式 GA 的 **TUI**，以及对 Windows 的原生支持。

- 既可以作为开发者交互式编程助手，也可以作为自动化流水线中的无头执行器。

- 与 OpenClaw 结合后，可以作为被自然语言调度的底层 Coding Agent 执行单元。
