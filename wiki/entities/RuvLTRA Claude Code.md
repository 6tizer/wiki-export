## 定义

RuvLTRA Claude Code 是一个面向编程任务优化的开源模型条目，提供 GGUF 量化版本，并强调自学习、群体协同优化与本地可部署能力。

## 关键要点

- 官方叙事将它定位为“为 Claude Code 场景优化的模型”，核心卖点是编码任务适配能力。

- 提供 GGUF 版本，便于通过本地推理工具链在不同硬件环境中运行。

- 模型介绍突出 SONA 自优化架构，强调会从交互中逐步吸收编码习惯与项目约定。

- 支持代码生成、调试、重构、脚本自动化与 API 构建等典型开发任务。
