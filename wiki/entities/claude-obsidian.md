## 定义

claude-obsidian 是一个面向 Claude Code 的 Obsidian 插件与技能包组合，用来把 Obsidian Vault 变成可持续增长的个人 Wiki，使对话、资料和研究结果能够被持续编译、链接与复用。

## 关键要点

- 它建立在 **Claude Code + Obsidian** 的组合之上，把本地 Markdown Vault 变成可维护的知识工作台

- 它遵循 **Karpathy LLM Wiki 方法论**，支持把 PDF、URL、转录稿等原始材料编译成结构化页面

- 它提供 `/wiki`、`/save`、`/autoresearch`、`/canvas` 等命令，覆盖初始化、归档、研究与可视化等常见知识工作流

- 它通过 `hot.md`、`index.md` 与按需加载相关页面的方式，控制上下文成本并维持长期记忆

- 它强调 **增量式知识积累**，新资料进入后会持续强化已有知识网络，而不是形成孤立笔记
