## 定义

Tempo 是 Stripe 与 Paradigm 孵化的支付型 L1 区块链，面向稳定币结算与机器支付场景。

## 关键要点

- 目标是低成本、低延迟的稳定币支付基础设施

- 与 Stripe API 体系深度衔接

- 是 MPP 背后的链上结算层
