> **🔌** **定义**：Mano-CUA Skill 是 Mano-P 开源路线第一阶段率先开放的技能形态，面向 OpenClaw、Claude Code 等 Agent 工作流提供 GUI 操作能力。

### 关键要点

- 目标是让 Agent 在需要图形界面操作时无需人工接手，直接完成 CUA 任务闭环。

- 文章将其视为“Skill 先行，开箱即用”阶段的核心交付物之一。

- 与 mano-cua CLI、mano-client Python SDK 共同构成同一能力体系的不同接入方式。
