## 定义

Chrome DevTools MCP 是 Google Chrome 官方提供的 MCP Server，让 AI 助手直接连接用户当前正在使用的 Chrome 浏览器会话。

## 核心要点

- 复用已有登录态、当前页面和浏览器上下文，而不是新开隔离实例

- 适合读取页面内容、执行脚本、分析网络请求和调试 UI

- 与持久连接方案相比，官方实现更偏通用调试能力
