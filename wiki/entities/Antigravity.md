## 定义

Antigravity 是 Google 围绕 agentic coding 布局的编程 Agent 平台，用于承接其在开发者工作流中的 Harness 与执行层能力。

## 关键要点

- 它体现了 Google 试图补齐 **AI 编程 Agent** 与 **Harness 产品层** 的动作。

- 文中语境下，Antigravity 被拿来与 Claude Code 对比，暴露出 Google 在开发者采用率与产品完成度上的差距。

- 它不是单一模型能力的代名词，而是 Google 把模型、工具、执行流程和开发者入口整合起来的产品尝试。
