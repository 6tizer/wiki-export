## 定义

Open Computer Use 是一个面向 macOS 的开源电脑控制项目，把 Computer Use 能力封装为可通过 MCP 调用的服务，让 AI Agent 或 MCP 客户端能够直接操作桌面。

## 关键要点

- 它定位为 Codex Computer Use 的开源替代方案。

- 项目通过 MCP 暴露电脑操作能力，便于接入 Claude Code、Codex 等 Agent 工作流。

- 当前重点平台是 macOS，依赖系统的辅助功能与屏幕录制权限。

- 项目还提供安装到 Codex / Claude Code 的便捷命令，说明其目标就是进入真实的 Agent 工具链。
