## 定义

Icarus Plugin 是 Hermes 生态中的一个自记忆与自动训练插件，强调 Agent 在执行任务过程中积累经验、训练后继能力，并把能力演化成可持续继承的资产。

## 核心要点

- 将记忆积累与能力训练放在同一条演化链路里

- 支持 Agent 在干活过程中沉淀经验，而不是只保留即时上下文

- 体现出“Agent 带徒弟”式的继承思路，让能力不只停留在单次会话

- 适合与长期运行、持续改进型的 Agent 工作流结合
