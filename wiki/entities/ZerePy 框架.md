ZerePy 是 ZEREBRO 项目开发的 Python 模块化 AI Agent 框架，专注于创意内容生成（音乐、艺术、自主社交媒体发文），定位为去中心化 AI 基础协议。

## 核心特点

- 模块化设计，专注社交媒体 Agent 场景（X/Twitter 自主发推）

- 与 Eliza 框架有合作关系

- 局限：不支持原生 LLM 集成和内存管理（早期版本）

## Zerebro 生态

- **Zentients**：新型基础设施（多 Agent 协作、去中心化 GPU 租用、跨链互操作性）

- 2025 年 1 月市値翻倍（+99%），Mindshare 上升

## 与 Eliza 对比

- Eliza：全功能 Agent OS，Web3 原生，TypeScript

- ZerePy：模块化，创意内容专注，Python，社区文化基因更强
