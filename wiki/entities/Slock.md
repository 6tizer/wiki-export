## 定义

Slock 是 RC 创办的人与多 Agent 协作平台，目标是把人类成员与 Agent 放进同一协作环境中，实现沟通、任务分工、共享文档与长期记忆沉淀。

## 关键要点

- 面向 1-100 人的小团队或初创公司，而不只是单个全能 Agent 的使用场景

- 试图把“一个人 + 多个本地 agent session”的工作方式升级为“多人 + 多 Agent”的组织协作界面

- 核心能力围绕沟通、任务认领、信息共享与共享记忆展开

- 强调 diversity：支持不同模型和不同 Agent 协同，而不是绑定单一模型厂

- 本质上是在探索 agent-native team 的组织形态
