## 定义

Kimi Code 是月之暗面围绕 Kimi K2.6 提供的 AI 编程产品形态，面向真实开发任务提供聊天模式之外的代码执行、工程操作与长链路任务承接能力。

## 关键要点

- 它是 Kimi K2.6 面向生产级编码场景的主要入口之一，承接更完整的软件开发与终端操作任务。

- 从文章语境看，Kimi Code 不是单纯的聊天界面，而是与 agent mode、工具调用和持续执行能力绑定的编程工作台。

- 结合官方 CLI 仓库信息，它已经延伸为终端代理与 IDE 集成能力，而不是单一网页功能页。
