## 定义

PI Agent 是文章中李超重点提到的一个极简开源 Agent 项目，设计思路是尽可能压缩系统复杂度，只保留完成电脑任务所需的最小能力集合。

## 关键要点

- 它强调“减功能”而不是“加功能”，追求更可控、更听话、更容易定制。

- 核心组合被概括为：一个基础大模型 + 读文件 + 写文件 + 编辑文件 + 执行命令。

- 相比上下文和功能越来越重的通用 Agent，它更适合个人按自身工作方式深度改造。

- 在这篇对谈里，PI Agent 被当作“超级个体”自定义工作流的重要实验对象。
