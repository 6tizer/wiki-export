## 定义

Qwen Code 是阿里巴巴 Qwen 团队推出的开源终端 AI 编程 Agent，强调在本地终端、远程服务器与多模型环境中执行开发任务。

## 关键要点

- 定位接近 Claude Code、Gemini CLI 一类终端型 Coding Agent

- 支持本地终端工作流，也支持通过消息渠道远程触发任务

- 兼顾免费额度、开源协议与多模型兼容能力
