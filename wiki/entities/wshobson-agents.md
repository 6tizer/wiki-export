## 定义

wshobson/agents 是一个面向 Claude Code 的插件生态仓库，通过多插件、多角色和多技能包的方式，把单一编程助手扩展为可协作的开发团队。

## 关键要点

- 重点不在单个 Agent，而在插件化组织方式与按需加载机制

- 适合大型项目中的角色分工、工作流编排和工具组合

- 是 Claude Code 生态里较有代表性的第三方扩展系统
