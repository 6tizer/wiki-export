## 定义

Claude Desktop Buddy 是 Anthropic 提供的参考项目，用来演示如何通过蓝牙把 Claude Code / Claude Cowork 桌面端的运行状态、消息与审批事件同步到外部硬件设备。

## 关键要点

- 作为官方参考实现，它展示了硬件 Buddy 的完整交互链路，而不只是单一固件示例。

- 仓库同时给出 BLE 通信、JSON schema、folder push transport 与字符/GIF 宠物显示方案。

- 其定位更偏向 Maker 生态的开发样板，帮助开发者快速做出可视化、可交互的 Claude 外设。
