## 定义

OKX Agent Trade Kit 是 OKX 面向 Agent 场景推出的交易工具包，支持通过 MCP、CLI 或 Skills 方式把行情发现、下单与策略执行接入自动化工作流。

## 关键要点

- 把交易所操作封装成 Agent 可调用的工具集合

- 支持从研究、信号到执行的端到端链路

- 上线真实资金前需要充分模拟盘验证与风控约束
