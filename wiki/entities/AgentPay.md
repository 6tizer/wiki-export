## 定义

AgentPay 是一个面向 AI Agent 的 x402 支付网关与工具分发层，支持 Agent 使用 USDC 按次购买数据工具调用。

## 关键要点

- 提供 14 个可直接付费调用的加密数据工具

- 支持 Stellar 与 Base 两条网络的 USDC 结算，并提供预算控制会话

- 同时提供 Python SDK 与 MCP Server，降低 Agent 集成成本
