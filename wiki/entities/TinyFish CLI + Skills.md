## 定义

TinyFish CLI + Skills 是 TinyFish 面向终端型 Agent 提供的一套网页访问与自动化能力组合，把 CLI 执行层与 Skills 教学层打包，让 Claude Code、Codex、OpenClaw、Cline 等助手可以更低成本地访问网页、抓取数据并执行操作。

## 关键要点

- 它的核心形态不是传统 GUI 工具，而是可被 Agent 调用的命令行能力层

- “CLI + Skills” 组合同时覆盖执行与教学两个层面，降低不同 Agent 的接入门槛

- 公开的 Skills 仓库和免费额度，有利于快速试用与二次开发
