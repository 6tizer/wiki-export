## 定义

Everything Claude Code 是一个围绕 Claude Code 的开源配置仓库与方法集合，把开发者在真实项目中的规则、技能、钩子和子代理沉淀为可复用的工程化配置。

## 关键要点

- 它不是单一配置文件，而是由 **47 个 sub-agent、181 个 skill、79 个命令** 组成的完整工作流系统。

- 核心价值在于把反复口头说明的项目约束，转化为 Claude Code 的默认行为。

- 文章将其总结为 **Rules、Skills、Hooks、Agents** 四层结构。

- 强调 TDD 流程、规则约束与工具链集成，而不是只追求更强模型。

- 适合用来参考如何把 AI 编程从“对话式使用”升级为“流程化协作”。
