**定义**：Claw Code 是 Sigrid Jin 等人在 Claude Code 源码泄露事件后，通过 AI 编排与净室重写方式快速复现出的开源 Coding Agent 项目。

**关键要点**

- 目标是复现 Claude Code 的核心体验与工作流

- 通过多 Agent 协同与模型分工完成快速重建

- 因未直接复制泄露代码而具备独立创作属性

- 体现了编排层与工作流设计对 Coding Agent 的决定性作用
