## 定义

WSL2 是微软提供的 Windows Subsystem for Linux 第二代实现，让开发者可以在 Windows 内获得更接近原生 Linux 的运行环境，常被用来承载需要 Unix-like 环境的开发工具与 Agent。

## 关键要点

- 它是 Hermes Agent 在 Windows 上的推荐运行方式，而不是可选优化项。

- 它解决的是原生 Windows 与 Linux 工具链不兼容的问题。

- 在代理、Python、网络代理和文件权限场景里，WSL2 往往决定了安装是否顺利。
