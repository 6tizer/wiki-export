## 定义

Jina Reader 是一个开源的网页内容提取服务，通过 `https://r.jina.ai/URL` 的简单接口将任意网页清洗为干净的结构化文本，专门为 LLM 和 AI Agent 的网页阅读场景设计。

## 关键要点

- 通过 curl 即可调用，无需复杂配置

- 自动清洗网页中的广告、导航栏等噪音，提取核心正文

- 是 Agent Reach 底层使用的网页解析工具之一

- 免费开源，零 API 费用

- 适合作为 AI Agent 联网阅读能力的基础组件
