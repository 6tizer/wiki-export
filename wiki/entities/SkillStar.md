## 定义

SkillStar 是一个围绕 AI Agent CLI 的 Skill 管理产品，提供 GUI 与 CLI 双形态，强调项目级同步、技能组合、市场浏览与安全扫描。

## 关键要点

- 支持从 GitHub 或 [skills.sh](http://skills.sh/) 安装 Skill，并管理本地与项目级 Skill 集。

- 通过 Deck、Projects 和 Agent 链接机制，把多组 Skill 精准分发到不同项目与不同 Agent。

- 将 AI 翻译、摘要、安全扫描、市场推荐等能力纳入同一工具，体现出 Skill 管理层的产品化趋势。

- 文章线程里把它作为同类参考项目，说明这一赛道已有多种实现路径。
