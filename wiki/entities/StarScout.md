## 定义

StarScout 是卡内基梅隆大学团队提出的 GitHub 异常 Star 检测工具，用于识别僵尸账号、同步批量点星等行为，并对可疑刷星仓库进行规模化排查。

## 关键要点

- 它面向 GitHub Star 造假问题，核心价值是把原本依赖人工经验的异常识别转化为可系统化执行的检测流程。

- 文中提到该工具基于 2019–2024 年 GitHub 数据识别出大规模疑似假 Star 与刷星仓库。

- 它的意义不只在于抓作弊，还在于为平台治理、项目评估和研究提供了量化依据。
