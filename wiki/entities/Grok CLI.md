## 定义

Grok CLI 是一个由社区维护的终端编程 Agent，把 xAI 的 Grok 模型接入命令行工作流，支持交互式开发、批处理与多子代理协作。

## 关键要点

- 面向终端开发场景，强调代码执行、项目验证与实时搜索

- 支持子 Agent、文件 @ 引用与验证命令，向 Claude Code 体验靠拢

- 是 xAI 模型生态里较完整的开源 Coding Agent 实现
