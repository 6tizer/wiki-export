## 定义

Cloudflare /crawl 是 Cloudflare Browser Rendering 服务提供的整站抓取接口，只需给一个起始 URL，就能自动发现页面、渲染内容并输出 HTML、Markdown 或 JSON。

## 关键要点

- 内置链接发现、队列、渲染与输出格式转换

- 适合 RAG 建库、站点监控与批量网页采集

- 把全站爬取从基础设施问题压缩成一次 API 调用
