## 定义

Atlas 是 OpenAI 开发的浏览器产品，正在被整合进 Codex 平台，未来将成为支持持久登录的完整浏览器，使 Codex 能直接在浏览器内执行操作。

## 关键要点

- 是 OpenAI 自研的浏览器，正被并入 Codex 生态

- 未来目标是成为持久登录的完整浏览器，而非一次性浏览

- 这使 Codex 与 Claude Code / Cursor 在产品边界上产生最大差异——后者不自带浏览器

- 与 Computer Use 能力结合，意味着「任何能用浏览器完成的事都能被自动化」
