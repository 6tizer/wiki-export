> **🧭** **定义**：Mano-P 是明略科技推出的面向端侧设备的 GUI-VLA 智能体模型，强调纯视觉界面理解、本地运行、零上云与跨桌面软件操作能力。

### 关键要点

- 定位为可直接操作桌面软件、网页界面的纯视觉 GUI Agent，不依赖传统 API 对接或浏览器专用协议。

- 提供从 Skill、模型到训练方法的三阶段开源路线，兼顾即用性、可定制性与生态扩展。

- 支持在 Apple M4 设备上本地推理，突出隐私保护、离线执行与低门槛部署。

- 文章将其视为 GUI Agent 从“爪”走向“手”的代表性产品。
