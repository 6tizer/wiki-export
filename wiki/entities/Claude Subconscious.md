## 定义

Claude Subconscious 是 Letta AI 出品的开源插件，为 Claude Code 添加「**潜意识**」记忆层，实现跨会话的隐式学习和持久化记忆。

## 关键特点

- 跨会话记忆：Claude Code 越用越懂用户

- 「潜意识」定位强调隐式学习而非显式配置

- Letta AI 出品，技术可靠

- GitHub：[https://github.com/letta-ai/claude-subconscious](https://github.com/letta-ai/claude-subconscious)

- 与 Mem0、MemOS 的显式记忆管理形成互补

## 对比同类

- **Claude Subconscious**：隐式/自动学习

- **Mem0**：显式 Markdown 结构化

- **MemOS**：团队级可视化管理
