## 定义

Claude Managed Agents 是 Anthropic 推出的托管式 Agent 方案，将更多 Harness、状态与执行能力收拢到官方 API 与平台内部。

## 关键要点

- 代表一种 **托管式 Agent** 产品形态

- 优势是降低部署和运维门槛

- 风险是 Harness、状态与长期记忆更容易被平台锁定

- 在本文语境里，它被用作“记忆所有权让渡给模型提供商”的典型例子
