## 定义

free-claude-code 是一个把 Claude Code 请求转发到免费或低成本模型提供商的代理层项目，用来降低顶级编程 Agent 的使用门槛。

## 关键要点

- 支持把 Claude Code CLI 与多种替代推理后端连接起来

- 更适合本地任务和低成本实验，不等于完整复刻官方订阅体验

- 其价值在于把“高质量编程 Agent”从高订阅成本中部分剥离出来
