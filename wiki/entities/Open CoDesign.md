## 定义

Open CoDesign 是一个本地优先的开源 AI 设计桌面工具，定位为 Claude Design 的开源替代品，支持通过提示词快速生成原型、幻灯片、PDF 和各类 UI 设计。

## 关键要点

- 以本地优先和 BYOK 为核心，强调模型选择权、数据控制权与运行环境控制权回到用户手中

- 支持 OpenAI Chat、OpenAI Responses、Anthropic Messages 协议，也可接入 Ollama

- 可从 Claude Code 或 Codex 一键导入 API 配置，降低已有工作流迁移成本

- 内置 15 个示例与 12 个设计技能模块，覆盖落地页、仪表板、幻灯片、数据表格等常见场景

- 支持响应式预览、滑块调参、执行日志查看，以及 HTML、PDF、PPTX、ZIP、Markdown 导出
