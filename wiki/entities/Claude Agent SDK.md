## 定义

Claude Agent SDK 是 Anthropic 提供的 Agent 接入层，用来通过 SDK 或 CLI 方式驱动 Claude 的工具调用与会话执行能力。

## 关键要点

- 它提供了比直接拼接 API 调用更稳定的 Agent 接口层

- 在 OAuth 受限之后，SDK/CLI 路线成为继续使用 Claude Agent 能力的重要替代路径

- 其价值在于把模型能力包装成更接近“可编排执行单元”的开发接口
