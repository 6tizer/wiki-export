## 定义

TinyAgentOS 是一个面向消费级硬件的自托管 AI Agent 平台与 Agent OS，目标是把 Mac、树莓派、Orange Pi 等设备组织成分布式的 Agent 运行与算力环境。

## 核心要点

- 提供 Web Desktop、应用商店、Agent 部署、模型管理与系统监控等统一入口

- 强调 *framework-agnostic*，把记忆、文件、模型访问和配置从具体 Agent 框架中抽离出来

- 支持 Apple Silicon、NVIDIA、AMD、Rockchip NPU、树莓派和移动端等多类硬件

- 当前仍处于 pre-beta 阶段，GUI 尚未完全接通核心功能，首个 beta 计划指向 2026-04-21
