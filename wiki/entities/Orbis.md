## 定义

Orbis 是一个面向开发者与 AI Agent 的 API Marketplace，强调通过 x402 以 USDC 按调用付费获取 API 能力。

## 关键要点

- 将 API 市场与 Agent 按次付费能力结合，降低接入门槛

- 在讨论中被作为 x402 已有落地案例，用来说明该支付轨道并非只停留在概念层

- 强调“无 Key、无注册、无人类介入”的机器支付体验
