**定义**：MuleRun 是一款云端自进化个人 AI 平台，通过给每个用户分配独立虚拟机（7×24小时在线），解决了 OpenClaw 类本地 Agent 需要电脑一直开着、安装复杂的问题。

**核心架构**

基座 Agent + Knowledge + Skills + Runtime 四层架构，每用户分配独立云端虚拟机、专属文件系统，物理隔离。

**两种核心模式**

- **Computer**：持续在线，适合定时任务、持续监控

- **Super Agent**：一次性交付，适合 PPT、报告等具体交付物

**自进化机制**

- **个体层**：用得越久就越懂用户，记录职业、偏好、常用工具

- **群体层**：用户沉淀的有效解决方案注入公共知识网络，越多人用越强

**Heartbeat 机制**

每天主动向用户汇报完成情况，发现重复操作时主动建议自动化方案。

**与 OpenClaw 的对比**

- OpenClaw：本地自主方向，依赖本地设备，适合技术用户

- MuleRun：云端托管方向，开箱即用，适合非技术用户
