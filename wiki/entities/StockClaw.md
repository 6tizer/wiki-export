## 定义

StockClaw 是一个围绕持久化根 Agent 架构设计的股票研究与回测系统，按需调度专家 Agent 参与分析、回测与执行。

## 关键要点

- 根 Agent 持有主对话与状态，避免无序堆叠子 Agent

- 强调股票研究、Paper Trading 与可复现回测

- 适合把多角色分析工作流收束到单一调度中枢
