## 定义

OpenASE 是一个以工单驱动的软件工程自动化平台，通过工作流、状态流转和 Ticket Agent 在本地或远程机器上执行 AI 编码任务。

## 关键要点

- 以 ticket、workflow、status 为核心控制面，强调可追踪的异步执行

- 支持 Claude Code、Codex、Gemini CLI 等多种 Agent CLI

- 以单个 Go 二进制集成 API 服务、编排器与嵌入式 Web UI
