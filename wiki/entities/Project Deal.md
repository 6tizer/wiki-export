## 定义

Project Deal 是 Anthropic 设计的一项多 Agent 二手交易实验，让 Claude 代理真实员工在 Slack 市场中完成挂牌、报价、还价与成交，以验证 Agent 是否能在现实约束下完成端到端交易。

## 关键要点

- **真实交易闭环**：实验并非纯模拟，至少有一轮要求参与者按 Agent 谈好的结果在线下完成真实交割。

- **多模型对照**：通过 Opus 4.5 与 Haiku 4.5 的混合与分组运行，观察模型能力差异如何影响成交率与成交价格。

- **市场化验证**：它把 Agent 从任务执行推进到交易撮合，证明语言模型已能处理带有预算、偏好与谈判策略的市场互动。

- **边界清晰**：实验场景仍局限于办公室内部，尚未覆盖支付、物流、平台博弈与外部风控等真实商业变量。
