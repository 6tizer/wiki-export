## 定义

ai-goofish-monitor 是一个基于 Playwright + AI 的闲鱼（Goofish）二手商品监控工具，支持多任务并发、AI 智能分析与实时推送通知，提供完整的 Web UI 后台管理界面。

## 关键要点

- 使用 Playwright 进行浏览器自动化爬取，模拟真人操作以规避反爬机制

- 集成 GPT-4o 等多模态模型进行图文+卖家画像深度分析

- 支持自然语言描述需求自动生成监控任务

- 支持 Cron 定时调度、[ntfy.sh](http://ntfy.sh/) / 企业微信 / Bark 多渠道推送

- 提供 Docker 一键部署，降低使用门槛

- 项目 90% 以上代码由 AI 自动生成，作者强调 HITL（人类审查）的重要性

- GitHub 仓库：[https://github.com/dingyufei615/ai-goofish-monitor](https://github.com/dingyufei615/ai-goofish-monitor)
