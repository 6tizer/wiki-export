## 定义

MiMo-V2-Pro 是小米发布的为 Agent 时代设计的大语言模型，登顶 OpenRouter 排行榜，免费使用期内可大幅降低 Agent 运行成本。

## 关键特点

- 登顶 OpenRouter 排行，社区认可度高

- 小米开源，免费使用

- 为 Agent 场景优化，与 GLM-5-Turbo 形成竞争

- 通过 OpenRouter 接入，与 OpenClaw 兼容
