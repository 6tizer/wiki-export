Hyperliquid 是一条专为高性能金融交易设计的 Layer-1 区块链，以全链上订单簿和极低延迟著称，$HYPE 代币 2024 年 11 月发行，FDV 约 272 亿美元。

## 核心技术特性

- **全链上订单簿**：所有订单匹配在链上完成，透明可验证

- **高性能设计**：针对低延迟交易优化，原生支持永续合约和现货交易

- **HIP-1/HIP-2**：原生代币发行机制，允许项目方在 Hyperliquid 上发行代币

- **HyperEVM**：EVM 兼容层，扩展智能合约生态

## AI Agent 生态

Hyperliquid 的结构优势：AI Agent 可直接调用链上订单簿 API 进行交易，与高性能金融场景天然结合。

2024 年底涌现的 AI Agent 项目：

- **$VAPOR**：无代码 AI Agent 发射台，基于 elizaOS

- **$FARM**：链上 AI 生物游戏，各生物是自主 Agent

- **$LAUNCH**：AI Launchpad + 基础设施
