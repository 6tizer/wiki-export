## 定义

GhostTrack 是一款开源 OSINT（开源情报）命令行工具，支持 IP 地理定位、手机号码情报查询和社交媒体用户名追踪。项目使用 Python 编写，可在 Linux（Debian 系）和 Android Termux 上运行。

## 关键要点

- **GitHub 仓库**：[HunxByts/GhostTrack](https://github.com/HunxByts/GhostTrack)，11,300+ Stars，1,535 Forks

- **创建时间**：2023-04-15

- **语言**：Python

- **许可证**：未明确标注

- **三大功能模块**：

  - IP Tracker：输入 IP 地址，获取地理位置、ISP、国家/地区/城市

  - Phone Tracker：输入手机号，获取运营商、注册国家、线路类型、地理区域

  - Username Tracker：输入用户名，跨社交平台搜索该 handle

- **Combo 模式**：可配合 Seeker 工具捕获目标真实 IP

- **运行环境**：Linux (Debian-based) 和 Termux (Android)，无需桌面环境

- **安装极简**：clone 仓库 → pip install → 运行单个 Python 文件
