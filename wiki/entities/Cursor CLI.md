## 定义

Cursor CLI 是 Cursor 提供的终端侧 AI 编程入口，把模型调用、agent 执行、命令式交互和运行时配置整合到命令行工作流中。

## 关键要点

- 它强调“在终端里完成真实工作”，适合处理 SSH、服务器排障、脚本自动化和跨环境操作。

- 这次更新展示了 Cursor 正在围绕终端工作流补齐更细粒度的 agent 能力，而不只是提供一个聊天壳。

- 从产品定位看，Cursor CLI 更像 Coding Agent 的运行界面，而不只是 IDE 的附属工具。
