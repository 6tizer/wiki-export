## 定义

feature-dev 是 Claude Code 官方插件里的结构化开发工具，强调按步骤理解需求、扫描代码库、拆解实现路径，再推进新功能开发，适合接手陌生项目时使用。

## 关键要点

- 用流程化步骤帮助 Agent 先理解需求，再进入实现阶段

- 对陌生代码库尤其有价值，可降低直接改代码带来的上下文偏差

- 本质上把“先分析再编码”的方法论封装成了可执行插件
