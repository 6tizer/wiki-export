## 定义

MetaClaw 是一个让 Agent 从真实对话中提炼技能并持续进化的代理层框架，位于 Agent 与底层模型 API 之间。

## 关键要点

- 把失败轨迹转化为可复用技能，再在后续任务中注入

- 同时结合长期记忆与可选的后台强化学习更新

- 目标是在不改业务代码的前提下，让 Agent 越用越聪明
