## 定义

Claude Code 是 Anthropic 推出的**命令行 AI 编程助手**，直接运行在终端里。不需要打开 IDE 或安装插件，可自主执行 Shell 命令、递归修改多个文件、修复 Bug——是真正的 Agent 模式而非代码补全。

## 当前规模（2026 年 3 月）

- GitHub Stars：40,000+

- 占全球 GitHub 提交的约 **4%**，日提交量超过 13.5 万次

- 创作者 Boris 公开表示上个月 100% 的代码贡献都由 Claude Code 自己写

## 典型用法

- 与 Cursor 配合使用：Cursor 日常编写 + Claude Code 处理复杂任务

- 国内用户可通过阿里百炼 API 代替官方订阅，支持 Kimi、精准 GLM、通义千问等模型

## 同类工具对比

| 工具 | 形态 | 特点 |

| --- | --- | --- |

| Claude Code | 终端 CLI | 深度 Agent，自主执行命令 |

| Cursor | IDE | 日常编写+补全，上下文感知强 |

| Windsurf | IDE | 轻量易用，适合入门 |

| Cline/OpenCode | 插件/终端 | 开源替代 |
