## 定义

Hermes Alpha 是一个把 Nous Research 的 Hermes Agent 云端化部署的实验性项目，目标是在几乎没有预置基础设施的前提下，让 Agent 自行搭建并持续改进自己的工作系统。

## 核心要点

- 用 [Fly.io](http://fly.io/)、Web 终端和持久化卷把 Hermes 变成可持续运行的云端 Agent

- 核心结构是 Overseer 与 Hunter 的双 Agent 分工：一个负责改进系统，一个负责执行具体任务

- 强调“少脚手架”路线：先给任务与身份文档，再让 Agent 自建流程、工具和策略

- 适合作为 Hermes 从本地试验走向长期运行的部署样板
