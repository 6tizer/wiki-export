## 定义

Skill Factory 是一个面向 Hermes Agent 的元技能插件，用来观察用户与 Agent 的重复工作流，并自动把这些流程沉淀成可复用的 `SKILL.md` 或插件命令。

## 核心要点

- 监控会话中的重复操作模式，识别值得固化的流程

- 可自动生成 `SKILL.md`，帮助 Agent 以后按相同方法复用经验

- 也可同时生成插件命令，把常见流程变成可直接调用的工具入口

- 本质上是在把“临时完成一次”转成“可长期复用的方法资产”
