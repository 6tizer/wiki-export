## 定义

Higgsfield Marketing Studio 是一个一体化 AI 营销平台，集成了 Hermes Agent（广告研究与策略引擎）和 Seedance 2.0（AI 视频生成引擎），为 solo founder 提供从竞品分析、创意生成到视频渲染的全自动广告生产管线。

## 关键要点

- **一键启动**：只需粘贴产品 URL / 官网 / App Store 页面，平台自动完成从研究到渲染的全流程

- **AI Persona 锁定**：支持上传真人面孔或生成虚拟代言人，保持跨视频一致性（同一张脸、同一种声音）

- **Hermes Agent 内置**：自动抓取 Meta Ads Library 和 TikTok Creative Center 热门 hooks，逆向分析竞品广告策略

- **Seedance 2.0 渲染**：自动生成带字幕、节奏、hooks 的广告视频，支持 9:16 / 1:1 / 16:9 多格式

- **日产 500+ 广告素材**：可直接投放到 Meta Ads Manager 和 TikTok Ads

- **闭环优化**：Hermes Agent 持续追踪哪些 hooks 表现好，自动加大赢家、淘汰输家

## 核心组件

- **Hermes Agent**（Higgsfield 内部版本）：广告研究与策略 AI，负责竞品分析、hook 提取、brief 生成

- **Seedance 2.0**：视频生成引擎，负责将策略转化为可投放的广告视频

- 注意：Higgsfield 内部的 Hermes Agent 与 Nous Research 开源的 Hermes Agent 是不同项目
