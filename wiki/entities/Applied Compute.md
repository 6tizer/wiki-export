## 定义

Applied Compute 是一家由前 OpenAI 研究员创办的企业 AI 公司，总融资 1.6 亿美元。公司核心产品为 AC Context Engine，专注于为企业 agent 提供上下文基础设施，将内部文档、工单历史和 agent 运行轨迹编译为可检索的知识库（Contextbase）。

## 关键要点

- 创始团队来自 OpenAI 研究部门

- 总融资规模 1.6 亿美元

- 核心产品：AC Context Engine —— 企业级 agent 上下文引擎

- 目标：降低 agent 推理成本，同时保持或提升任务准确率
