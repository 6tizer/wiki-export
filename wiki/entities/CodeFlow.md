### 定义

CodeFlow 是一个以单文件 HTML 形式分发的代码仓库分析工具，可在浏览器中直接读取 GitHub 仓库或本地文件夹，生成交互式架构图并做风险分析。

### 关键要点

- 无需安装和登录，主要在浏览器本地完成分析与可视化。

- 支持依赖图、爆炸半径、代码所有权、安全扫描、模式识别、健康评分、PR 影响分析等能力。

- 可用于公共仓库、私有仓库和本地文件夹分析。

- 通过 GitHub API、Tree-sitter WASM、D3.js、React 等组合实现静态分析与可视化。
