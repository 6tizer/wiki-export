## 一句话定义

FinceptTerminal 是一个以 C++20 + Qt 原生桌面架构构建的开源金融智能终端，试图把数据分析、AI Agent、交易接入与可视化工作流整合到一个本地优先的研究环境中。

## 关键要点

- 它不是 Electron 壳或纯 Python 脚本，而是以原生桌面性能为底座，再嵌入 Python 金融分析生态。

- 项目整合 100+ 数据源、本地 SQLite 缓存、QuantLib 计算能力与多券商/交易所接口，目标是形成可分析、可交易、可离线的金融工作台。

- 37 个 AI Agent 覆盖投资、宏观、地缘政治与研究任务，体现其产品定位更接近“金融分析助手”而非单一行情面板。

- 对 Ollama、LM Studio 及多家云模型的支持，使敏感数据可以在本地优先的前提下接入 LLM 能力。

- Node Editor、MCP 集成与 AI Quant Lab 说明它的真正野心是把金融分析流程 agent 化、工作流化。
