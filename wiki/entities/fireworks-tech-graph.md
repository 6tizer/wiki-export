## 定义

fireworks-tech-graph 是一个面向 Claude Skills 生态的架构图生成工具，允许用户用自然语言描述系统结构，并输出可编辑的 SVG 与可直接使用的 PNG 技术图表。

## 关键要点

- 支持自然语言生成架构图、数据流图、流程图、序列图、思维导图等 8 类图表

- 内置 5 种视觉风格，适合 README、Wiki、演示文稿与技术文章等场景

- 针对 AI / Agent 领域内置了 Agentic RAG、Mem0 记忆层、多 Agent 协作、Tool Call Flow 等常见模式

- 既适合作为 Claude Code 的出图技能，也适合作为技术内容生产中的可视化组件
