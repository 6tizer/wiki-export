## 定义

Agent Package Manager（APM）是微软提出的 Agent 依赖管理工具，试图像 npm 或 pip 一样标准化 Agent 配置、技能包与环境复现。

## 关键要点

- 用声明式配置文件管理 Agent 依赖，而不是手动复制提示词和工具集

- 目标是让团队在 clone 项目后快速复现同一套 Agent 工作环境

- 代表 Agent 工具链从“临时手工配置”走向“可安装、可版本化、可分发”的基础设施化
