## 定义

Claude Token Efficient 是一种通过在仓库中放置精简型 [CLAUDE.md](concepts/CLAUDE.md.md) 规则文件来强制代理保持简洁输出的工具/配置方案，目标是在不改代码的前提下减少回复冗余和 token 开销。

## 档案要点

- 它强调“零代码改动”的接入方式，核心是通过项目规则约束回答风格。

- 相比 Caveman Mode 的更强风格化压缩，它更像是工程化的简洁输出规范。

- 适合想快速降低解释性废话、又不想调整复杂工作流的 Claude Code 使用场景。
