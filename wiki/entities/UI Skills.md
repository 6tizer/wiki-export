## 定义

UI Skills（[ui-skills.com](http://ui-skills.com/)）是一个开源的 UI/设计相关 Skill 文件合集平台，专为 AI 编程工作流设计。收录了大量面向前端开发和设计工程的 skill 文件，涵盖无障碍合规、设计探索、框架最佳实践等场景。GitHub 约 1.4k stars。

**别名**：[ui-skills.com](http://ui-skills.com/)

## 关键要点

- 收录多个领域的 UI skill 文件：设计实验（design-lab）、WCAG 无障碍检查（contrast-checker、link-purpose、refactor、use-of-color）、框架专用 skill（Nuxt、Pinia、UnoCSS、Vite、VitePress、Vitest 等）

- 支持 local skill 和 registry skill 两种模式

- 由社区贡献，包含 antfu、AccessLint 等知名开发者/组织的 skill

- 面向「设计工程」（Design Engineering）工作流，帮助 AI 编码助手在 UI 开发中遵循最佳实践
