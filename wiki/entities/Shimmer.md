## 定义

Shimmer 是 Poolside 推出的云端 Agent 开发环境产品，支持在沙箱 VM 中运行 AI Agent，可导入已有代码仓库、推送至 GitHub，快速构建 Web 应用、API 和 CLI 工具。

官网：[https://shimmer.run/](https://shimmer.run/) / [https://shimmer.poolside.ai/](https://shimmer.poolside.ai/)

## 关键要点

- **产品定位**：All-in-one 的 Agent 云开发环境，类似于 Poolside 版的云端 IDE + Agent 运行时

- **核心功能**：秒级启动 Agent 沙箱、导入 GitHub 仓库、推送代码变更

- **与 pool 的关系**：pool 是轻量终端 UI，Shimmer 是功能更完整的云端环境；两者共享同一个 agent harness（ACP 环境）

- **目标用户**：需要快速迭代 Web 应用和 API 的开发者

- **状态**：Research Preview 阶段
