### 定义

Open Lovable 是 Firecrawl 团队开源的一个 AI 前端生成项目，目标是把任意网站快速复刻为可运行的 React 应用。

### 关键要点

- 重点能力是把现有网页“克隆”为较高还原度的 React 前端，而不是只生成静态截图。

- 支持接入多种模型，包括 OpenAI、Anthropic、Gemini、Grok 等，模型可以按需切换。

- 依赖 Firecrawl 对网页结构进行抓取与解析，从而提升布局、样式和交互还原的一致性。

- 支持本地调试与部署，适合把灵感页、竞品页或参考页快速转成可迭代原型。

- 项目采用 MIT 协议开源，但实际运行通常仍需要配置 Firecrawl 与模型服务的 API Key。
