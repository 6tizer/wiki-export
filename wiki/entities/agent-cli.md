## 定义

agent-cli 是 Nunchi 团队开源的交易 Agent 命令行工具，面向 Hyperliquid 与 YEX 场景，支持策略执行、MCP 接入与 OpenClaw / Claude Code 集成。

## 关键要点

- 内置做市、动量、套利与 LLM 驱动策略

- 可作为 MCP Server 与 Agent 工具后端使用

- 是链上交易工具向 Agent 原生接口演进的典型产品
