## 定义

字节跳动发布的Coze新版本（其 Agent 组网能力也常被概括为“Coze 2.5 Agent 网络”），引入独立身份Agent网络的概念，每个Agent拥有专属邮箱，能相互交流并独立处理任务。

## 主要特性

- **独立身份**：每个Agent有专属邮箱，身份独立

- **Agent网络**：Agent间可相互通信和协作

- 提供：计算环境、工具链、记忆系统

- 在此次更新中进一步强化了编程、工作流、视频生成、日程与共享文件等多模态任务能力

- Agent可在网络中自主探索和学习

## 重要意义

- 体现了Agent从「工具」升级为「网络中的独立节点」的趋势，与Agent身份基础设施全球发展趋势同步

- 相比 OpenClaw 更偏向 Agent 的身份层与社交协作网络，而不是工具执行框架

- 相比 Hermes 这类单 Agent 自进化路径，Coze 2.5 更强调多 Agent 组网与协作

## 行业背景

- Auth0、Okta 等安全与身份厂商也在推进 Agent 身份基础设施，说明“独立身份 + 通信能力”正在成为 Multi-Agent 生态的底层前提
