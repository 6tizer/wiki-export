## 定义

CC Switch 是一个用于管理 Claude Code 配置和环境变量的工具。它适合需要切换多套配置的使用场景，让不同工作流可以按需启用不同参数，而不必频繁改动 shell 配置文件。

## 关键要点

- 可作为 Claude Code 环境变量的集中配置入口

- 适合管理多套运行配置，例如是否开启无闪烁渲染

- 相比直接修改 `.zshrc` 或 `.bashrc`，切换更灵活

- 在团队或多项目环境里，有利于把终端 Agent 的运行参数做成可复用预设
