## 定义

cursor/cookbook 是 Cursor 官方在 GitHub 上开源的示例代码仓库，包含使用 Cursor SDK 构建 Agent 应用的完整示例项目，包括 CLI 工具、看板应用和原型设计工具等。

## 关键要点

- **仓库地址**：[https://github.com/cursor/cookbook](https://github.com/cursor/cookbook)

- **语言**：TypeScript

- **Stars**：1297+（截至 2026-04-30）

- **创建时间**：2026-04-27

- **包含示例**：

  - **Quickstart**：最小化 Node.js 示例，创建本地 Agent 并流式输出

  - **Prototyping tool**：Web 应用，在沙箱云环境中启动 Agent 来搭建新项目

  - **Kanban board**：看板应用，按状态/仓库分组查看 Cursor Cloud Agent

  - **Coding agent CLI**：命令行界面，从终端启动 Cursor Agent

## 关联概念

- Cursor SDK：cookbook 是 SDK 的官方示例集

- Agent Harness：示例展示了如何复用 Cursor 的 Harness 层
