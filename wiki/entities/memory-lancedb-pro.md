## 定义

memory-lancedb-pro 是 OpenClaw 生态中的长期记忆插件，使用 LanceDB、混合检索和重排机制提升复杂任务下的记忆召回质量。

## 关键要点

- 它解决默认 Markdown 记忆方案在长期使用中的召回漂移和噪声堆积问题

- 支持多 Scope 隔离，适合多 Agent 或多项目场景

- 更适合结构化、长期型任务，而不是开放式闲聊
