## 定义

ClawHub 是 OpenClaw 的**官方技能注册中心（Skill Marketplace）**，类似于 App Store，集中托管和分发社区贡献的 OpenClaw 技能（Skills）和 Agent 模板。

## 关键数据

- 技能数量：13,700+（截至 2026 年 4 月）

- 社区贡献模板：187+（awesome-openclaw-agents 仓库）

- awesome 列表过滤掉约 2,748 个技能（含可疑账号、加密货币类、重复和高风险）

- 安装方式：`npx clawhub@latest install <skill-slug>`，或直接把 GitHub 链接发给 OpenClaw 助手自动安装

- 优先级：工作区 > 本地 > 内置

- GitHub: [https://github.com/openclaw/clawhub](https://github.com/openclaw/clawhub)

- 官网: [https://clawhub.com](https://clawhub.com/)

## 安全问题

Cisco 研究发现 ClawHub 插件市场中约 **20%** 的插件存在恶意行为，建议用户在安装前审查插件代码。
