## 定义

codex-plugin-cc 是 OpenAI 官方发布的 Claude Code 插件，让开发者直接在 Claude Code 工作流中调用 Codex 做审查、救场与后台任务执行。

## 关键要点

- 用插件方式把 Codex 接入 Claude Code，而不是切换到另一套主工作流

- 提供 review、adversarial review、rescue 等命令

- 体现 AI 编程工具从“抢用户”转向“嵌入对方工作流”的策略变化
