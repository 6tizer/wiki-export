## 定义

OpenAI Agents SDK 是 OpenAI 发布的轻量级多 Agent 工作流框架，用于把工具调用、技能路由与工程规范结合进可维护的开发流程。

## 关键要点

- 强调少原语、快上手与和 Codex 工具链的原生配合

- 通过 [AGENTS.md](http://agents.md/)、skills 与脚本把隐性规范显式化

- 适合 OSS 维护、工程自动化与仓库内 Agent 协作场景
