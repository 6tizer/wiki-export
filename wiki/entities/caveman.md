## 定义

caveman 是由 Julius Brussee 发布的开源 skill / plugin 项目，用来把 **Claude Code** 及其他 Coding Agent 的回复切换到更短、更直接的风格，以减少输出 token，并提供统一的安装、切换与扩展机制。

## 档案要点

- GitHub 仓库：JuliusBrussee/caveman

- 形态上既是一个可安装 repo，也是 Caveman Mode 的工程化封装

- 支持 Claude Code、Codex、Gemini CLI、Cursor、Windsurf、Copilot 等多种代理环境

- 除基础 caveman 模式外，还扩展了 commit、review、compress 等配套能力
