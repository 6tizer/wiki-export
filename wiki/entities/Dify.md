## 定义

Dify 是一个开源的 AI 应用开发平台，v1.0 引入插件化架构，支持通过插件市场组合模型、工具和 Agent 策略来构建 AI 应用。

## 核心要点

- 插件化架构：模型、工具、Agent 策略解耦为独立插件，支持热插拔

- Marketplace：中央插件市场，120+ 插件（模型 + 工具）

- Agent 节点：工作流中的智能决策中心，支持 ReAct / Function Calling

- 多模态交互：原生支持图像生成、语音交互

- 生态连接器：与 Slack 等外部平台无缝对接
