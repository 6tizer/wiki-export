## 定义

ALIVE 是文中作者正在推进的一个 **structured context substrate** 项目，主打 file-native、agent-agnostic 与零基础设施依赖，试图把长期可累积的上下文做成 Agent 可直接工作的底层载体。

## 关键要点

- 强调 plain files 是上下文与记忆的真实来源，而不是附属缓存

- 支持 portable context containers，文中将其称为 walnuts

- 面向连续运行的 Agent 场景，强调跨会话复利而非单次召回

- 可运行在 Hermes Agent 与 Claude Code 等不同执行层之上
