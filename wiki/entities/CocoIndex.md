## 定义

CocoIndex 是一个面向 AI 工程师与 Agent 构建者的增量数据管线引擎，旨在把代码库、文档、会话、事件流等来源持续转化为 Agent 可用的最新上下文、记忆与检索基础设施。

## 关键要点

- 产品定位不是单点检索工具，而是长期运行 Agent 的数据底座

- 强调增量处理、状态驱动与持续刷新，避免每次全量重算

- 面向 coding intelligence、RAG、记忆系统、知识图谱等典型 Agent 场景

- 价值主张在于让工程团队把精力放在管线逻辑本身，而不是大量脚手架与维护工作
