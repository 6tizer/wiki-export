## 定义

CodePilot 是围绕 Claude Code 构建的桌面客户端产品，主打把长期使用 Agent 的基础设施建设做成可感知的用户体验。

## 核心要点

- 以桌面 GUI 承接 Claude Code 的终端能力

- 用宠物系统把记忆、工具、技能等隐性能力转成可见成长反馈

- 重点不只是提效，还在于提升用户持续投入 Agent 生态的意愿
