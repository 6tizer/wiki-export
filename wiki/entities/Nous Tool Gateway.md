## 定义

Nous Tool Gateway 是 Nous Research 在 Nous Portal 中提供的托管工具网关，为付费订阅用户在 Hermes Agent 中统一接入网页抓取、浏览器自动化、图像生成、云终端和文本转语音等第三方工具，无需分别管理独立 API Key。

## 关键要点

- 通过 Nous Portal 统一完成订阅、鉴权与计费，把多工具接入收敛到一个入口。

- 在 Hermes Agent 的模型配置流程中即可启用，定位是执行层能力的统一网关，而不是单一工具插件。

- 首批支持网页抓取、浏览器自动化、图像生成、云终端后端和文本转语音，覆盖 Agent 常见外部执行需求。

- 工具网关与 300+ 模型订阅一起提供，强调的是“统一可用性”而不只是模型聚合。
