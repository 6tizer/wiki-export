## 定义

Claude API 中转站是指在官方服务受限地区，由第三方代理和账号体系向用户转售或转发 Claude 能力的接入模式。

## 关键要点

- 解决访问、支付和注册门槛

- 风险集中在稳定性、真实性和合规性

- 常见于开发者和重度用户的替代接入链路
