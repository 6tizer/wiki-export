## 定义

CyberMolt 是一个预装币安官方技能的 AI 交易 Agent 项目，主打“领养即用”的低门槛加密交易辅助体验。

## 关键要点

- 构建在 OpenClaw 生态之上，强调零配置上手

- 把行情分析、链上数据与交易辅助能力预先封装好

- 适合想快速试验 Agent 化交易而不想手动接 API 的用户
