## 定义

AutoGen（现称 AG2）是 Microsoft 开发的多 Agent 对话框架，支持多个 AI Agent 相互对话协作完成任务，具备强可观测性和代码生成能力。由 Chi Wang、Qingyun Wu 等微软研究员主导开发。

**GitHub**：microsoft/autogen | 核心团队：@Chi_Wang_、@qingyun_wu

## 核心特点

- **多 Agent 对话系统**：多个专职 Agent 可相互交互，支持复杂协商流程

- **强可观测性**：内置对话追踪和调试能力

- **代码生成**：擅长多 Agent 协作完成代码生成和审查任务

- **企业背书**：微软官方维护，适合企业生产环境

## 与 LangGraph 的差异

AutoGen 侧重多 Agent 自由对话协作；LangGraph 侧重有状态图结构工作流编排，控制流更明确。
