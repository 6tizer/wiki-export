## 定义

Page Agent 是阿里开源的网页内 GUI Agent 库，可直接嵌入 Web 应用，让用户通过自然语言驱动页面中的界面操作。

## 关键要点

- 纯前端 JavaScript 方案，不依赖浏览器插件或无头浏览器

- 通过 DOM 分析而非截图理解页面结构

- 适合把 AI Copilot 直接嵌入自有 Web 产品
