## 定义

Telegram 是一个以 Bot API、开放平台能力和即时通讯为核心的平台型产品。在本文语境中，它不是普通聊天工具，而是 Agent 大规模部署、分发与交互的基础入口。

## 关键要点

- 以对话框承接意图输入，天然适合 Agent 交互

- Bot 创建门槛低，API 完整，适合快速接入各类 Agent

- 支持 Bot 管理 Bot、Bot 与 Bot 对话，推动多 Agent 网络形成

- 与 TON 生态联动后，进一步补足身份、支付与可信执行能力
