## 定义

Architecture Diagram Generator 是一个面向 Claude Skills 生态的开源架构图生成工具，能够把自然语言形式的系统描述转换为可直接浏览、分享和展示的 HTML 架构图。

## 关键要点

- 它的定位更接近“架构表达加速器”，而不是传统可自由拖拽的绘图软件。

- 输出是独立 HTML 文件，适合快速嵌入文档、发给同事预览或托管到静态页面。

- 典型输入既可以来自人工撰写的系统说明，也可以来自 Cursor、Claude Code 对代码库的结构化分析结果。

- 价值重点在于降低沟通成本，让架构可视化更早进入开发流程，而不是等到项目后期再补文档。
