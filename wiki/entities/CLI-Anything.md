## 定义

CLI-Anything 是香港大学数据科学实验室开源的工具，能通过**一条命令**将任意软件自动转化为 AI Agent 原生工具，彻底取代脆弱的 UI 自动化方案。

## 关键特点

- 自动生成完整 CLI 接口，支持真实软件操作

- 绕过 Playwright 等 UI 自动化的不稳定性问题

- 目标：让所有有代码库的软件成为 Agent 原生工具

- 计划与其他 Agent 框架集成

- 是 A2A 通信标准化的重要基础设施
