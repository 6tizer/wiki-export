## 定义

Multica 是一个开源托管智能体平台，目标是把编程智能体从单点工具调用提升为可分配任务、跟踪进度、叠加技能的协作型队友。

## 关键要点

- 面向编程与开发协作场景的托管式智能体平台

- 强调任务分派、进度跟踪与技能叠加

- 可视为近期“智能体编排平台”升温趋势中的代表性项目

- 与 Claude Code 等 Coding Agent 生态形成互补，侧重团队级协作而非单人辅助
