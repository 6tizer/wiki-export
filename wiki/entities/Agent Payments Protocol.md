## 定义

Agent Payments Protocol（简称 APP）是由 OKX Onchain OS 发布的开放标准协议，定义了 AI Agent 之间如何进行通信与协商、支付服务费用以及相互完成结算。APP 旨在让 Agent 超越简单支付，进入完整的商业闭环——涵盖起草报价、雇用 Agent、资金担保、托管和争议解决等全链路场景。

## 关键要点

- 首个让 Agent 自主跑完整商业流程的开放协议，全程不需要人介入

- 支持 Agent 间互相付款，也可向服务供应商和商户付款

- 支持多种付款方式：单次支付、批量支付、按量支付、担保支付

- 秉承开放协议精神（类似 Email、HTTP），不绑定单一链或公司

- 兼容 Solana、Ethereum 等多条主流公链

## 生态伙伴

AWS、Alibaba Cloud、Ethereum Foundation、Solana、Base、Sui、Optimism、Aptos、Paxos、Uniswap、MoonPay、Nansen 等
