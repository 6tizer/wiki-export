## 定义

Claude Code Desktop 是 Anthropic 为 Claude Code 推出的图形化桌面客户端，将原本偏命令行的编程 Agent 体验迁移到桌面端，并补齐多任务并行、可视化审查、应用预览与插件市场等交互能力。

## 关键要点

- 基于 [Git Worktree](concepts/Git Worktree.md) 做会话隔离，每个任务自动持有独立代码副本

- 支持在同一仓库中并行推进多条任务线，合并前互不干扰

- 提供可视化 Diff 审查、应用预览与更友好的桌面交互

- 新增云端自动化能力，使 Coding Agent 从单次会话走向持续运行工作流
