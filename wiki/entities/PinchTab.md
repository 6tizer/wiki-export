## 定义

PinchTab 是一个轻量级浏览器控制服务器，用单个小体积二进制文件向 Agent 暴露 HTTP API，使其能以更低 token 成本操控浏览器。

## 关键要点

- 通过 HTTP API 控制 Chrome，而非依赖完整 Playwright 工具链

- 以 Accessibility Tree 代替截图作为主要页面表示

- 适合对 token 成本和部署复杂度敏感的 Agent 浏览器工作流
