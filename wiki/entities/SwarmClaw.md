## 定义

SwarmClaw 是一个围绕 OpenClaw 生态构建的多 Agent 编排与管理平台，用可视化面板统一纳管多个 Agent、Gateway 与工作流。

## 关键要点

- 提供跨机器、跨 Gateway 的多 Agent 管理界面

- 通过 LangGraph 引入有状态工作流编排能力

- 更像 OpenClaw 团队化运维层，而不是单一 Agent 本体
