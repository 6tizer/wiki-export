## 定义

Always-On Memory Agent 是 Google 开源的一种持续运行型记忆系统，用轻量模型持续摄取、整合和查询长期记忆。

## 关键要点

- 不只是存信息，还会周期性整合信息之间的关系

- 采用轻量模型与 SQLite 等简单底座降低运行成本

- 适合个人助手、小型项目与成本优先的长期记忆场景
