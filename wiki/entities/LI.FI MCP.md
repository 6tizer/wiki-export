## 定义

[LI.FI](http://li.fi/) MCP 是 [LI.FI](http://li.fi/) 提供的 MCP 接口层，让 AI Agent 通过统一工具调用方式完成跨链报价与交易执行。

## 关键要点

- 把复杂的跨链路由和交易能力封装成 Agent 可调用工具

- 适合多链资产调度、自动交易与 Agent 化 DeFi 工作流

- 在 Agent + Web3 场景中充当执行胶水层
