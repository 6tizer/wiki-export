## 定义

Context Hub 是一个面向 Coding Agent 的运行时文档获取工具，通过 CLI 在生成代码前拉取最新 API 文档，减少模型使用过时接口的情况。

## 关键要点

- 让 Agent 在运行时读取最新文档而不是依赖训练快照

- 支持搜索、按语言过滤和文档获取

- 适合 Cursor、Claude Code 等编程助手接入
