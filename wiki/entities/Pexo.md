**定义**：Pexo 是一个为 AI Agent 原生设计的 AI 视频生成平台，通过 Skill 的形式集成到 OpenClaw、Claude Code 等 Agent 中，用户只需一句话即可完成视频创作。

**核心设计理念**

- **天然面向 Agent**：核心交互只有对话框，几乎没有 GUI

- **模型调度层**：整合 Seedance、Sora、Kling 等 SOTA 级模型，自动判断最适合的模型

- **分层 Skill 架构**：pexo-agent（一句话全自动）、分镜和提示词工具、细粒度控制

**能力范围**

- 针对 Agent 调用设计，支持 OpenClaw、Claude Code、自动化脚本

- 支持人物一致性：提取照片特征生成卡通形象，不同场景保持角色统一

- 支持飞书、WhatsApp、Telegram、Discord 等多种 IM 平台
