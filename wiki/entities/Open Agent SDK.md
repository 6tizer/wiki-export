## 定义

Open Agent SDK 是一个兼容 claude-agent-sdk 接口的开源替代方案，目标是把 Agent 循环能力从闭源本地进程中解耦出来，支持多模型与云端部署。

## 关键要点

- 以 in-process 方式运行，不依赖本地 CLI 子进程

- 兼容 OpenAI 风格接口，方便接入 OpenRouter 等多种模型后端

- 代表 Agent SDK 从黑盒工具走向可读、可改、可多语言移植的趋势
