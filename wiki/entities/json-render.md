## 定义

json-render 是一个 JSON 驱动的 UI 渲染引擎/实体项目，用于把结构化描述转换成实际界面。在本文语境里，它被列为已与 A2UI 0.9 集成的生态组件之一。

## 关键要点

- 它强调以结构化数据描述界面，而不是直接手写前端代码。

- 与 A2UI 的结合说明 Agent 输出的界面描述可以更顺畅地接入现有渲染层。

- 这类引擎的价值在于把协议层与具体 UI 组件实现解耦。
