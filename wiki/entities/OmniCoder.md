## 定义

OmniCoder 是 Tesslate 推出的本地化 coding agent 模型系列，主打通过高质量 agentic 编程轨迹蒸馏，把顶级模型的工程行为压缩到消费级显卡可运行的小模型中。

## 关键要点

- 代表版本为 OmniCoder-9B 与 OmniCoder-2-9B

- 强调工具调用、终端操作、差量编辑与错误自恢复

- 定位不是代码补全，而是可执行多步任务的 coding agent 模型
