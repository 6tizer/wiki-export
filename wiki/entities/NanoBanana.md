## 定义

NanoBanana 是文中提到的图片生成 API / 工具，被作者设想封装进 Agent 的图片生成 Skill，用来覆盖公众号封面图、小红书封面图、PPT 配图等多个场景。

## 关键要点

- 更适合作为一个上层图片生成 Skill 的底层能力，而不是为每种图片需求各建一个独立 Skill

- 体现了“同类需求合并、在 Skill 内部分流”的设计思路

- 在内容工作流里属于可复用的生成式能力模块
