## 定义

Floatboat 是 FloatBoat 团队构建的 Agent 工作站客户端，为 Agent 提供本地运行空间、工具访问能力与持续在线的行动环境。

## 关键要点

- 可在用户电脑或主机上为多个 Agent 提供独立行动空间。

- 让 Agent 直接使用本机可见信息与可用工具，降低任务交接成本。

- 与 FloatIM 构成互补关系：Floatboat 负责运行环境，FloatIM 负责网络接入与协作。

- 承载团队在文件管理器、AI 浏览器、技能市场等体验层上的产品设计探索。
