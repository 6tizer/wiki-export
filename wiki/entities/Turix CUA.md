## 定义

Turix CUA 是一个开源的 Computer Use Agent 产品，可独立以桌面应用运行，也可作为 Skill 挂载到其他主 Agent 上，让 AI 通过视觉识别与鼠标键盘操作去使用本地 App。

## 关键要点

- 它试图把微信、QQ 音乐等缺少开放 API 或 CLI 的封闭应用，转化为 Agent 可调用的执行能力

- 既支持桌面版开箱即用，也支持作为子智能体接入 OpenClaw、Hermes Agent、Claude Code、Codex 等 Agent 工作流

- 支持自定义视觉模型配置，同时提供官方的 turix-brain 与 turix-actor 组合

- 相比传统脚本式自动化，它更依赖自然语言指令与视觉反馈来完成 GUI 操作
