## 定义

create-agent-tui 是 OpenRouter 官方推出的一个 Skill，用于快速搭建自定义 Agent Harness 及终端 UI（TUI）。开发者可通过交互式向导选择 4 种不同的界面风格，并启用数十种可选功能。

别名：无

## 关键要点

- 基于 OpenRouter Agent SDK 构建，提供类型安全的 Tool Calling 和多轮 Agent 工作流

- 支持 4 种 UI 自定义方式：自定义 Banner、输入框风格（Codex 风格 / Claude Code 风格 / Pi 风格 / 自定义）、加载动画、工具显示样式

- 附带 headless 版本 create-headless-agent，可在 CI 环境中无终端运行

- 作为 OpenRouter Skills 生态的一部分，通过 GitHub 仓库 OpenRouterTeam/skills 分发
