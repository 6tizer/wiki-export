## 定义

Terminal Bench 2 是衡量终端编程 Agent 执行能力的基准测试，用来比较不同 CLI Agent 在真实终端任务上的表现。

## 关键要点

- 更贴近命令行开发与工具调用场景

- 常被用来对比 Claude Code、Grok CLI 等终端 Agent 的执行水平

- 是终端编程 Agent 赛道的重要公开跑分坐标
