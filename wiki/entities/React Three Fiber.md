## 定义

React Three Fiber 是 Three.js 的 React 渲染器，让开发者可以用 React 组件方式构建浏览器中的 3D 场景、材质、灯光与交互。

## 关键要点

- 它把原本偏底层的 Three.js 场景组织方式，转成更适合现代前端工程的组件化写法。

- 在这条发布内容中，React Three Fiber 与 Three.js 一起被用来承载可滚动驱动的真实 3D 场景和物理光照效果。

- 对 AI 编程 Agent 而言，这类声明式 3D 栈更容易与 React 页面其他部分一并生成和维护。

- 它常用于产品展示页、交互式 hero、3D 演示和沉浸式网页体验。
