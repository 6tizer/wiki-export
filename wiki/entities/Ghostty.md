## 定义

Ghostty 是一个高性能终端模拟器，因 GPU 渲染和现代终端体验而被大量 Claude Code 用户当作日常工作终端。

## 关键要点

- 强调丝滑渲染、分屏和原生体验

- 适合长输出、代码块和高频交互场景

- 在 AI 编程工作流中常被视为 Claude Code 的优选终端

- 内置 Quick Terminal 等高频入口能力，适合碎片化命令行工作流
