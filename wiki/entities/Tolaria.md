## 定义

Tolaria 是 Luca Rossi 发布的一款 macOS 桌面应用，用于管理基于 Markdown 文件的知识库，并作为人类与 AI Agent 协作编辑知识库的主要工作台。

## 关键要点

- 以普通 Markdown 文件和 Git 仓库为底层存储，强调可移植与可审计。

- 支持 Claude Code、Codex 等 AI 工具直接在知识库上创建、整理和更新笔记。

- 用“类型”“属性”“前置元数据”等机制辅助导航，而非用强约束数据库去限制笔记。

- 项目本身开源，公开展示大量提交、测试、ADR 与 Agent 配置，具有很强的工程样本价值。
