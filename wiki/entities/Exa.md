## 定义

Exa 是面向 AI Agent 的搜索与内容基础设施，提供可程序化调用的网页搜索与内容提取能力，并开始支持通过 x402 进行按次付费访问。

## 关键要点

- 面向 Agent 场景提供搜索与内容接口，强调可被模型直接调用

- 在无 API Key 的情况下，可通过 x402 返回 402 支付信息并完成重试支付流程

- 这使 Exa 从传统 API 服务进一步转向“可被 Agent 自主购买和调用”的基础设施
