## 定义

SkillNote 是一个面向 AI 编码工具生态的 Skills 自托管注册中心与分发层，目标是让 Skill 像软件包一样被发布、发现、安装和按需加载。

## 关键要点

- 它把分散在本地目录中的 Skill 组织成可检索、可共享、可复用的注册表。

- 它试图解决团队协作中“项目依赖了哪些 Skill、如何同步安装”的问题。

- 它支持多种 Coding Agent / Agent 工具链场景，定位类似 Skills 世界里的 `package.json` 或私有 npm registry。

- 在本文语境里，SkillNote 的意义不仅是管理 Skill，更是帮助缓解 Skill 数量增长后带来的上下文预算浪费与加载失控。
