## 定义

Mem0 是一个轻量级 AI 记忆插件，通过 **Markdown 结构化**知识管理，支持跨会话与跨 Agent 记忆复用，显著降低 token 消耗。已集成到 OpenClaw。

## 关键特点

- Markdown 格式存储记忆，透明可审计

- 跨会话与跨 Agent 记忆共享

- 相比原生记忆插件显著降低 token 消耗

- 安装配置简单

- 适合个人 / 小团队的长期工作助手场景

## 对比同类

- **Mem0**：轻量、个人优先

- **MemOS**：团队级、可视化优先
