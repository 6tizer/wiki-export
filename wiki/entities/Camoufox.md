## 定义

Camoufox 是一个基于 Firefox 深度改造的反检测浏览器，通过底层指纹伪造减少自动化脚本被网站识别的概率。

## 关键要点

- 不只是“隐身模式”，而是从浏览器底层伪装设备特征

- **纯本地运行**：无需云端 API Key，适合作为本地浏览器自动化底座

- 适合需要稳定访问高风控网站的自动化和 Agent 场景

- 也常被用作 Browser Use 云端方案的本地替代后端，适合对隐私、稳定性或费用更敏感的场景

- 可通过 Hermes Agent 的 skill 封装方式接入本地浏览器自动化流程

- 是 camoufox-cli 等工具的核心底座
