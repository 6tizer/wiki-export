## 定义

Cognee 是一个面向 Agent 记忆与知识演化的知识引擎，强调让技能和知识随执行历史持续自我更新。

## 关键要点

- 结合向量搜索、图谱与长期记忆思路

- 关注技能自改进和知识演化

- 适合长期运行的 Agent 系统

- 采用关系型存储、向量存储和图存储三层协同的记忆架构

- 以 `add / cognify / memify / search` 四步 API 把摄取、结构化、优化和检索串成完整闭环
