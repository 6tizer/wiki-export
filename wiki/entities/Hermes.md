## 定义

Hermes 是一个 AI Agent 平台/产品，定位为可持续成长的 Agent 系统，强调从「一次性工具」转变为「可积累、可进化」的长期协作伙伴。

## 关键特点

- 将 Agent 从一次性工具推向可持续成长的系统

- 支持通过插件扩展能力（如 MemOS 记忆插件）

- 社区讨论度较高，生态活跃

- 与 OpenClaw 生态有关联但定位不同

## 与 OpenClaw 的区别

- 评论区提到「MemOS 基本上是给 OpenClaw 使用的，Hermes 没办法自动记录，因为没有 hook」

- Hermes 更偏向通用 Agent 平台，OpenClaw 在记忆和 Skill 生态上更深入
