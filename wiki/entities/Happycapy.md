## 定义

Happycapy 是一个把 OpenClaw 与 Claude Code 能力封装进浏览器沙盒环境的 Agent 原生电脑产品，强调无安装门槛与 Skill 生态。

## 关键要点

- 通过云端沙盒降低本地部署和权限风险

- 用 Skill 体系把 Agent 扩展能力产品化为可安装模块

- 面向非技术用户提供更低摩擦的 Agent 入口
