## 定义

camoufox-cli 是一个面向 Agent 的浏览器自动化 CLI，封装了基于 Firefox 的反检测浏览器 Camoufox，帮助 Agent 在高风控网站上更稳定地访问页面。

## 关键要点

- 以命令行方式暴露浏览器操作能力，适合被 Agent 直接调用

- 聚焦验证码、登录墙和自动化指纹暴露问题

- 适合作为 OpenClaw、Claude Code 等工具链中的浏览器 Skill
