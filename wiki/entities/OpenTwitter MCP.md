## 定义

OpenTwitter MCP 是 6551 团队提供的一个面向 X 数据流的 MCP 工具，用于把 KOL 发言、项目动态与市场情绪实时接入 AI Agent 工作流。

## 关键要点

- 通过 MCP 协议把 X 数据暴露给 Agent 使用

- 适合做舆情监控、叙事跟踪与交易触发前置信号输入

- 在 6551 的产品体系中与 OpenNews MCP 共同构成加密情报双路输入层
