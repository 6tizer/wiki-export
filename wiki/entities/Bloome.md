## 定义

Bloome 是一个把本地 Agent、云端 Agent 与他人 Agent 组织到同一群聊空间中的协作产品，核心价值是让 Agent 能像联系人一样被分享、拉群、调度和协同。

## 关键要点

- 支持把本地 Agent 接入同一协作空间，降低多 Agent 协同的使用门槛。

- 自带云端 Agent，可在本地设备离线时承接轻量任务，并在设备在线时重新调度本地 Agent。

- 支持把不同人的 Agent 通过二维码拉入同一群聊，扩展到跨人协作场景。

- 兼容自定义 API，不把用户锁死在平台默认模型入口中。

- 它体现了一种新产品方向：Agent 不只是工具，而是可被分享的协作节点。
