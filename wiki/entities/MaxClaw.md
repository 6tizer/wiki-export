## 定义

MaxClaw 是 MiniMax 推出的 OpenClaw 变体，将多个专家 Agent 整合进统一入口，提供云端一站式 AI 助手体验。

## 关键要点

- 在同一个 gateway 中集成 6 个子智能体（热点追踪、可视化助手、生图、调研等），共享知识文件和记忆系统

- 内置事实确认机制：调研任务完成后独立查证 Agent 二次确认信息准确性

- 无需额外 API Key，内置搜索聚合替代 Brave API

- 支持从专家社区获取 Agent（已有 1 万个公开 Agent）

- 月费 39 元起（Coding Plan）
