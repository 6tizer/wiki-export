## 定义

Maps JavaScript API 是 Google Maps Platform 中用于在网页中嵌入地图与交互能力的核心接口产品。

## 关键要点

- 是许多地图原型和 Web 应用的首选接入层

- 与 Demo Key 的可用范围直接相关

- 适合地图展示类 MVP，但不覆盖全部地理服务能力
