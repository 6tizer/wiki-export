## 定义

UniVA 是面向复杂视频创作的通用视频 Agent 系统，强调通过自然语言完成视频的规划、编排与生成。

## 关键要点

- 采用 Plan Agent 与 Act Agent 分离的多 Agent 架构

- 支持长期记忆、隐式意图理解与主动修正

- 兼容文本、图像、实体和现有视频等多种输入条件

- 适合多镜头、多角色和长流程视频生产
