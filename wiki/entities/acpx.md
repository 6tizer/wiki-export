## 定义

acpx 是一款 Agent 控制工具（CLI），用于在命令行中编排多个 Coding Agent（如 Claude Code、Codex、Gemini CLI）协同工作。它让脚本可以创建会话、向指定 Agent 发送指令并收集输出，是 autoresearch 软件开发流程中的关键基础设施。

## 关键要点

- 支持 Codex、Claude Code、OpenCode 等多种 Coding Agent

- 通过 CLI 会话管理，实现 Agent 间的轮流执行和信息传递

- 是 autoresearch 软件开发工作流中，实现多 Agent 交叉审核的底层工具
