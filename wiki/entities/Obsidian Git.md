**定义**

Obsidian Git 是 Obsidian 社区插件，用于把本地笔记库接入 Git 工作流，实现提交、拉取、推送和定时同步等操作。

### 关键要点

- 作为社区插件，它把 Git 命令封装进 Obsidian 内部，降低了日常同步门槛。

- 文章重点使用它的自动 `commit-and-sync` 能力，把笔记改动定时推送到远端仓库。

- 插件能配合 `Pull on startup`、`Pull on commit-and-sync`、`Push on commit-and-sync` 等设置，改善多设备使用体验。

- 它依赖底层 Git 环境先完成仓库初始化和远程仓库绑定，不能完全替代首次命令行配置。
