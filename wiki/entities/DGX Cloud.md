## 定义

DGX Cloud 是英伟达面向 AI 训练与推理场景提供的云端算力平台，NIM 这类服务可基于这类 GPU 基础设施对外提供模型能力。

## 核心要点

- 代表英伟达把自家高端 GPU 算力服务化、平台化输出

- 对开发者而言，意义不在直接操作底层硬件，而在可消费的推理与模型服务

- 在文章语境里，它是 NIM 能够承载多模型 API 的底层算力基础之一
