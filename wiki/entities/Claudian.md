## 定义

Claudian 是面向 Obsidian 的第三方插件，用来把 Claude Code 的能力接入本地笔记环境，让用户能直接在笔记上下文里调用 Agent 完成写作、整理与结构化任务。

## 关键要点

- 它把 Coding Agent 从终端工作流延伸到了知识管理场景。

- 文章中的安装方式是手动下载 `main.js`、`manifest.json`、`styles.css`，放入 Vault 下的 `.obsidian/plugins/claudian/` 目录。

- 接入后，用户可以围绕当前笔记执行续写、重构、总结、论文大纲生成与社媒内容整理等任务。

- 它的价值不只是“聊天”，而是让 Agent 直接作用于本地笔记文件与知识库结构。
