## 定义

Qwen3.6-Plus 是 Qwen Code 在该版本中重点接入的新旗舰模型，提供超长上下文和较高的免费调用额度，用来支撑大型代码库理解与执行。

## 关键要点

- 支持 1M 上下文，适合长代码库和多文件任务

- 与 Qwen Code 的产品体验和定价策略强绑定

- 在 Coding Agent 场景里承担高质量主模型角色
