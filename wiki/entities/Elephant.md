## 定义

Elephant 是一个在 OpenRouter 上匿名上线的 100B 纯文本模型，主打高 token 效率、低延迟，以及面向轻量级 Agent 场景的高频交互能力。

## 关键要点

- 支持 256k 上下文和 32k 输出

- 适合代码补全、调试、长文档处理和轻量级 Agent 交互

- 上线不到 48 小时即登上 OpenRouter Trending 第一，调用量超过 1850 亿 token

- 实测显示其在前端原型、长文件梳理和开放式 Agent 任务上表现突出，但完整项目级应用能力仍有限

- 模型来源尚未公开，外界推测可能是某个新模型的 Flash 版本或新实验室产品
