### 定义

MediaCrawler 是一个开源的多平台自媒体数据采集项目，可抓取小红书等平台的帖子与评论数据，并提供 API 与可视化入口，便于后续分析。

### 关键要点

- 支持通过 uv、Node.js 与 Playwright 完成环境搭建。

- 可通过本地 API 服务和 Web 界面执行关键词爬取。

- 适合把社媒评论转成结构化数据，再接入 NotebookLM 等分析工具。

- 大规模抓取时需要关注登录、风控与代理配置。
