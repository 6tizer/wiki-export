## 定义

**达尔文.skill** 是一个面向 Claude Code / Skills 生态的 Skill 优化系统，受 Karpathy 的 autoresearch 启发，用“评估 → 改进 → 测试 → 保留或回滚”的循环持续提升已有 skill 的质量。

## 核心特征

- 优化对象是已有的 `SKILL.md`，而不是从零创建单个 skill

- 采用棘轮式 keep / revert 机制，只保留分数提升的修改

- 引入 **Human in the Loop**，在人类确认后再继续下一轮

- 将结构质量与实测效果拆开评估，并通过独立子 Agent 打分

- 支持批量扫描多个 skill，适合大规模 skill 资产维护

## 关键要点

- 核心受众是已经拥有较多 skill 资产的使用者

- 目标不是“写出新 skill”，而是“让已有 skill 系统性变好”

- 与 [女娲.skill](concepts/女娲.skill.md) 形成“生成 + 优化”的闭环
