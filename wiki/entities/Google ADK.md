## 定义

Google ADK 是 Google 提供的 Agent Development Kit，用于构建、编排和运行 Agent 应用。

## 关键要点

- 提供 Agent 运行与工具调用的开发底座

- 在 Always-On Memory Agent 中承担系统框架角色

- 适合需要接入 Google 模型与 Agent 示例工程的开发者
