## 定义

hermes-agent-camel 是一个基于 Hermes Agent 的社区分叉项目，在原有自进化 Agent 循环之上整合了 CaMeL 信任边界机制，用来增强生产环境中的安全控制。

## 核心要点

- 把 CaMeL 的信任边界原则嵌入 Hermes 运行时，重点缓解提示注入与越权调用风险

- 保留 Hermes Agent 原有的自我改进、技能沉淀和跨会话工作能力

- 适合希望把 Hermes 更稳妥地用于真实生产任务的使用者

- 属于 Hermes 生态里“安全增强型分支”的代表项目
