## 定义

妙搭是飞书推出的 AI 开发 Agent 产品，目标是让用户通过自然语言完成业务系统设计、开发和交付。

## 关键要点

- 从低代码平台升级为 Code Agent 形态

- 与飞书权限体系和协同场景深度集成

- 适合企业内部工具与流程应用快速搭建
