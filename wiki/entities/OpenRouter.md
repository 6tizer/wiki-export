## 定义

OpenRouter 是全球最大的模型聚合平台，提供统一 API 接入多家 LLM 提供商的服务，是 Agent 时代开发者选择模型的核心「战场」。

## 关键要点

- 聚合多家 LLM 提供商，提供统一调用接口

- 提供 Fastest 榜和 Trending 榜等排行榜

- Step 3.5 Flash 在其上调用量跻身全球第四，完全靠开发者自发选择

- 是 OpenClaw 生态中模型调用的主要渠道

- 排行榜数据反映真实开发者偏好，而非营销推广效果
