## 定义

CoPaw 是一个可部署、可扩展的个人 AI 助手平台，支持跨多平台使用，提供高度可定制的技能系统和本地 LLM 运行能力。

## 关键要点

- 跨平台支持：钉钉、飞书、QQ、Discord、iMessage 等

- 高度可扩展的技能系统，内置 Cron 定时任务

- 支持本地 LLM 运行（llama.cpp 或 MLX），无需 API Key

- 多种部署方式：一键安装、pip、Docker、ModelScope 云端

- 配备 Web 控制台进行可视化管理和 Agent 配置

- GitHub: [https://github.com/agentscope-ai/CoPaw](https://github.com/agentscope-ai/CoPaw)
