## 定义

Oz 是 Warp 于 2026 年 2 月推出的云端 Agent 编排平台，可同时运行最多 40 个并发 Agent，每个 Agent 在独立 Docker 容器中执行，自带审计追踪。

**别名**：Warp Oz

## 关键要点

- 支持通过事件、定时任务、Slack/Linear/GitHub 集成触发 Agent

- 每个 Agent 运行在独立 Docker 容器中，具备完整审计追踪

- Warp 内部已用 Oz 编写 60% 的 PR

- 曾在一次欺诈检测任务中发现 6 万美元问题

- 是 Warp 开源后社区贡献管理的核心基础设施——PR 自动分配 review、AI 初审后转人工专家

## 关联概念

- Warp：Oz 的母产品

- Open Agentic Development：Oz 是该开发范式的基础设施层
