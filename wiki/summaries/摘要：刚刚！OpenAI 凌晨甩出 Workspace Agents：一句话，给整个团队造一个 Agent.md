## 一句话摘要

OpenAI 将 Workspace Agents 带入 ChatGPT，把过去偏个人化的 GPTs 升级为面向团队协作的共享 Agent 平台：可跨工具执行长流程任务、支持定时与后台运行，并补齐企业级治理能力。

## 关键洞察

- Workspace Agents 的核心跃迁不只是“更强的聊天机器人”，而是把 Agent 从一次性对话工具升级为可持续运行的团队工作流执行者。

- 底层由 Codex 驱动，使 Agent 具备写代码、调工具、跨步骤处理复杂任务的能力，适合产品、销售、财务、IT 与员工支持等组织场景。

- OpenAI 通过 Scout、Spark、Tally、Slate、Trove、Angle 等模板，试图把 Agent 搭建门槛从工程团队下放给业务团队。

- 企业级可用性的关键不只在模型能力，还包括共享调用、Slack 接入、审批机制、细粒度权限、运行记录与合规审计能力。

- 这次发布也释放出明确信号：GPTs 会继续存在，但未来组织级 Agent 形态大概率会向 Workspace Agents 收敛。
