**一句话摘要**：即梦 AI 发布官方 CLI 工具，Agent 一行命令即可调用 Seedance 2.0 文生视频等功能，成为国内首个面向 Agent 设计的视频生成平台，标志 AI 视频生产全面 Agent 化。

---

## 关键洞察

- **8 个生成命令**：text2image、text2video、image2image、image2video、image_upscale、multimodal2video（Seedance 2.0 旗舰）、multiframe2video、frames2video

- **Agent 设计导向**：官方文档标题即是“面向 AGENT 的工具包”，只需一键配置后 Agent 自动使用用户即梦账号在任意环境生成内容

- **重要意义**：火山引擎不先开放 Seedance 2.0 API，即梦 CLI 先给 Agent 用上了；视频生产流程 Agent 化全面自动化

- **安装**：`curl -s https://jimeng.jianying.com/cli | bash`
