### 一句话摘要

CodePilot 通过宠物成长、心跳唤醒和持续陪伴感，把 Agent 基础设施的搭建过程从“隐性成本”变成“可见反馈”。

### 关键洞察

- 真正被产品化的不是 Claude Code 本身，而是用户和 Agent 的长期关系。

- Buddy 系统把记忆、技能、活跃度等抽象能力映射成可感知的成长曲线。

- 心跳系统强调 Agent 在用户离开后也持续准备和整理，而不是静态等待命令。
