### 一句话摘要

Autoresearch 的真正方法论，不是“让 AI 自己做研究”，而是给 Agent 加上纪律化实验约束与可追溯记忆，让它能持续进行可归因的探索。

### 关键洞察

- 每次只改一个变量，是让 Agent 研究结果可解释、可累计的核心纪律。

- Git 在这里不是版本管理工具，而是实验记忆系统。

- 人类负责定义方向、边界和评价函数，Agent 负责穷举与坚持。

- 自动化研究的上限，取决于评价函数是否真的对准用户关心的问题。
