## 一句话摘要

Coinbase 试图用 [Agentic.market](http://agentic.market/) 在 x402 支付能力之上补齐 Agent 服务发现层，让智能体不仅能“付钱”，还能“找到并接入值得购买的服务”。

## 关键洞察

- 这条动态的核心不是单一工具发布，而是把 Agent 经济往“发现 + 支付 + 分发”的完整市场结构推进了一步。

- x402 解决的是 Agent 原生支付问题，[Agentic.market](http://agentic.market/) 解决的是服务可发现性问题，两者组合才更接近可运行的 Agent 商业闭环。

- 从评论区反馈看，社区普遍认可“discovery is the missing primitive”，但同时指出信誉、身份、质量评估仍是未完成的下一层基础设施。

- 对服务提供方而言，这不只是一个目录页，而是一个面向 Agent 的新分发入口，意味着未来可能出现“给 Agent 卖服务”的独立供给侧市场。

- 这类基础设施的真正价值，不在于让 Agent 更会聊天，而在于让 Agent 能在标准化协议下自主发现、调用、支付并完成交易。

## 提取的概念

- Agentic.market

- x402 协议

- Agent 服务发现层
