## 一句话摘要

Anthropic 与 Cerebral Valley 联合举办的 Claude Code Hackathon 结束，参赛者使用 Opus 4.7 模型进行了一周的开发竞赛，公布了获奖项目。

## 关键洞察

- Anthropic 与 Cerebral Valley 联合举办了为期一周的线上 Claude Code Hackathon，奖金池超 10 万美元 API 额度

- 参赛者使用 Claude 最新的 Opus 4.7 模型进行开发

- 获奖项目包括 MedKit（医学诊断 AI 反馈工具）、Maieutic（辅助人类提升技能而非替代）、Protagonist（基于 Claude 的交互式小说平台）

- 社区反馈以积极为主，但也有大量用户抱怨 Pro 订阅 token 限制过严
