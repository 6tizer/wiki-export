**一句话摘要**

这篇文章提出，最适合给 AI 编程助手读取的设计系统形式，不是复杂设计文件，而是项目根目录下一份结构清晰的 [DESIGN.md](http://design.md/)。

## 关键洞察

- Markdown 设计系统对 LLM 更友好，能直接约束 Claude Code、Cursor 等工具的界面输出。

- [DESIGN.md](http://design.md/) 把设计规范从设计师工作区转成了开发仓库内的长期上下文。

- awesome-design-md 这样的资源库正在把设计风格模板化、可复制化。

- 这套做法本质上是在为 Coding Agent 构建“可执行的设计记忆”。
