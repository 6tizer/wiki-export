## 一句话摘要

ReasoningBank 提出了一种面向 Agent 的记忆框架，让 LLM 代理能够同时从成功与失败经历中持续学习，从而提升任务成功率与执行效率。

## 关键洞察

- 该框架强调记忆不应只保存成功案例，也应沉淀失败经验。

- 核心目标是把过往任务中的经验转化为后续可复用的推理与行动依据。

- 按公开介绍，ReasoningBank 能提升 agent effectiveness、success rates 与效率表现。

- 这一方向直接回应了 Agent 系统在长期记忆与经验积累上的常见短板。
