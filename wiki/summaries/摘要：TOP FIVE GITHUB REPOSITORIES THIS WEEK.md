## 一句话摘要

这条 X 帖子用“本周 GitHub 热门仓库”视角，串起了 Coding Agent、长期记忆、自我进化与极简自治 Agent 四条正在快速收敛的技术主线。

## 关键洞察

- 热门仓库不再只是单点工具，而是围绕 **编码行为约束、持久化记忆、技能沉淀、自我进化** 形成了一整套 Agent 基础设施栈。

- [andrej-karpathy-skills](concepts/andrej-karpathy-skills.md) 代表的是 **把编程方法论固化为上下文文件/skill** 的路线，本质是用规则约束减少 LLM 在写代码时的误判与过度设计。

- [claude-mem](entities/claude-mem.md) 与 [Hermes Agent](entities/Hermes Agent.md) 的高关注度，说明“跨会话记忆”和“可持续积累经验”正在从加分项变成 Agent 产品的核心竞争力。

- [Evolver](entities/Evolver.md) 和 [GenericAgent](entities/GenericAgent.md) 把焦点进一步推向 **自我进化**：一个强调协议化演化与可审计闭环，一个强调极简内核加技能树生长。

- 这类周榜内容更像“信号雷达”而不是完整评测：它适合发现趋势与关键词，但落地时仍需二次编译、去重和场景化验证。
