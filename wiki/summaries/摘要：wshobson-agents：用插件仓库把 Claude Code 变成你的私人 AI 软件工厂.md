### 一句话摘要

wshobson/agents 通过插件化角色、技能和工作流编排，把 Claude Code 从单一编程助手扩展成可协作的开发团队。

### 关键洞察

- 这个仓库的核心创新并不是“多几个 Agent”，而是把角色、工具和流程做成可安装组合件。

- 分层模型路由说明，多模型协作的收益往往来自任务分配而不是模型本身。

- 当项目规模扩大后，插件化组织比无约束的提示词堆叠更容易维护。
