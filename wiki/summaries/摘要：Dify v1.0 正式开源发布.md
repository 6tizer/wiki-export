## 一句话摘要

Dify v1.0 通过全新的插件化架构和中央市场，将 AI 应用开发从「整体式平台」升级为「可组合的插件生态」，大幅提升了灵活性和迭代速度。

## 关键洞察

- **插件化架构重构**：模型、工具、Agent 策略全部解耦为独立插件，支持热插拔更新，无需等待平台整体升级

- **中央市场（Marketplace）**：已上架 120+ 插件（OpenAI o1、Gemini 2.0、DeepSeek-R1 等模型及 Perplexity、Slack 等工具），形成开发者生态闭环

- **Agent 节点加入工作流**：新增 Agent 节点作为工作流中的智能决策中心，支持 ReAct 和 Function Calling 等策略，让工作流从「执行预设流程」进化为「自主决策」

- **多模态交互支持**：原生支持图像生成、语音交互等多种输入输出形式

- **生态连接器**：通过扩展类型插件与 Slack 等外部平台无缝对接，打破信息孤岛

## 个人评注

Dify 的插件化方向与 Tizer 的内容管道和自动化工作流高度相关。Agent 节点的引入意味着可以在 Dify 工作流中实现类似 HITL 的审核环节——让 Agent 在关键步骤做决策、人工在必要时介入。Marketplace 生态也为 OpenClaw 等项目提供了集成思路参考。
