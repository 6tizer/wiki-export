## 一句话摘要

这篇盘点文用 GitHub Star 增长榜勾勒出 Coding AI 的一周趋势，显示 OpenClaw、autoresearch、CLI-Anything、OpenViking 等项目正在把 Agent、研究自动化和本地推理基础设施推向新阶段。

## 关键洞察

- OpenClaw 以跨平台个人 AI 助手定位拿下榜首，说明“能直接做事”的本地 Agent 仍是最强叙事。

- autoresearch 把研究流程抽象成可自动循环的实验框架，代表 AI 自主科研方向的可操作化。

- CLI-Anything、OpenViking 这类项目说明生态焦点正在从单个模型能力转向工具接入和上下文基础设施。

- BitNet、AstrBot 分别代表低成本推理和中文多平台 Agent 入口两条重要支线。
