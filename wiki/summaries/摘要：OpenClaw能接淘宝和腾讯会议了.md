## 一句话摘要

国内多个大平台（腾讯会议、淘宝、高德地图）已开通 MCP 接口，OpenClaw 可直接接入进行会议管理、商品购买、路线规划等操作。

## 关键洞察

- **腾讯会议 MCP**：安装包+配置 Token，支持创建/修改/取消会议、查日程、获取会议纪要

- **淘宝 MCP**：安装 PC 客户端后本地运行 MCP Server，无需 API Key，支持搜商品/下单/查订单/催发货

- **高德地图**：注册开放平台获取 API Key，支持路线规划、POI 搜索、天气查询等12类接口

- **MCP 生态在国内加速落地**：主流 SaaS 平台已开始将 MCP 接口作为标配
