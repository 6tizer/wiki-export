## 一句话摘要

Superpowers 是赋予 Claude Code 完整工程化能力的 Skills 库，强调工程纪律优先于代码生成，采用标准化五步工作流和强制 TDD。

## 关键洞察

- **工程纪律 > 代码生成**：强制苏格拉底式提问和需求澄清，生成设计文档后才允许动手

- **五步工作流**：设计 → 准备（Git Worktree）→ 规划（2-5分钟微任务）→ 执行（Spec+Quality 双重审查）→ 质保（强制 TDD + 4阶段 RCA）

- **模块化与插件化**：深度集成 Claude Code 插件系统，支持热插拔

- **自我进化**：内置 writing-skills 辅助编写新技能
