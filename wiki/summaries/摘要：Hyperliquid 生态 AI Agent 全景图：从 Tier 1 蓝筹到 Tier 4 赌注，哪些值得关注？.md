## 一句话摘要

Hyperliquid 生态 AI Agent 四层排行快照：$VAPOR（无代码发射台）和 $FARM（链上 AI 游戏）领跑，但多数项目至 2025 年已大幅回调，印证叙事先行、产品滞后的规律。

## 关键洞察

- Hyperliquid 对 AI Agent 的结构优势：原生高性能订单簿 + HIP-1/HIP-2 代币发行机制，Agent 可直接调用链上交易 API

- $VAPOR：ai16z/ElizaOS 驱动的无代码发射台，市值峰值约 $100M，但长期处于 alpha 状态，被社区称「name就是命运」

- $FARM：融合 CryptoKitties+星露谷+Pokemon Go 的链上 AI 游戏，每个生物都是有独立钱包的自主 Agent

- Tier 3-4 项目普遍流动性薄，$PEAR 存在诈骗风险，需区分「叙事」和「产品」

- 2025 年整体大幅回落，验证了链上 AI Agent 代币的高波动性和叙事周期依赖
