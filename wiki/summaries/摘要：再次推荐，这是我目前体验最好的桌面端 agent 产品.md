## 一句话摘要

推荐 Craft Agents 作为目前体验最好的桌面端 Agent 产品，编程用 opencode、非编程用 Craft Agent 的组合方案。

## 关键洞察

- **桌面端 Agent 二分法**：作者提出编程场景用 opencode、非编程场景用 Craft Agent 的分工思路，反映了当前 Agent 工具正在走向场景细分

- **Craft Agents 核心优势**：UI 美观、体验流畅、内置 MCP/Skill 管理、任务状态跟踪与归档、定时任务、Workspace 隔离、Telegram 集成

- **社区反馈两极化**：部分用户高度认可（如比「龙虾」好用），但 DeepSeek 模型兼容性问题和 LiteLLM 接入 bug 导致另一部分用户体验不佳

- **开源生态涌现**：回复中出现了 pawwork 等受 Craft Agents 启发的替代品，说明桌面端 Agent 赛道正在快速生长
