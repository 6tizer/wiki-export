## 一句话摘要

Manus Desktop 通过 My Computer 功能把通用 Agent 从云端执行延伸到本地文件、终端和应用控制，标志着桌面 Agent 形态开始成熟。

## 关键洞察

- Manus 的核心升级不是聊天质量，而是本地执行边界被打开。

- My Computer 让文件处理、命令执行和应用操控进入统一 Agent 界面。

- 这一路线更强调“开箱即用”的消费级体验，而不是极客式自托管。

- 本地操作能力也同步抬高了权限、安全和审批设计的重要性。
