## 一句话摘要

Browser Use 是让 AI Agent 能像人类一样操控网页浏览器的开源 Python 库，85,000+ GitHub Stars、YC W25 背书、$17M 融资，是 Browser Automation + AI Agent 赛道的领头项目。

## 关键洞察

- **与传统浏览器自动化的差异**：Selenium/Playwright 是确定性脚本，Browser Use 封装了 AI 推理层，支持自主决策和异常处理

- **极低上手门槛**：5 分钟即可跑通，支持 OpenAI、Claude、Gemini、Ollama 多种 LLM

- **85000+ Stars / 3 个月**：2024 年 10 月成立，2 个年轻创始人用 4 天做出原型战统 HN 首页

- **已知局限**：复杂任务推理较慢，LLM 非确定性导致可靠性有波动，部分网站会检测封锁

- **Cloud 版多了企业级能力**：stealth 浏览器、自动破解 CAPTCHA、住宅代理
