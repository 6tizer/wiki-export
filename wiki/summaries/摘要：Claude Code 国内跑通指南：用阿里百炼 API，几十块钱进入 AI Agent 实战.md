## 一句话摘要

国内用户可通过阿里百炼 Coding Plan 或直接对接国产模型 API，以几十元的成本跑通 Claude Code，进入 AI Agent 编程时代。

## 关键洞察

- Claude Code 已占全球 GitHub 提交的约 4%，日提交超 13.5 万次，创作者 Boris 上月 100% 代码由其自己写

- **四步跑通**：获取百炼 API Key → 安装 Claude Code → 跳过强制登录 + 配置 API → 启动测试

- 推荐模型：kimi-k2.5（社区反馈综合体验好，支持图片理解）

- 已知问题：v2.1.63 版本 API Error 400，需关闭 Thinking mode 或降级到 v2.1.62

- Cursor 日常编写 + Claude Code 处理复杂任务是专业开发者的主流组合
