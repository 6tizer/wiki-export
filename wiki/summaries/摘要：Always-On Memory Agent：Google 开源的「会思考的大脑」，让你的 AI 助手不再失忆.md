### 一句话摘要

Always-On Memory Agent 把长期记忆从“被动检索”推进到“主动整合”，让 Agent 能持续把新旧信息连接成更高层洞察。

### 关键洞察

- [Always-On Memory Agent](entities/Always-On Memory Agent.md) 的关键不只是存储，而是周期性做 [记忆整合](concepts/记忆整合.md)。

- [Google ADK](entities/Google ADK.md) 提供了系统运行框架，轻量模型与 SQLite 降低了持续运行门槛。

- 这种设计明显区别于传统 RAG 的被动查询逻辑。

- 它更适合个人助手、小型知识系统和成本敏感场景。
