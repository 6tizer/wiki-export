## 一句话摘要

LibTV 是 LibLib 推出的同时面向人类和 AI Agent 的视频创作平台，支持无限画布节点式创作，通过 API Key 让 Agent 调用所有图像视频生成模型，实现自动化内容生产流水线。

## 关键洞察

- **双模操作**：人类操作、Agent 操作、人类+Agent 协作三种模式都支持

- **五种节点类型**：文本、图片、视频、音频、脚本，构成完整内容生产链路

- **Agent 接入极简**：生成 API Key 后把 Skills 地址发给 Agent 即可，支持 OpenClaw/Claude Code/Codex

- **自动化生产场景**：让 OpenClaw 收集 AI 新闻，调用 LibTV 自动生成播客或视频

- **纯 Python 标准库实现**：Skill 无需额外安装第三方依赖，支持200MB以内图片/视频上传

- **定价有竞争力**：年卡最低39折
