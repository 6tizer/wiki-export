### 一句话摘要

Nunchi 的 agent-cli 把链上交易策略、MCP 接口和 Agent 调用方式打包到一个统一工具后端里，展示了 DeFi 交易工具 Agent 原生化的清晰形态。

### 关键洞察

- agent-cli 的价值不只是“会交易”，而是把做市、动量、套利和 LLM 决策统一成可编排模块。

- 它同时支持 OpenClaw、Claude Code 和 MCP，说明交易能力正从独立 bot 变成通用 Agent 工具层。

- YEX 与 Hyperliquid 场景的绑定，让它兼具研究、执行和策略迭代的完整链路。

- 私钥管理、Builder Fee 与主网风险仍是从 demo 走向生产前的关键门槛。
