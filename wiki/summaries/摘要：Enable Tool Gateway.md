## 一句话摘要

Nous Research 在 Nous Portal 中上线 Tool Gateway，让付费订阅用户无需分别管理 API Key，就能在 Hermes Agent 中统一调用 300+ 模型以及网页抓取、浏览器自动化、图像生成、云终端和文本转语音等托管工具。

## 关键洞察

- 这次上线的重点不是单个新工具，而是把“模型订阅 + 工具接入 + 账户鉴权”收敛到 Nous Portal 的统一入口。

- Tool Gateway 首批覆盖网页抓取、浏览器自动化、图像生成、云终端后端和文本转语音，说明 Nous 正在把 Hermes Agent 的能力从模型层延伸到执行层。

- 对 Hermes Agent 用户来说，它降低了外部工具接入门槛，尤其适合需要网页操作、内容抓取和媒体生成的复合型 Agent 工作流。

- 从评论区反馈看，用户最关心的仍是套餐额度、隐私边界、兼容性，以及它是否真的能替代分散采购的第三方服务。
