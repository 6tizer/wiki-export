## 一句话摘要

TeamClaw 是一个基于 OpenCode 与 Tauri 2.0 构建的本地 AI Agent 桌面工作空间，试图把多渠道接入、团队协作、知识库同步与企业系统集成收敛到同一套本地优先的协作界面中。

## 关键洞察

- 它的定位不是单点的 AI 编程插件，而是一个围绕 Agent 运行、协作与管理展开的桌面工作空间。

- 产品采用本地运行架构，基于 Tauri 2.0（Rust）构建，强调 macOS / Windows 双平台与本地文件操作能力。

- 在交互层面，三栏式界面把渠道、聊天与详情拆分开，更接近团队长期使用的工作台而非一次性对话框。

- 在连接层面，TeamClaw 同时支持 Discord、飞书、邮件、Kook、企业微信、微信等渠道，并可通过 MCP 接入企业系统。

- 在协作层面，它把 P2P / S3/OSS 团队同步、Skills 分发、MCP 配置共享和知识库文档同步放进同一产品，显示出 Agent 工作空间平台化的方向。
