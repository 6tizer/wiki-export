**一句话摘脁**：卡尔的AI沃兹将 Claude Code 泳露源码中最有价值的六种模式提炼为 Skill，开源为 cc-harness 项目，支持 Claude Code、Codex、OpenClaw、OpenCode 等多个平台。

---

## 关键洞察

- **6 个 Skills**：① CC Dream Memory（记忆合并整理）；② CC Memory Extractor（用户偏好提取）；③ CC Context Compressor（9段式压缩）；④ CC Verification Gate（只读验证视角）；⑤ CC Swarm Coordinator（四段分工）；⑥ CC Kairos Lite（轻量主动任务）

- **跨平台**：不强依赖 Claude Code 环境，OpenClaw 也能用
