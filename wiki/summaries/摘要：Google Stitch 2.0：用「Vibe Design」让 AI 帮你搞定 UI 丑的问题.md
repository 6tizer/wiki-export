**一句话摘要**：Google Stitch 2.0 引入「Vibe Design」理念，通过 AI + MCP Server 让用户用自然语言描述风格即可生成高质量 UI，解决 AI 生成 UI 普遍偏丑的问题。

**关键洞察**

- Vibe Design = Vibe Coding 的 UI 设计版本

- MCP Server 集成使 Stitch 可与现有开发工具无缝协作

- 自然语言风格描述降低了设计门槛

- Google 进入 AI UI 生成赛道，竞争格局升级

- 对 Cursor、OpenClaw 等工具的 UI 生成能力构成挑战

**提取的概念**：Vibe Design、Google Stitch 2.0
