## 一句话摘要

慢雾科技推出 MistTrack Skills，让 AI Agent 通过 API 调用实现链上地址风险评分、交易调查和资金流向分析，增强 Web3 操作安全性。

## 关键洞察

- 集成多条主流区块链的反洗钱风险分析能力

- AI Agent 可自动调用进行安全检测（支持 OpenClaw、Claude Code 等）

- 提供地址风险评分、交易调查和资金流向分析功能

- 通过简单 API 调用即可使用
