### 一句话摘要

OpenHarness 试图用极小代码量提炼 Claude Code 的核心 harness 层，成为研究和二次开发 Agent 的轻量参考实现。

### 关键洞察

- 它的价值不在替代 Claude Code，而在于把复杂 Agent 基础设施压缩成可读、可拆解的实现。

- Harness 工程被拆成工具、权限、上下文、生命周期等可研究模块。

- 轻量实现会成为学习和快速实验 Agent 架构的重要入口。
