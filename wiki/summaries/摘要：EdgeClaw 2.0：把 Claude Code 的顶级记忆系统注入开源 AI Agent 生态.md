### 一句话摘要

EdgeClaw 2.0 把 Claude Code 风格的长期记忆迁移到 OpenClaw 生态里，并通过双记忆文件和三级安全策略解决隐私与能力的冲突。

### 关键洞察

- 它不是简单抄记忆文件，而是在开源 Agent 里重构长期记忆机制。

- 双记忆文件架构让云端模型和本地模型看到不同粒度的信息。

- 三级安全策略说明未来 Agent 记忆系统会越来越依赖分级路由与脱敏治理。
