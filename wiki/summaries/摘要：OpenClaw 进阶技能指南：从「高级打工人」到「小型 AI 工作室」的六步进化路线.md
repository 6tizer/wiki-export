## 一句话摘要

六个进阶 OpenClaw 技能可让个人 AI 助手从单个 Agent 进化为能独立承接复杂任务的「小型 AI 工作室」。

## 关键洞察

- **Multi-Agent Orchestrator**：四个各司其职的 Agent 分分工执行，适合复杂调研、链上分析等高认知任务

- **Advanced RAG + Knowledge Pack Builder**：成为「不会幻觉」的专属知识库，支持向量检索+知识图谱结构

- **Autonomous Task Planner**：收到模糊需求后自己拆解并闭环执行，实现真正的自主能力

- **Browser Automation Pro + Stealth Mode**：内置 Agent Browser 的升级版，支持多标签并行、会话持久化、无痕运行

- **安全警告**：ClawHub 上约 12% 的技能存在恶意代码风险，优先选择带 Verified 标记的包

- **建议安装顺序**：Multi-Agent 框架 → RAG/知识库 → 自主任务规划器 → 按需补充
