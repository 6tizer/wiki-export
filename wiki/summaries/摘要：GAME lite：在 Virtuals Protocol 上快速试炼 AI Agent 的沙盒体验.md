**一句话摘要**：GAME lite 是 Virtuals Protocol 的轻量级 Agent 沙盒界面，上手极快但生产部署存在黑盒感，配合官方 Python/TypeScript SDK 可弥补不足。

**关键洞察**

- GAME（Generative Autonomous Multimodal Entities）是 Virtuals Protocol 的模块化 Agent 决策引擎，分高层规划与底层执行两级架构

- GAME lite 是浏览器端的零代码沙盒，无需本地环境即可测试 Agent 逻辑

- 主要痛点：内部推理过程不透明（黑盒），从沙盒到生产需掌握 SDK，旧版 SDK（virtuals-python）已弃用迁移至 game-python

- SDK 对所有开发者免费开放，有慷慨默认调用额度

- 与 Eliza 对比：GAME 与代币经济深度绑定（Agent 可代币化，持有者共享收益），而 Eliza 更偏纯技术框架

**提取的概念**：G.A.M.E 框架、Virtuals Protocol、AI Agent 发射台
