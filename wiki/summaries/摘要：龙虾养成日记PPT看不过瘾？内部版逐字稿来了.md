## 一句话摘要

傅盛通过 14 天 1157 条消息的亲身实践，将一只 AI 龙虾从零培养成 8 个 Agent 的 7×24 小时自动运转团队。

## 关键洞察

- **龙虾 = 工具的 AGI**：不是把 Agent 当软件，而是把整台电脑的所有权限交给 Agent，让它像人一样工作

- **记忆系统五次进化**：从上下文 → 文件存储 → TODO 清单 → 向量化搜索，核心原则是「不靠脑子，靠文件」

- **Skill 积累是核心壁垒**：每个 Skill 背后都是真实经验，不会消失，可瞬间传递给其他 Agent

- **Agent 间知识传递成本趋零**：一只龙虾学会发语音后群发邮件教其他龙虾，其他龙虾读完立刻就会

- **Multi-Agent 自组织**：三万自己设计组织架构（总指挥+笔杆子+参谋+运营官等），参谋和笔杆子之间有 5 轮以上打磨机制

- **Cron 自动化是关键**：「你是 Agent，不是人，没有深夜的概念」，24 小时不停运转
