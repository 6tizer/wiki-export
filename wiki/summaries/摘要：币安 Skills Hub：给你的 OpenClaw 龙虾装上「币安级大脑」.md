## 一句话摘要

币安 Skills Hub 让 OpenClaw 等 AI Agent 具备行情扫描→地址监控→合约审计→下单执行的完整闭环能力，策略由人制定、执行交给 AI。

## 关键洞察

- OpenClaw 龙虾 + 币安 7 个技能 = 全流程加密交易闭环能力

- **核心配对逻辑**：叙事发酵→地址监控→合约体检→现货交易

- **指令模板丰富**：现货下单含余额检查+止盈止损、聪明钱筛选、合约安全体检等可直接复用的模板

- **安全警告**：OpenClaw 技能市场曾出现使用拼写变体攻击窃取环境变量的恶意包；API Key 务必设置 IP 白名单和子账户隔离资金

- Eliza（AI16Z）占 TypeScript Agent 生态市占约 60%，币安 Skills 的主要差异化在于直连币安数据和交易接口
