## 一句话摘要

mcp-use 是一个全栈跨语言 MCP 开发框架，支持 TS/Python 双语言 SDK，一次编写即可在 ChatGPT、Claude 等多平台运行，内置调试器和云端一键部署。

## 关键洞察

- **一次编写多平台运行**：构建的 MCP 服务可同时适配 ChatGPT、Claude 等不同 AI 平台

- **全栈覆盖**：同时支持 MCP 服务端、MCP 应用（带交互 Widget）、AI Agent、MCP 客户端

- **双语言 SDK**：TypeScript + Python，覆盖主流 AI 开发语言

- **开箱即用**：内置 Inspector 调试器 + 云端一键部署
