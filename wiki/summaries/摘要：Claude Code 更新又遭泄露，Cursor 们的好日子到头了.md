## 一句话摘要

Anthropic 正把 Claude 从模型供应商推进为产品平台，借由 Claude Code 与 Managed Agents 等产品形态向上吞噬应用层价值，这让 Cursor 一类依赖模型 API 的 Coding Agent 产品面临更强的平台挤压。

## 关键洞察

- Claude Code 泄露的新功能，显示 Anthropic 正从终端编程助手继续走向更完整的全栈开发环境。

- 卖 API 是价格竞争，卖订阅产品、托管 Agent 与工作流解决方案，则意味着更高利润率与更强用户锁定。

- Cursor、Lovable、Bolt 等应用层产品的核心风险，是底层能力依赖模型供应商，而供应商本身正在亲自下场做同类产品。

- 这种“平台先开放生态、再向上整合价值”的路径，在 Apple、AWS、Google、Microsoft 的历史中都反复出现。

- 应用层并非完全没有空间，但更可行的路径会是做模型公司不愿深做的垂直场景、企业定制与深度集成，同时争取模型中立能力。
