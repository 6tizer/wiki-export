### 一句话摘要

Unusual Whales 把专业美股另类数据封装成 MCP 接口，意味着垂直数据平台开始主动进入 Agent 工具链，而不是只服务人工分析师。

### 关键洞察

- 真正限制 AI 交易助手的往往不是模型，而是可信、可调用的数据源。

- MCP 让交易数据接入从“自己写脚本”变成“对话式调用”。

- Unusual Whales 的价值在于期权流、暗池和议员交易等另类数据深度，而不是通用行情覆盖。

- 官方 MCP 的出现说明垂直 SaaS 已经意识到 Agent 是新的分发渠道。
