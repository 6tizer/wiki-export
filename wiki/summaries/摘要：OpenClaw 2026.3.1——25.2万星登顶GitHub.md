## 一句话摘要

OpenClaw 2026.3.1 发布合并 90+ PR，GitHub 星标 25.2 万超越 React 登顶，核心更新包括 Claude 自适应思维、OpenAI WebSocket 传输和飞书表格技能。

## 关键洞察

- **Claude 自适应思维**：根据任务难度动态切换 Low/Medium/High，简单任务跳过思考省 Token

- **OpenAI WebSocket**：响应更快、连接更稳，warm-up 预热降低首轮延迟

- **飞书表格技能**：在飞书文档中创建/修改表格、上传图片文件

- **安卓增强**：回复消息、摄像头、权限管理、设备健康检查

- **更新风险**：有用户反馈记忆被清空，务必备份
