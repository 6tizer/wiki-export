### 一句话摘要

self-improving-agent 通过日志、规则晋升和 Hook 自动化，让 Agent 把错误与教训沉淀成跨会话可复用的工作记忆。

### 关键洞察

- 真正的自我提升，不是模型自动变聪明，而是把经验从事故记录晋升为稳定规则。

- 本地 Markdown 文件让记忆过程可读、可改、可审查，降低了黑盒感。

- Hook 机制让记忆采集从主动记录变成半自动后台流程。

- 这条路线特别适合长期运行的编程 Agent 和知识型 Agent。
