**一句话摘要**

Zread CLI 是一个可在任意代码目录运行的 AI 代码解读工具，自动分析项目结构并生成详细文档，是理解 Claude Code 泄露源码的高效辅助工具。

**关键洞察**

- **Zread CLI**：命令行工具，自动分析代码项目并生成结构化文档

- 无需打开特定 IDE，在任意目录执行即可

- 生成的文档为 AI 编程工具（如 Claude Code/Codex）提供清晰的上下文

- 适用场景：新接手项目、刚 Clone 的开源代码、理解大型代码库

- 应用案例：将 Claude Code 泄露的51万行源码喂给 Zread 生成理解文档
