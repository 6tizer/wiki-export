**一句话摘要**

Hermes Agent 是具备三层记忆系统和技能自迭代能力的自进化 AI 助手，与 OpenClaw 互补，安装简单且支持导入 OpenClaw 配置。

**关键洞察**

- **三层记忆系统**：工作记忆 + 情节记忆 + 语义记忆，自动记录和更新解决方案

- **技能自迭代**：能从任务执行中学习并更新自身技能，实现自我优化

- **安全机制**：内置安全边界，防止技能自迭代超出预期范围

- **多平台自动化**：支持跨平台的自动化任务执行

- 与 OpenClaw 互补定位：OpenClaw 更注重工具调用和代码执行，Hermes 更注重自主学习

- 支持直接导入 OpenClaw 的设置和配置

Hermes Agent 的"自进化"特性对 HITL 工作流有特殊意义：高度自动化的任务可考虑用 Hermes，需要精确控制和透明度的任务仍用 OpenClaw。
