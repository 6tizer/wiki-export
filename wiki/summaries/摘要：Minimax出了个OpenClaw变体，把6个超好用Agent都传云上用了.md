## 一句话摘要

MiniMax 推出 MaxClaw，将 6 个专家 Agent 整合进 OpenClaw 框架，实现一个入口多 Agent 切换的云端体验。

## 关键洞察

- **多 Agent 统一入口**：MaxClaw 在同一个 gateway 中集成热点追踪、可视化助手、生图、调研等 6 个子智能体，共享知识和记忆系统

- **事实确认环节**：调研任务完成后，独立查证 Agent 会二次确认报告中信息的准确性，用红色标注存疑内容

- **社区 Agent 生态**：MaxClaw 支持从专家社区获取额外 Agent（已有 1 万个公开 Agent），可复制到个人空间二次优化

- **降低使用门槛**：无需额外 API Key，内置搜索聚合替代 Brave API，解决了 OpenClaw 部署配置复杂的痛点

- **Skill 自动封装**：社区 Agent（如 OpenClaw Skill 搜索器）能自动从 GitHub 找到开源项目并封装为新技能包
