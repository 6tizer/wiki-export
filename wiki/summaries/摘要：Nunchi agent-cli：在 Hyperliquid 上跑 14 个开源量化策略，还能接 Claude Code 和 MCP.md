## 一句话摘要

Nunchi agent-cli 把 Hyperliquid 上的多种量化策略、风险控制和 AI 调用接口打包成一个可直接被 Agent 使用的交易工具栈。

## 关键洞察

- 它的亮点不只是 14 个开源策略，而是把选币、执行、止损和复盘都模块化了。

- APEX、Radar、Guard、REFLECT 共同构成了一个带反馈闭环的自主交易系统。

- 通过 MCP、CLI 和 OpenClaw Skill 接口，交易框架开始直接面向 Agent 暴露能力。

- 公开策略并不等于可直接盈利，真实风险仍在滑点、手续费和极端行情里。
