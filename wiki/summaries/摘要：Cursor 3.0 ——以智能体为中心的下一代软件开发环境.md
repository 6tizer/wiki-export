**一句话摘要**：Cursor 3.0 推出以 Agent 为中心的统一工作区，支持多工作区并行、浏览器集成、智能体市场，但当前市场仗置将导致用户群体下降。

**关键洞察**

- **三大改变**：智能体窗口（侧边栏管理所有 Agent）、多工作区并行（跨 Repo 协同）、设计模式（浏览器内标注 UI 元素）

- **智能体市场**：内置 MCP、Skills 以及 Agents 插件，一键安装

- **不关闭 IDE 模式**：打开 Agent 窗口时，原始 IDE 保留并运行，两者相互独立

- **市场价即面临的压力**：功能设计保持第一梯队水平，但付费策略可能导致用户群体下除

- Cursor 3.0
