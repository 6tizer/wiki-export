**一句话摘要**：OpenAI 为 Codex 推出 Plugins 系统，通过 MCP 协议为 AI 编程 Agent 补上工具连接层，使其能接入外部服务与数据源。

**关键洞察**

- Codex Plugins 基于 MCP 协议，与现有工具生态兼容

- 工具连接层是 AI 编程 Agent 从「能写代码」到「能做事」的关键跨越

- 与 OpenClaw 的 Skill 生态形成竞争但也可能互通

- OpenAI 补上这块拼图后，Codex 成为更完整的 Agent 平台
