## 一句话摘要

OpenClaw 2026.2.17 版本引入 1M 上下文 + 对话内派生子 Agent，让个人 AI 助手第一次具备处理复杂长流程任务的底气。

## 关键洞察

- **子 Agent 派生 + 1M 上下文组合**：子 Agent 交接文档可无限长，长流程任务终于可行

- 其他重要更新：Sonnet 4.6 模型支持、iOS 分享扩展、Slack 原生流式输出、Telegram 样式内联按钮

- **1M 上下文 Bug**：OAuth 模式下可能不生效，需切换为 API Key 认证

- 项目数据：GitHub Stars 344,137，ClawHub 技能 13,729+

- 已知问题：Telegram 群组集成再次出现 bug；本地 LLM 模式速度极慢；API 费用每天可达 30 美元以上
