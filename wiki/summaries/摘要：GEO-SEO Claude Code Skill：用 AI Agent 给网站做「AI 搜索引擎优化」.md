### 一句话摘要

GEO-SEO Claude Code Skill 把生成式搜索优化和传统 SEO 审计打包成 Claude Code 可直接执行的一套网站优化工作流。

### 关键洞察

- 它反映出 GEO 已经从概念讨论走向可执行的工具链与服务化形态。

- 工具把 AI 引用率、结构化数据和品牌提及等 GEO 指标前置到审计流程中。

- llms.txt、品牌可见度和平台定向优化，说明生成式搜索优化与传统 SEO 的侧重点正在分化。

- 这类 Skill 也表明 Claude Code 正逐渐成为通用知识工作流的执行宿主，而不只是编程助手。
