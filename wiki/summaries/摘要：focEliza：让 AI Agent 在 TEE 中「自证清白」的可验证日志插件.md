**一句话摘要**：focEliza 是一个让 AI Agent 在可信执行环境（TEE）中生成可验证运行日志的开源插件，解决了 Agent 行为「无法自证清白」的信任问题。

**关键洞察**

- TEE（可信执行环境）确保日志不可篡改、可链上验证

- 解决 Agent 行为透明度问题，是去中心化 Agent 信任基础设施的重要组成

- 与 ERC-8183 的「评估者」机制形成互补

- ElizaOS 是当前重要的开源 Agent 框架之一

- 可验证日志是 Agent 商业化的合规基础

**提取的概念**：focEliza、TEE 可验证 Agent 日志
