## 一句话摘要

小米大模型负责人罗福莉深度解读 2026 年 AI 本质变化：从预训练主导的 Chatbot 时代彻底转向后训练主导的 Agent 时代，OpenClaw 是这一范式转换的标志性框架。

## 关键洞察

- **范式转换已发生**：AI 竞争从预训练主导的 Chat 时代转向后训练主导的 Agent 时代，预训练与后训练的算力投入比例从 3:1/5:1 变为至少 1:1

- **OpenClaw 的本质差异**：不是 Claude Code 的 UI 升级，而是通过厚重的框架编排弥补模型所有短板的端到端任务完成框架，具备分层分级持久 Memory 和自主多模型调度能力

- **研究效率指数级提升**：以前两周的 idea 验证流程，现在一小时内完成，Agent 框架 + 模型组合彻底改变了研究范式

- **1T+ 参数是 Agent 时代入场券**：中美在预训练上的代差基本消失，后训练和 Agent 框架才是真正战场

- **组织创新同样关键**：完全扁平、无 KPI 绑死、群体智能驱动的团队管理方式是小米 MiMo 团队的核心竞争力
