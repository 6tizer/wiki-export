**一句话摘衑**：Chops 是一款 macOS 应用，自动扫描并统一管理散落在各 AI Coding Agent（Claude Code、Cursor、Codex 等）中的 Skill 文件。

**关键洞察**

- 自动发现所有隐藏目录中的 Skill 文件，不管它藏在 `.claude/skills/` 还是 `.cursor/rules` 里

- 支持 7 个工具：Claude Code、Cursor、Codex、Windsurf、Copilot、Aider、Amp

- 内建编辑器支持 frontmatter 和 Cursor 的 .mdc 格式，Cmd+S 实时同步到原文件

- 支持 Collections 分组、FSEvents 文件监控、全文搜索、远程 skill 服务器连接（如 OpenClaw）

- 要求 macOS 15 Sequoia 以上
