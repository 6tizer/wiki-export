## 一句话摘要

本文分享了一个 GitHub 上 8K Star 的 GPT-Image-2 提示词开源仓库 awesome-gpt-image-2-prompts，涵盖电商广告、人像摄影、海报插画、角色设计和 UI 截图等多类高质量 prompt 模板。

## 关键洞察

- awesome-gpt-image-2-prompts 按垂直场景分类收录了大量可直接复用的提示词，每条 prompt 均配有生成效果示例图

- 提示词模板广泛使用 `{argument}` 参数化占位符（品牌名、标语、颜色等），支持一键替换变量批量出图

- 涵盖中英文双语 prompt，包括新中式美学、科普百科图、人物关系图等中文特色模板，适合中文创作者直接取用

- 电商广告类 prompt 已达到商业级水准，包含精确的构图描述、材质指定、排版指令和品牌署名

- 作者感叹 GPT-Image-2 生成能力之强，"现在真是不敢相信任何截图"——UI 截图仿真已达到以假乱真的程度
