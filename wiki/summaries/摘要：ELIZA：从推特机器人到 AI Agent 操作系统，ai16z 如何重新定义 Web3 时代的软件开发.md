**一句话摘要**：elizaOS（原名 ELIZA/ai16z）是用 TypeScript 编写的开源 AI Agent 框架，以模块化插件体系、多平台支持和区块链原生集成为核心竞争力，被誉为"Agent 领域的 Next.js"。

## 关键洞察

- GitHub Stars 约 17,900，曾登顶全球趋势榜，社区插件覆盖 DeFi、社交媒体等数百个场景

- 支持跨平台部署（Twitter/X、Discord、Telegram）及多 LLM 后端（OpenAI、Claude、本地模型）

- 内置 Solana、ETH 等区块链原生支持，Agent 可直接执行链上交易

- 与 Phala Network TEE 集成（`plugin-tee`），为链上 Agent 私钥管理提供硬件级信任保障

- 面临挑战：破坏性变更频繁（"每周追移动靶"）、与 X Corp 法律纠纷、品牌更名（ai16z → elizaOS）

## 提取的概念

- **ElizaOS** — 框架核心架构与实现

- **TEE 可信执行环境** — 与 Phala dstack 的硬件信任集成

- **Agent Swarms** — ai16z DAO 的多 Agent 协作生态愿景

- **Virtuals Protocol** — 同类框架横向对比
