## 一句话摘要

Codex App 已开始通过安装 Computer MCP 直接控制电脑，这标志着 Coding Agent 正在从“写代码”进一步走向“操作桌面”。

## 关键洞察

- 这条信息给出了非常直接的启用路径：下载 Codex App，在设置中开启“电脑使用”，安装 Computer MCP 后即可开始控制电脑。

- 回复区补充了当前落地限制：现阶段主要是 macOS 可用，部分地区登录和可用性仍有限制。

- 用户反馈已经把这项能力联想到电脑配置检测、网络配置修复等实际自动化场景，说明它不只是演示功能，而是有明确的生产力想象空间。

- 回复里出现的开源项目 open-computer-use，说明基于 MCP 的电脑控制能力正在快速模块化、可替代化。

## 个人评注

这类“Coding Agent + Computer Use”组合，对 Tizer 的启发在于：当内容整理、后台配置、跨应用搬运没有现成 API 时，可以把桌面操作封装成 MCP 技能接给 Agent 兜底。但一旦进入真实工作流，就必须同时设计权限边界、日志记录和人工确认点。
