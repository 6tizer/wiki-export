**一句话摘要**

这篇文章借 Sigrid Jin 一年消耗 250 亿 Token 的经历，展现了 Coding Agent 正从“模型使用”走向“编排系统竞争”，而 Claw Code 的净室重写则把这种能力推向了开源公共基础设施层。

**关键洞察**

- 高 Token 消耗本身不是重点，真正关键的是能否把模型调用沉淀为可复用的编排能力。

- AI Harness 代表了一种“让 AI 管 AI”的系统思路，核心在调度、验证和持续运行，而不是单次生成效果。

- Claude Code 源码泄露后，Sigrid Jin 团队通过净室重写快速做出 Claw Code，说明编排经验已经可以替代部分闭源产品壁垒。

- oh-my-codex 这类编排层正在成为 Coding Agent 生态里的新基础设施，决定了多模型、多代理协作效率。

- IRL 模式反映出一个新趋势：人在流程设计和边界约束中发挥作用，而不是持续盯盘式参与执行。
