**一句话摘要**：Claude 的 Computer Use 功能将 AI 助手从“只能动文件”升级为“可操作任意应用”，采用分层权限体系确保安全性，和 Dispatch 组合可实现手机遥控电脑全自动完成任务。

**关键洞察**

- Claude 能力进化：1 月 Cowork（文件系统读写）→ 3 月 Dispatch（远程派任务）→ 3/24 Computer Use（授几任何应用）

- 分层权限体系：主动开启 → 逐个授权 App → App 级别隔离（可设置 Full Control 或 View Only）

- 工作逻辑：优先利用 API 集成，只有没有 API 时才请求直接操作屏幕，操作时隐藏其他窗口

- **Dispatch + Computer Use 组合**：手机发一句指令 → Claude 在电脑上自动完成所有操作→你到了会议室活已完成

- 目前仅支持 macOS，Pro/Max 订阅用户
