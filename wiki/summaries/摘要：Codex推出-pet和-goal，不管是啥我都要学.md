## 一句话摘要

Codex CLI 新增 /pet（终端虚拟宠物）和 /goal（跨会话目标追踪）两个命令，前者是开发者体验彩蛋，后者是向委托式执行演进的实质功能。

## 关键洞察

- **/pet 引发社区两极反应**：部分开发者认为这是最佳 DX 决策，另一部分质疑 OpenAI 把资源花在了「AI版电子宠物」上，折射出开发者对 AI 工具定位的分歧

- **/goal 才是核心更新**：允许开发者设置长期目标，AI 助手跨对话轮次持续追踪进度，从对话式向委托式转变

- **Ralph Loop 的官方化**：/goal 被视为对社区 Ralph Loop 的重新诠释，将目标持久化从社区 hack 提升为一等功能

- **Pyright 作者 Eric Traut 主导开发**：/goal 目前仅 CLI 可用，需配置文件手动启用

- **Codex 追赶 Claude Code 的竞争态势**：文章认为 Anthropic 是当前「当红辣子鸡」，Codex 正全面复刻 Claude Code 能力，但进步速度很快
