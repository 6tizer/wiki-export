**一句话摘要**

TinyFish CLI + Skills 试图把“网页访问能力”从截图理解或零散 API 调用，收敛为可被 Agent 直接调用的终端执行层与 Skills 教学层。

## 关键洞察

- 这类产品的价值不只是“能上网”，而是让 Claude Code、Codex、OpenClaw、Cline 等终端型助手获得统一的网页执行接口。

- 用 Bash 驱动浏览器自动化，说明它更偏向低 token 成本、可脚本化和可嵌入工作流的执行路线。

- Skills 在这里承担“把工具教给 Agent”的作用，降低了能力接入和工作流复用门槛。

- 开源 Skills 仓库与 500 步免费额度，降低了开发者试用和社区扩散成本。
