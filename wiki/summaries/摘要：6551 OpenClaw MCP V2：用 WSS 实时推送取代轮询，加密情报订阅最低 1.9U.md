### 一句话摘要

6551 的 OpenClaw MCP V2 通过 WSS 实时推送替代轮询，把加密新闻和社交流的 Agent 接入从“能用但贵”推进到“低延迟且更省 token”。

### 关键洞察

- [OpenTwitter MCP](entities/OpenTwitter MCP.md) 和 [OpenNews MCP](entities/OpenNews MCP.md) 共同构成加密情报双路输入层。

- [WSS 实时推送](concepts/WSS 实时推送.md) 的核心价值是用服务端主动推送替代定时询问，从而显著降低延迟和 token 消耗。

- 订阅设计把实时情报能力从高门槛基础设施包装成可直接接入 OpenClaw 的产品。

- 这类工具的差异化不只是信息聚合，而是能否直接进入 Agent 工作流并触发后续动作。
