## 一句话摘要

利用海外服务器和 CLIProxyAPI 工具，可以零成本搭建私有 AI API 中转站，通过 Codex Team 账号实现超高性价比的 token 使用，并用 ccswitch 统一管理多个渠道的环境变量。

## 关键洞察

- **CLIProxyAPI**：开源代理服务器，为 CLI 提供 OpenAI/Gemini/Claude/Codex 兼容 API 接口，支持多账号统一管理

- **ccswitch**：可视化快速切换不同编程工具环境变量的开源工具，支持 win/mac/linux

- **百万上下文配置**：在 config.toml 中添加 `model_context_window = 1000000` 开启

- **中转站价值**：降低成本、绕过地区限制、统一多账号管理
