## 一句话摘要

这条讨论指出，围绕 Skillify、可复用技能、自进化与轻量 Harness 的 Agent 架构，正在被多个团队同时独立发现，说明 agentic engineering 正在沿着相似的系统约束快速收敛。

## 关键洞察

- Garry Tan 观察到，自己提出的 Skillify 想法与 Claude Code 内部已有的 bundled skill 非常相近，说明“把流程固化为技能”正在成为共识。

- 讨论区里多位实践者都把 skills、memory 与 harness 视为 Agent 的基础原语，而不是临时 prompt 技巧。

- “skills-as-primitives” 这一路线的核心价值，是把一次性的成功经验压缩成可复用、可调用、可维护的能力模块。

- 多人独立得到相似结论，更像是受限于上下文窗口、工具成本、失败恢复与长期运行需求后的架构收敛，而不只是相互模仿。
