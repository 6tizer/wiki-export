### 一句话摘要

agency-agents 把不同岗位的提示词、流程和交付标准封装成 Markdown 角色文件，让多 Agent 协作从框架开发变成可直接复用的组织模板。

### 关键洞察

- 它的核心创新不是底层技术，而是把组织结构代码化。

- Markdown 角色库显著降低了多 Agent 协作的上手门槛。

- 这种方案强在低成本与高可移植性，弱在流程传递仍高度依赖人工。

- 它非常适合独立开发者、小团队和内容生产场景做快速试验。
