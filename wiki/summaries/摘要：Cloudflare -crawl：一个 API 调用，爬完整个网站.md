## 一句话摘要

Cloudflare /crawl 把全站抓取、渲染和正文提取压缩成一个 API，使网站内容进入 LLM 工作流的门槛明显下降。

## 关键洞察

- 它把链接发现、队列、浏览器渲染和输出格式转换整合成了基础设施能力。

- Markdown 输出对 RAG 与摘要场景特别友好，能减少大量无关网页噪声。

- 合规能力被内建进抓取流程，说明 AI 时代的爬取正在从“脚本技巧”走向“平台治理”。
