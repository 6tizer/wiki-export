## 一句话摘要

YC 总裁 Garry Tan 将自己真实运行的生产级 AI Agent 记忆系统 **GBrain** 完整开源，可管理万级 Markdown 文件并实现全量召回。

## 关键洞察

- **规模验证**：GBrain 已在真实场景中管理 10,000+ Markdown 文件、3,000+ 人物档案、13 年日历数据及 5,800 条苹果笔记

- **框架集成**：专为 OpenClaw / Hermes Agent 体系设计，是该生态的官方记忆后端方案

- **开源可复用**：MIT 协议，任何人均可直接采用此配置构建个人 mini-AGI

- **来源权威**：作者本人（Garry Tan）发布并持续使用，不是演示项目
