**一句话摘要**：通过将 Apify 的确定性网页抓取能力与 Claude Code 的智能分析能力结合，实现零幻觉的结构化全网数据获取，彻底解决 AI Agent 的数据入口瓶颈。

**关键洞察**

- "确定性抓取"与"智能分析"必须彻底分离，不能全交给 LLM，否则大规模抓取时错误无法察觉

- Apify 积累了 20 年网页抓取能力，通过 Agent Skills 可以一句自然语言完成从抓取到分析的全流程

- 支持 Google Maps、YouTube、Instagram 等上千平台，且按结果计费，成本可控

- 通过 `.env` 配置 API Token 后，Claude Code 可直接调用 12 种内置 Skill（lead generation、competitor analysis 等）

- 本质是把 AI 从"概率搜索时代"推向"结构化知识管道时代"

**提取的概念**

- Apify Agent Skills（已有词条）

- 数据管道架构（确定性抓取 + LLM 分析的分离架构模式）
