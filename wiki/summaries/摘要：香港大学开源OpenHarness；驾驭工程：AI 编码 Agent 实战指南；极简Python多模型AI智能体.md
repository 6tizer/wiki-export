**一句话摘要**：香港大学开源 OpenHarness 轻量级代理框架，同期多个项目（CC Harness Skills、Nano Claude Code、OpenClaude）共同推动 Agent 开发工具生态发展。

**关键洞察**

- OpenHarness：轻量级开源代理框架，将 LLM 转化为功能性智能代理，支持多种工具和插件

- CC Harness Skills：可移植代理技能项目，解决 Agent 开发中的关键挑战

- 《驾驭工程》：深入分析 AI 编码 Agent 架构和实践的技术书籍

- OpenClaude：允许多种 LLM 集成到 Claude Code，提升灵活性

- Nano Claude Code：精简 Python 实现，支持多种闭源 API 模型和多代理工作流

**提取的概念**：OpenHarness、Harness 工程
