## 一句话摘要

这篇转述文把 Context Hub 描述为 Coding Agent 的实时文档中心和注释层，强调它对 API 过期、跨会话踩坑和群体学习问题的缓解作用。

## 关键洞察

- Context Hub 把文档鲜活度而非模型参数视为编程 Agent 正确性的关键变量。

- 注释层设计让 Agent 的踩坑经验能在后续任务中被复用。

- 社区维护文档也引入了投毒和供应链风险，需要额外的治理手段。

- 它的价值在于把“文档”从静态说明书变成动态协作层。
