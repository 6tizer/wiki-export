### 一句话摘要

这篇文章提出，游戏里的 Agent 更适合扮演 NPC 与系统角色来增强世界互动，而不该直接替玩家完成核心行为，否则会损害参与感。

### 关键洞察

- 自动化不等于更好玩，玩家的情绪价值来自自己做选择并承受结果。

- Agent 当 NPC 比 Agent 当玩家更容易创造张力、故事和互动空间。

- 多 Agent 社区模拟为未来更真实的游戏 NPC 提供了研究方向，但离成熟产品仍有距离。
