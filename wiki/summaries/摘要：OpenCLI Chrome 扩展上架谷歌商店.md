## 一句话摘要

OpenCLI 的 Chrome 浏览器扩展正式上架 Google Chrome Web Store，用户可直接从商店安装，无需手动加载，大幅降低使用门槛。

## 关键洞察

- OpenCLI Chrome 扩展（v1.0.2）已通过谷歌审核并上架 Chrome Web Store，审核周期近一个月

- 扩展功能定位为「Browser automation bridge」，通过本地 daemon 在隔离的 Chrome 窗口中执行命令

- 上架商店后用户不再需要手动安装/更新扩展，版本迭代更加便捷

- 社区反馈积极，已有约 2,000 用户安装
