## 一句话摘要

Anthropic 通过 Mythos、Project Glasswing、Claude Cowork 与 Claude Code 的连续实践，展示出一个新拐点：模型能力已经开始快于产品化速度，而真正拉开差距的是 skills、harness、本地执行与低成本原型实验共同构成的系统层能力。

## 关键洞察

- Mythos Preview 带来的不是常规增量，而是让工程团队明显感到“断层式跃迁”，尤其体现在漏洞发现、安全推理与代码理解上。

- 执行成本迅速下降后，Anthropic 内部已经可以并行尝试上百个产品原型，产品越来越像从原型池里筛选出来，而不是按传统排期逐步做出来。

- Claude Cowork 的核心价值不只是把 Claude 放进一个新界面，而是把虚拟机沙盒、权限边界、任务拆解、skills 文件与本地执行整合成面向知识工作的 agent 系统。

- skills 之所以意外有效，不在于技术复杂，而在于它把流程、偏好与经验写成可复用的文本规范，让模型像新同事一样被 onboarding。

- Anthropic 反复强调“真正成功的产品往往来自减法而不是加法”，说明未来 AI 产品竞争点会越来越偏向体验、信任与工作流贴合度，而不只是参数。

- 从受限开放的 Project Glasswing，到对本地优先和渐进授权的坚持，这家公司正在把高能力 agent 的发布问题，转化成产品治理与组织治理问题。
