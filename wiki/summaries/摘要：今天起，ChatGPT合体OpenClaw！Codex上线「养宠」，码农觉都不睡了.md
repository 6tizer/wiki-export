## 一句话摘要

OpenAI 官宣 ChatGPT 账号直接登录 OpenClaw，同时 Codex 密集更新推出电子宠物、/goal 自主迭代、一键迁移等功能，与 Anthropic Claude Code 正面竞争进入白热化阶段。

## 关键洞察

- **ChatGPT × OpenClaw 合体**：ChatGPT 账号可直接登录 OpenClaw，Plus 20 美元月费自带 Codex 额度，OpenAI 彻底打通了 Codex 与 OpenClaw 的生态闭环

- **Codex 电子宠物上线**：通过 `/pet` 命令唤醒像素风虚拟宠物，本质是游戏化的 Codex 状态指示器，灵感来自 Claude Code 泄露的 Buddy System

- **自主迭代能力突破**：`/goal` 命令实现目标驱动的自主编码，配合 Ralph Loop++ 实现智能自愈和连续改进，标志 Codex 从代码助手向全能 App 转型

- **一键迁移抢市场**：三步导入其他平台的插件、Agent、项目配置，直接对标 Anthropic 此前推出的一键搬家记忆功能

- **GPT-5.5 带动增长**：上线一周 API 增速超以往版本 2 倍，Codex 营收不到 7 天翻倍
