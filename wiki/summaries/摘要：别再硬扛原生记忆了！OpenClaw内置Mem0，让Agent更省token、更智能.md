**一句话摘要**：OpenClaw 引入 Mem0 插件优化记忆系统，通过 Markdown 组织知识、跨会话记忆管理，解决原生记忆插件高 token 消耗和信息筛选问题。

**关键洞察**

- Mem0 用 Markdown 结构化记忆，支持跨会话与跨 Agent 复用

- 相比原生记忆插件，显著降低 token 消耗

- 记忆可审计：Markdown 格式使记忆内容透明可检查

- 安装配置流程简化，适合快速接入

- 长期运行的智能助手场景下优势最为突出
