## 一句话摘要

Ghostty 之所以在 Claude Code 用户里爆红，不是因为它只是一个新终端，而是它把高频 AI 编程交互最在意的渲染速度、分屏体验和视觉连续性都做对了。

## 关键洞察

- GPU 渲染和原生界面体验，让长代码输出和频繁滚动明显更顺手。

- 对 Claude Code 用户来说，终端本身已经是生产界面，而不是底层附属工具。

- Kitty 图形协议让终端能承载更丰富的视觉输出，减少上下文跳转。

- 它的弱点也很真实，尤其在复制、搜索和部分 SSH 场景上仍不如老牌工具成熟。
