**一句话摘要**：ai16z DAO 于 2024 年底正式宣布 Agent Infra Ecosystem，以 Agent Swarms（智能体集群）为核心，设计了交易手续费回购 $ai16z 的代币飞轮，目标是打造开源 AGI 生态。

**关键洞察**

- Agent Swarms 定位：多个自主 Agent 协同工作，形成去中心化的「自主代理经济」

- 代币飞轮：平台手续费和 Agent 部署费用回购 $ai16z，形成「使用量→手续费→回购→价格」正向循环

- 2024 年 12 月 ElizaOS 拥有约 6,108 GitHub Stars、193 位贡献者，Discord 超 13,000 成员

- 与 Virtuals 异同：ElizaOS 更侧重开发者工具和开源基础设施，Virtuals 更偏向普通用户 Agent 发行

- 主要争议：与 a16z 品牌混淆问题促使 2025 年 1 月更名 elizaOS；前 30 个钱包持有约 43% 代币，治理中心化风险存在

**提取的概念**：ElizaOS、Agent Swarms、代币经济飞轮
