## 一句话摘要

从 OpenClaw 5705 个社区技能中筛选出 20 个最实用的 Skills，涵盖编程、生产力、网络自动化、安全等领域。

## 关键洞察

- **Skill 生态规模**：ClawHub 已有 5705 个社区技能，awesome 列表过滤掉了约 2748 个（含可疑/重复/高风险）

- **核心推荐**：coding-agent（编程代理）、cognitive-memory（多级记忆系统）、browse（浏览器自动化）、skill-vetting（安全检查）

- **Multi-Agent 协作**：通过 agent-council 创建研究员、分析师、写手、审核 Agent 团队

- **工作流自动化管线**：browse → get-tldr → essence-distiller → avatar-video-messages 实现全自动内容生产

- **安全优先**：装任何 Skill 前先用 skill-vetting 跑安全检查，敏感操作先在小号测试
