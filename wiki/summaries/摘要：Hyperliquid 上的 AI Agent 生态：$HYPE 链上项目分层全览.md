## 一句话摘要

2024 年底 Hyperliquid 生态出现了一批 AI Agent 项目，以 $VAPOR（AI Agent 发射台）和 $FARM（链上 AI 游戏）为头部，但整体仍以叙事驱动为主。

## 关键洞察

- Hyperliquid 对 AI Agent 的吸引力在于原生代币化能力 + 高性能订单簿 + DeFi 流动性，Agent 天然可调用链上交易 API

- Tier 1：$VAPOR（ai16z/ElizaOS 驱动的无代码 Agent 发射台，市值峰值约 $1.2 亿）、$FARM（链上 AI 生成游戏，市值约 $5200 万）

- Tier 2：AI Waifu 聊天机器人（$LQNA/$SCHIZO）和 AI Launchpad（$LAUNCH）

- Tier 3-4：小市值赌注，部分项目流动性极低，存在诈骗风险（$PEAR X 账号已封禁）

- 事后复盘：2025 年多数项目大幅回调，印证链上 AI Agent 代币高度依赖叙事周期
