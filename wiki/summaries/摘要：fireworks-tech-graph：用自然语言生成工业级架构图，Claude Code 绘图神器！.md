## 一句话摘要

fireworks-tech-graph 将自然语言描述、预设图表语义和多种视觉风格结合起来，让 Claude Code 用户无需手写 Mermaid 或手动拖拽 [draw.io](http://draw.io/)，就能快速生成工业级技术架构图。

## 关键洞察

- 它不是单一模板工具，而是覆盖架构图、数据流图、流程图、序列图、思维导图等 8 类技术图表的自然语言出图系统

- 产品设计明显面向 AI / Agent 场景，内置了 Agentic RAG、Mem0 记忆层、多 Agent 协作、Tool Call Flow 等常见结构模式

- 通过 5 种视觉风格与 SVG + PNG 双输出，它兼顾了 Wiki、README、演示文稿与内容发布等不同使用场景

- 相比 Mermaid 和 [draw.io](http://draw.io/)，它强调“描述系统即可出图”，把语法编写和手工排版的成本前移为语义理解能力
