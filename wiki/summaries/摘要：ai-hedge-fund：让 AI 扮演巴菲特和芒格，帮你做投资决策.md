**一句话摘要**：ai-hedge-fund 是基于 LangChain 和 LangGraph 的开源多 Agent 投资决策系统，让 AI 模拟巴菲特、芒格、Michael Burry 等投资人进行协作分析，定位是教育与研究工具，而非实盘系统。

## 关键洞察

- 核心思路是把不同投资风格 Agent 与分析型 Agent 组织成虚拟基金团队

- 典型实现覆盖价值投资、成长投资、做空与量化分析等多种角色，形成多 Agent 协作决策流程

- 技术栈以 LangChain 和 LangGraph 为核心，支持多模型、CLI、Web UI 与回测能力

- GitHub 热度很高，常被视为学习 LangGraph 多 Agent 架构的代表性实战项目

- 主要局限在于缺乏真正的风险管理能力，因此更适合研究、教学和架构参考
