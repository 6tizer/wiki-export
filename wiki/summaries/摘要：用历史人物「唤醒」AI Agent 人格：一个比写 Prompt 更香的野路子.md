### 一句话摘要

与其手写抽象人格描述，不如直接调用模型已经熟悉的真实历史人物，这是一种低成本激活 Agent 风格与思维模型的 Prompt 策略。

### 关键洞察

- 历史人物方案的优势在于调用已有语料表征，而不是从零定义人格。

- 这种方法适合快速搭建多角色 Agent 原型，但岗位边界仍要人工补全。

- Persona Pattern 是它的通用方法论底层。

- 与固定角色扮演相比，自生长人格路线更强调长期经验积累与可编辑身份文件。
