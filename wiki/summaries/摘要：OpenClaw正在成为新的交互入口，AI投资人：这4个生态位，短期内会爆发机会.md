## 一句话摘要

从 AI 投资人视角分析 OpenClaw 正在成为 AI 时代的 Linux，并预判通信、协作、交易市场和物理世界四个生态位将短期爆发。

## 关键洞察

- **OpenClaw 带动四层爆发式增长**：模型层（Kimi API 20 天收入超 2025 全年）、硬件载体（MacMini 二手价从 1700→3300 元）、Skill/Agent 数量指数级增长（Clawhub 5000→11232）、Infra 重构（Supabase 作为 Agent 默认数据库）。

- **OpenClaw 类比 Linux 的四波演进逻辑**：开源+社区驱动→低成本替代→生态闭环→成为不可替代的基础设施。核心框架 + Skill 插件的模式对应 Linux 内核 + 发行版。

- **四个短期生态位机会**：① 通信（AI-Native 邮件/消息系统替代传统工具）② 群聊和协作（Multi-Agent 组织架构问题）③ 交易市场（A2A 撮合非标服务）④ 物理世界（视觉/语音融合到 Agent 交互入口）。

- **现有 SaaS 工作流可能被 Agent 原子化**：部分功能被拆解到每个人的 OpenClaw 中，人与 AI 的管理关系开始反转。
