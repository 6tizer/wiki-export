## 一句话摘要

gstack 把 Claude Code 的使用方式从单次代码生成升级为多角色工程团队协作，通过斜杠命令和工序分离把 AI 编码流程工程化。

## 关键洞察

- 它的关键思想不是“更多工具”，而是把规划、审查、测试和发布拆成不同认知角色。

- 角色分工式 Agent 工作流能减少单个 Agent 同时扮演所有岗位时的认知混乱。

- 斜杠命令工作流把复杂流程压缩成可调用接口，降低了多角色协作门槛。

- 这类模式说明 Coding Agent 正在从对话式助手演变为流程化团队系统。
