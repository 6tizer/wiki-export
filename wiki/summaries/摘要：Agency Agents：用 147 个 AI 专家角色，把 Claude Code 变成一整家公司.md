## 一句话摘要

Agency Agents 把不同岗位的职责、流程与交付标准做成可复用 Markdown 角色文件，让 Claude Code 一类工具获得接近“虚拟团队分工”的工作方式。

## 关键洞察

- 它的创新点不在模型能力，而在角色化 prompt 的工程化组织方式。

- 通过 100+ 个专家角色，把开发、设计、营销和 QA 的分工显式化。

- 这类配置集合更像“角色定义层”，而不是独立的多 Agent 执行框架。

- 真正的使用价值往往来自挑出少量高频角色，而不是一次性启动整家公司。
