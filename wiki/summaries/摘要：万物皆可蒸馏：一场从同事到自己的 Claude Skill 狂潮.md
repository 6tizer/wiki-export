## 一句话摘要

蒸馏 Skill 热潮说明 Claude Code 的 Skill 机制已从“功能扩展”演化到“人格、关系和经验打包”的知识社会化阶段。

## 关键洞察

- Skill 已不只是工具说明书，而是可以承载人的风格、记忆和决策方式。

- colleague-skill、自己.skill、反蒸馏 共同构成了“蒸馏—复用—防御”的完整生态。

- 这波热潮的门槛不在代码，而在可供蒸馏的数据整理与隐私边界。

- 人格蒸馏让知识管理第一次和情感关系、劳动价值直接连接起来。
