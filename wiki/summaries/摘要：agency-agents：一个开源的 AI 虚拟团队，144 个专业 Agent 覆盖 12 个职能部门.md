## 一句话摘要

agency-agents 通过一整套可复用的角色文件，把“AI 虚拟团队”从抽象概念变成跨工具可安装的职能化 Agent 配置体系。

## 关键洞察

- 它强调的不是执行引擎，而是角色定义层和工作规范层。

- 一个高质量 Agent 不只是一个提示词，而是一套职责、流程、风格和交付标准。

- 这种做法天然适合 Claude Code、Cursor 等工具复用，也方便团队共享。

- 多角色 Agent 的组织方式，正在从零散 prompt 走向类组织结构的模块化配置。
