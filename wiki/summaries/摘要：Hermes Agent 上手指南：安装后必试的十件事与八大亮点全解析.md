## 一句话摘要

Hermes Agent 的核心竞争力不只是“会用工具”，而是把副驾模型、分层记忆、技能沉淀与安全授权组合成一个更适合长期使用的自进化 Agent 系统。

## 关键洞察

- Hermes 通过 Auxiliary 路由把压缩、搜索、审批等任务拆给副模型，降低了主模型成本。

- 三层记忆和 Session Search 让它更像长期搭档，而不是一次性对话机器人。

- Camofox、Docker 沙箱、多 Profile 等配置说明 Hermes 很强调真实执行环境而非纯聊天体验。

- 技能系统会把高价值操作自动沉淀成 Skill，这让使用越久越有复利。
