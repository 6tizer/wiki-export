## 一句话摘要

一篇精选清单，推荐了 12 个最能提升 Claude Code 开发效率的 GitHub 仓库，覆盖持久记忆、UI/UX 设计、工作流集成、RAG 检索、Skills 集合、Agent 蓝图等方向。

## 关键洞察

- **持久记忆是刚需**：claude-mem（68K Stars）通过 Hook + RAG 实现跨会话记忆注入，解决 AI 每次重新学习代码库的痛点

- **UI/UX 设计自动化已成熟**：UI UX Pro Max（71K Stars）提供 67 种风格、161 套配色、99 条 UX 规则，让 AI 不再生成丑陋界面

- **MCP 协议打通工作流自动化**：n8n-MCP（18K Stars）将 Claude Code 连接到 1,505 个 n8n 自动化节点，实现 400+ 集成

- **Graph + Vector 混合 RAG 增强代码理解**：LightRAG 让 AI 结构化理解大型代码库

- **AI 编程需要版本控制升级**：h5i 作为 Git 副车，记录 AI 推理轨迹和决策上下文，支持跨 Agent 会话接力

- **Skills 生态爆发**：从 Antigravity（1,200+ Skills）到 Awesome Claude Code（社区圣经），Claude Code 的 Skills/Plugin 生态已形成规模
