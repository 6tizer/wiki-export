**一句话摘要**：Anthropic 封杀 OpenClaw 等第三方平台，龙虾之父对此不满，指出 OpenClaw 高强度使用推高了 API 费用，用户开始退订。

**关键洞察**

- Anthropic 宣布封杀 OpenClaw 等第三方平台，用户只能使用 Claude 官方服务

- 原因：第三方工具导致系统负担过重

- 龙虾之父对此表示不满，指出高强度使用导致 API 费用攀升

- 用户在社交媒体上开始退订，体现对此政策的强烈不满

- 此事件折射出 AI 平台方与第三方生态之间的根本利益冲突
