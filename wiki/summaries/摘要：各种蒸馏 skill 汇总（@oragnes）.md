## 一句话摘要

[@oragnes](https://x.com/oragnes) 整理了 36 个基于 Claude Code `.skill` 系统的人格蒸馏仓库，覆盖自我蒸馏、职场关系、亲密关系、公众人物四大类别，折射出 2026 年初 Claude Code skill 生态的爆发式增长。

## 关键洞察

- **colleague-skill 是整个生态的鼻祖**：12681 Stars，数十个衍生项目在 README 中注明"Inspired by colleague-skill"，是事实上的社区标准

- **女娲.skill 是元生成器**：6623 Stars，可以蒸馏任何人的认知操作系统，已批量产出芒格、乔布斯、费曼、马斯克等公众人物 skill

- **反蒸馏的出现揭示了生态矛盾**：anti-distill（1700 Stars）专门用于清洗被迫写出的 skill 文件，核心知识产权保护诉求浮现

- **数字永生赛道已有系统级框架**：immortal-skill 支持 12+ 平台数据导入，七维数字分身，对齐 OpenClaw Soul Spec 标准

- **用于逝者的情感向 skill 正在兴起**：reunion-skill、MamaSkill 等针对已逝亲人的 skill 代表了一个高度情感化的应用细分
