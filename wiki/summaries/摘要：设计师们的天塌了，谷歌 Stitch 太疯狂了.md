## 一句话摘要

谷歌 Stitch 2.0 通过「Vibe Design（氛围设计）」理念，让用户用自然语言直接生成高保真 UI 和可运行代码，消除了设计工具的学习曲线护城河，Figma 股价在历史高点跌去80%+。

## 关键洞察

- **Vibe Design 替代传统拖拽设计**：说一句话就能出高保真 UI + React 代码，拉平学习曲线

- **SaaS 护城河被颠覆**：Figma 的护城河一直是「学习曲线」，Stitch 完全免费（Google 账号即可），按生成次数限制

- **上下文感知设计**：Smarter Design Agent 有长时记忆，理解整个项目历史，生成组件风格一致

- [**DESIGN.md**](http://design.md/)** 标准化**：设计规范以 Markdown 定义，打通设计与工程

- **语音驱动设计**：Voice Vibe 可以边说边改界面，实时反馈

- **设计师角色转变**：从「工具熟练度」转向「驾驭 Agent 的产品导演」
