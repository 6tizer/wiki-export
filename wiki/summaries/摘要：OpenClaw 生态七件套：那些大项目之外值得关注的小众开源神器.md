## 一句话摘要

文章盘点了 OpenClaw 主项目之外的七个生态分支，说明社区正在从“一个 Agent 框架”扩展到轻量运行时、记忆系统、信息过滤和安全治理的完整外延。

## 关键洞察

- ZeroClaw 和 PicoClaw 分别代表 Rust 重写与嵌入式轻量化两条性能路线。

- OpenFang 把 Agent 从应用层工具抬升到内核级系统思路，野心明显大于普通插件项目。

- ClawFeed 与 lossless-claw 对准了两个高频痛点：信息过载和长对话失忆。

- opentwitter-mcp 与 opennews-mcp 展示了 OpenClaw 生态对外部实时信息接入的迫切需求。

- SlowMist 安全指南说明高权限 Agent 的真正瓶颈不是功能，而是可控性。
