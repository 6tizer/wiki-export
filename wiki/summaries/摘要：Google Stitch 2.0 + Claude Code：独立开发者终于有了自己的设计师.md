**一句话摘要**

这篇文章展示了一条完整链路：先用 Google Stitch 生成设计系统，再通过 [DESIGN.md](http://design.md/) 和 MCP 让 Claude Code 持续按同一风格完成界面实现。

## 关键洞察

- Stitch 的真正价值不只是生成一版 UI，而是输出可持续复用的设计规则。

- [DESIGN.md](http://design.md/) 把“设计一致性”从临时提示变成了项目级长期约束。

- MCP 让编码 Agent 不再靠猜测还原设计，而是直接读取设计语境。

- 对独立开发者来说，这套链路把 UI 设计成本明显压缩到可接受范围。
