## 一句话摘要

小米 MiMo 团队发布 MiMo-V2.5-Pro（1.02T MoE）和 MiMo-V2.5（310B 多模态），在通用 Agent 三大核心榜反超 DeepSeek V4 Pro 夺得开源 SOTA，同时以更低 token 消耗（省 40%-60%）挑战闭源旗舰，并配套百万亿 Token 创造者激励计划 MiMo Orbit。

## 关键洞察

- MiMo-V2.5-Pro 在 GDPVal-AA（1581）、τ³-bench（72.9）、ClawEval（pass^3 63.8）三项通用 Agent 基准均获开源第一，但与 GPT-5.4、Claude Opus 4.6 仍有明确差距

- Coding 维度未碾压：SWE-bench Pro 不及 Kimi K2.6 和 GLM 5.1，SWE-bench Verified 不及 DeepSeek V4 Pro，开源 Coding 赛道竞争激烈

- Token 效率是核心差异化卖点：ClawEval 64% Pass^3 仅需约 70K tokens/轨迹，比闭源旗舰省 40%-60%，对企业长程 Agent 任务成本控制意义重大

- 架构亮点：70 层 MoE（384 路由专家/层，激活 8 个）、SWA+GA 6:1 混合注意力、3 层 MTP 模块；后训练采用 SFT → Agent RL → MOPD 三段式

- 生态配套齐全：SGLang/vLLM Day-0 适配、多芯片厂商支持、MIT 协议全面开源、Orbit 百万亿 Token 激励池
