### 一句话摘要

OpenAI 正把 Codex 从 AI 编程工具升级为承载 Computer Use、插件生态、图像生成、记忆与长程自动化的个人开发 Agent 平台，并将其作为 Super App 战略的产品化入口。

### 关键洞察

- Codex 一次性补齐 **Computer Use**、内置浏览器、`gpt-image-1.5`、90+ 个插件、记忆、自动化六块能力，明显从单点编程助手转向全周期开发伙伴。

- OpenAI 把 **Computer Use** 直接整合进 Codex，而不是做成单独模式，说明桌面控制能力被视为默认执行层，而非实验性附加功能。

- 插件体系被明确拆成 skills、app integrations 和 MCP servers 三层，意味着 Codex 正在把跨工具编排能力平台化。

- 记忆与自动化不再是孤立功能，重点变化在于自动化任务可以沿用既有线程继续执行，这让长期任务运营成为现实。

- 从 OpenAI 的表述和产品路径看，Codex 正在承担个人 Agent 与 Super App 的主入口角色，战略意义已经超过单纯的 Coding Agent。
