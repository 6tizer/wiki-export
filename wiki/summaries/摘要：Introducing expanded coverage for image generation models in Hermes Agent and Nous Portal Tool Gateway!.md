## 一句话摘要

Hermes Agent 现已在 Nous Tool Gateway 中扩展接入多款图像生成模型，用户更新客户端后即可通过 Fal 通道快速切换 ChatGPT Image、Nano Banana、z-Image、FLUX 等出图能力。

## 关键洞察

- 这次更新把 Hermes 的图像生成能力从单一入口升级为**多模型可切换**的统一调用体验。

- 交互路径被压缩为 `hermes update` → `hermes tools` → 选择 Fal → 选择 image model，降低了接入门槛。

- Nous 正在把 Tool Gateway 打造成 Agent 的**能力分发层**，而不只是简单的模型列表。

- 对内容生产型工作流来说，这意味着同一个 Agent 可以按成本、速度和风格在不同图像模型之间切换。
