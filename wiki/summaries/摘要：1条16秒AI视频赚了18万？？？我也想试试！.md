**一句话摘要**：Pexo 是一个为 AI Agent 原生设计的 AI 视频生成 Skill，用户只需一句话即可在聊天窗口内完成视频创作，支持接入 OpenClaw、Claude Code 等各种 Agent 环境。

**关键洞察**

- Pexo 天然面向 Agent 设计：核心交互只有对话框，几乎没有 GUI，适合作为 Skill 被 Agent 调用

- 通过 ClawHub 或 GitHub 安装 Skill 后，龙虾/Claude Code 可自然语言驱动完整视频生成流程（分镜→选模型→渲染→合成）

- Pexo 本身不是视频模型厂商，而是整合了 Seedance、Sora、Kling 等 SOTA 级模型，自动判断最适合的模型

- 支持人物一致性：提取真实照片特征生成卡通形象，在不同场景保持角色统一

- 飞书、WhatsApp、Telegram、Discord 都支持——在哪聊天就在哪做视频
