## 一句话摘要

Obsidian-agent-client 插件让用户在 Obsidian 中无缝切换 Claude Code、Codex、Gemini CLI 等多种 AI Agent，实现笔记与 AI 的深度融合。

## 关键洞察

- **多 Agent 自由切换**：一键切换 Claude Code、Codex、Gemini CLI、OpenCode、Qwen Code 等，无需重复安装插件

- **多会话并行**：支持开启多个 AI 对话窗口同时进行，互不干扰

- **笔记联动**：支持 @提及笔记，AI 直接读取笔记内容进行创作和分析

- **安装方式**：通过 BRAT 插件安装，需配置 Node.js 路径和 claude-code-acp 依赖

- **智能会话搜索**：支持按仓库筛选和隐藏无关会话
