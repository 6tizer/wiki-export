## 一句话摘要

2025 年初，五大 AI Agent 开源框架（Swarms、ElizaOS、ARC/Rig、ZerePy、Dolos）同时涌现，各自定位不同赛道，尝试成为下一代 AI Agent 的基础设施。

## 关键洞察

- **多 Agent 编排**：Swarms 是五个框架中最「企业级」的，支持多 Agent 并行协作和 Agent 间支付（ATP 协议），GitHub Stars 5500+

- **跨平台部署**：ElizaOS 用 TypeScript 编写，支持一键部署到 X、Telegram、Discord 等平台，曾登上 GitHub 全球趋势榜首

- **性能极客**：ARC/Rig 用 Rust 编写，主打内存安全和低延迟，适合 Solana 高频交易 Agent

- **社交媒体垂直**：ZerePy 由 Zerebro 后端剥离而来，专注 Twitter/X 内容自动化发布

- **无代码尖兵**：Dolos/DOLION 最另类，结合 Meme 属性和代币燃烧机制，技术深度有限
