## 一句话摘要

geo-seo-claude 把 GEO 审计流程做成 Claude Code 技能包，帮助网站从“传统 SEO 优化”转向“提高被 AI 搜索系统引用的概率”。

## 关键洞察

- GEO 的目标不再是搜索结果排名，而是 AI 回答中的曝光与引用。

- 工具把 AI 爬虫检查、品牌曝光扫描、llms.txt 和内容可引用性评估整合到同一工作流里。

- 它代表了 Claude Code 这类 Coding Agent 正在从写代码扩展到营销与内容诊断。

- 真正的价值不在一次性评分，而在持续发现结构化内容资产的缺口。
