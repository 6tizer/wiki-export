## 一句话摘要

Lovart 正在把品牌手册解析、字体生成、批量出图、视频生成、Skill 沉淀与 PSD 交付串成一条完整链路，使一人公司也能以较低成本搭建可复用的品牌内容生产系统。

## 关键洞察

- **Brand Kit 是核心枢纽**：品牌手册 PDF 不只是被读取为 Logo 和配色，还会被解析成可挂载到项目中的风格约束，帮助不同物料维持统一调性。

- **AI 设计开始从单次出图走向流程化生产**：从造字、海报到视频，Lovart 把多个创意环节串起来，降低了反复描述与反复返工的成本。

- **Skill 机制把经验沉淀成资产**：一次成功的对话流程可以被总结为模板，下一次只替换品牌信息就能复用，接近 Claude Code 中“先跑通再抽象”为 skill 的工作方式。

- **PSD 分层导出补齐了交付短板**：AI 先完成初稿，设计师再进入 Photoshop 做精修，使 AI 工具真正接入专业设计流程。

- **Lovart 的差异化在于 Agent 式工作方式**：它不是直接按 prompt 出图，而是先采集参考、理解品牌视觉 DNA，再组织生成任务。
