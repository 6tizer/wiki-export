## 一句话摘要

Anthropic 用 Project Deal 证明，Agent 已经可以在真实办公室场景中独立完成二手物品的挂牌、议价与成交，而更强模型会把能力差距直接转化为价格优势。

## 关键洞察

- Anthropic 让 69 个 Claude agent 代表员工在 Slack 中完成二手交易，真实交割轮次共促成 186 笔交易、总价值 4010 美元。

- Opus 4.5 相比 Haiku 4.5 不仅更容易促成成交，还能在相同物品上卖出更高价格、买入时付出更低价格。

- 参与者主观上未必能明显感知模型差距，但差距会持续积累在每一次报价、还价和机会发现里。

- 把提示词写得更强硬，并不会稳定提升交易结果，真正决定胜负的仍然是模型能力本身。

- 这类实验说明 Agent 交易市场具备可行性，但真实商业化仍受支付、物流、风控、虚构信息与平台规则等边界约束。
