### 一句话摘要

飞书正在把自身的数据、权限和协同体系整体升级成 Agent 原生平台，aily、妙搭和多维表格 Agent 分别对应工作助手、开发 Agent 与分析入口。

### 关键洞察

- aily 的关键优势是原生权限继承，而不是单点模型能力。

- 妙搭从低代码工具升级为 Code Agent，说明企业应用搭建也在被 Agent 化。

- 多维表格 Agent 让数据分析从配置式操作变成自然语言交互。

- 飞书真正的竞争力在于上下文、工具与权限三者已经天然存在于同一平台中。
