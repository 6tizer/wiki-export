**一句话摘要**：探险李整理的链上 AI Agent 学习资源全景，核心框架是 elizaOS（GitHub ~18k Stars）和 Virtuals Protocol（"AI Agent 的纳斯达克"），判断项目基本面需从技术/代币经济/社区三维度综合评估。

## 关键洞察

- AI Agent 四大能力：感知、规划、记忆、行动——真正的"自主行动"是与普通 LLM 调用的本质区别

- elizaOS 最新进展：Shaw 的 V2 更注重模块化，代币已从 AI16Z 迁移为 ELIZAOS

- Virtuals 野心：Agent Commerce Protocol（ACP）让 Agent 成为原生 DeFi 用户，达到"AI Agent 市场"ᄽ指

- 投资判断三维度：技术基本面（真实运行 Agent）+ 代币经济（可持续价值捕获）+ 社区（真实用户留存）

- 核心差异：通用框架（elizaOS）vs 特定场景 Agent 应用

## 提取的概念

- **ElizaOS** — 当前最活跃的链上 Agent 开源框架

- **Virtuals Protocol** — Agent 代币化与商业化平台
