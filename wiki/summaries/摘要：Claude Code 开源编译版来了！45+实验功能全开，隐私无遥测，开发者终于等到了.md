**一句话摘要**："free-code" 是去除遥测与安全限制、解锁 45 个实验功能的 Claude Code 开源编译版，一条命令安装即可使用。

**关键洞察**

- 去除了 Claude Code 官方的遥测模块和安全限制，保护用户隐私

- 解锁 45+ 实验功能，包括多智能体协作、深度思考模式、语音输入

- 项目代码已永久存储在 IPFS，确保不可删除的持久可用性

- 只需一条命令安装，门槛极低，适合开发者快速上手

- 是 Claude Code 源码泄露后社区二次创作的直接产物

**提取的概念**：free-code、Vibe Coding
