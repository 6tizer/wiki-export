**一句话摘要**：五款主流 AI 记忆开源框架（Mem0、Zep/Graphiti、Letta/MemGPT、A-MEM、MemOS）各有战场，选对架构构建真正「有脑子」的 Agent。

**关键洞察**

- **Mem0**：最成熟生产级选择，48000 stars，LongMemEval 49%，公共 API 三行接入，适合个性化推荐/客服 Agent

- **Zep/Graphiti**：时序知识图谱，记录事实有效期和变动历史，LongMemEval 63.8%，适合金融、医疗、CRM 类场景

- **Letta/MemGPT**：OS 级分层内存管理，Agent 自己调用工具读写记忆，原生支持多 Agent 协作

- **A-MEM**：NeurIPS 2025 论文级别，卡片笔记法构建记忆关联网络，适合知识关联推理

- **MemOS**：三态记忆统一调度（文本+激活+参数），系统级未来方向，当前仍处于研究阶段
