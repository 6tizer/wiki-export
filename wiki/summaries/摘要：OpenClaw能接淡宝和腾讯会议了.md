**一句话摘要**：国内天王级平台（腾讯会议、淡宝、高德地图）已开通官方 MCP 接口， OpenClaw 可通过安装对应 MCP Server 直接调用，无需 API Key。

**关键洞察**

- **腾讯会议**：安装 MCP 包配置 Token 后可用，支持创建/修改/取消会议、查日程、获取会议纪要

- **淡宝**：装淡宝 PC 客户端，MCP Server 自动在本机运行，支持搜商品、下单、查订单、催发货——**无需 API Key**

- 高德地图早已开放，支持路线规划、POI 搜索、天气查询等 12 类接口

- MCP 协议正在成为国内天王级平台的标准接口，这对 AI Agent 解锁实世界应用有极大意义
