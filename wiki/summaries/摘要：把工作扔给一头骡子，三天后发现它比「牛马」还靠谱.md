**一句话摘要**：MuleRun 是一款云端自进化个人 AI，通过分配独立虚拟机（7×24小时在线）和"个体学习+群体共享"双飞轮，解决 OpenClaw 对普通用户门槛过高的问题。

**关键洞察**

- OpenClaw 的核心问题不是不够强，而是它的"强"普通用户用不上（安装复杂、依赖本地设备、电脑关了就停）

- MuleRun 架构：基座 Agent + Knowledge + Skills + Runtime 四层，每用户分配独立云端虚拟机，物理隔离

- 两种核心模式：Computer（持续在线、定时任务）vs Super Agent（一次性交付型任务）

- Heartbeat 机制：每天主动汇报完成情况，发现重复操作时主动建议自动化

- 自进化双飞轮：个体层越用越懂你，群体层用户经验汇入公共知识网络，越多人用越强

**提取的概念**

- MuleRun（云端自进化个人 AI Agent 平台）

- Heartbeat 机制（Agent 主动汇报与建议机制）
