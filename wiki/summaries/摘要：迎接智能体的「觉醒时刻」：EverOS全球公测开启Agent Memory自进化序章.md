## 一句话摘要

EverOS 试图把 Agent 的长期记忆从外挂式检索能力升级为原生记忆底座，让智能体能够从交互中沉淀技能、持续进化，并在复杂任务中显著提升成功率。

## 关键洞察

- Agent 记忆之所以成为行业焦点，本质上是上下文窗口、推理成本与长期行为一致性三重瓶颈同时暴露。

- EverOS 的核心机制不是简单存日志，而是把对话经验提炼为 Agent Case，再通过语义聚类和技能蒸馏形成可复用 SOP。

- 文章强调的 mRAG 混合检索架构，说明长期记忆系统必须同时处理文本、图像、文档与结构化信息，而不是只做向量召回。

- HyperMem 代表其底层记忆组织方式正从扁平向量库走向超图结构，目标是支持跨时间、多实体、多跳关联推理。

- EvoAgentBench 的结果表明，记忆与技能进化可以成为比单纯堆参数更高杠杆的能力增强路径，尤其在软件工程类任务中效果最明显。
