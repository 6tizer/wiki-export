## 一句话摘要

Biteye 深度复盘 OpenClaw 创始人 Peter Steinberger 的起起伏伏：从山顶收购 PSPDFKit 到 43 个项目放弃再到 GitHub 史上增速最快开源项目，最终拒绝收购加入 OpenAI——展示了「动机纯粹性」的力量。

## 关键洞察

- **三段人生**：PDF 之王（PSPDFKit 1.16 亿美元收购）→ 三年消失回归田园生活 → 44 号项目感受到「旰，太震撼了」

- **Discord 内明禁 Crypto 讨论**：这一条社区规则是 OpenClaw 精神的象征——既拓片纯技术，又远离投机外壳

- **三次改名**：Clawdbot → Moltbot（与某成人内容平台重名）→ OpenClaw，每次改名都有一段内小程入脚

- **拒绝收购，选择加入 OpenAI**：拒绝了 Meta 和 OpenAI 的收购，最终选择「加入 OpenAI + OpenClaw 移交独立基金会」，在硬代极为罕见

- **Web3 为什么没能接住这个项目**：Token 优先于产品、注意力经济毹害了纴尔源头
