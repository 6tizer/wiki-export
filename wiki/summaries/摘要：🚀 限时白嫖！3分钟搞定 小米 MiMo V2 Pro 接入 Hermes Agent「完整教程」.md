## 一句话摘要

这篇教程展示了如何通过 Hermes Agent 的模型配置流程接入 Nous Portal，并在限时免费活动中启用小米 MiMo V2 Pro 作为实际运行模型。

## 关键洞察

- 核心入口不是手动配置 API Key，而是在终端运行 `hermes model`，沿着 Hermes 内置流程完成模型切换。

- 接入链路分为三步：选择 Nous Portal、订阅 Free 套餐、批准 Hermes Agent 登录授权。

- 免费套餐虽然标价为 $0/mo，但仍需要先绑定支付方式，这也是用户评论区最关心的门槛之一。

- 完成授权后可以直接选择 `xiaomi/mimo-v2-pro`；若需要多模态能力，可手动填写 `xiaomi/mimo-v2-omni`。

- 这类限时接入教程的价值不在模型原理，而在于把“能不能用、怎么最快用起来”的路径压缩到可执行步骤。
