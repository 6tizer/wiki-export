## 一句话摘要

这篇文章把 Anthropic 近期围绕 Claude Mythos、Managed Agents 与 Advisor 体系的更新串成一条主线，认为 Anthropic 正从单纯卖模型转向通过安全叙事与托管式 Agent 基础设施收紧生态控制。

## 关键洞察

- Mythos 在文中不仅是模型发布，更被视为 Anthropic 争夺定价权与安全话语权的信号

- Claude Managed Agents 与 Vaults 把 Harness、凭证管理、长时运行等能力上收为官方平台能力

- Advisor Strategy 与 Advisor Tool 体现出“便宜模型执行，强模型按需顾问”的原生协作范式

- OpenClaw 等第三方 Agent 工具在配额、接入与系统提示词层面的限制，反映出平台边界正在收紧
