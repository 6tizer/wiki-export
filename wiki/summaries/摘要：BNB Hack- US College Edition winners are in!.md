## 一句话摘要

BNB Hack 美国高校版黑客松公布三个获奖项目——链上信用评分（Credence）、AI Agent 技能市场（The Dojo）和 AI SaaS 财务管理（Messier），展示了链上 Agent 经济与 DeFi 信用基础设施的前沿探索方向。

## 关键洞察

- **链上信用评分突破超额抵押**：Credence 将 FICO 评分方法论搬到链上，结合钱包行为数据与 ZK 验证的链下信用凭证，在连续曲线上动态调整抵押率，为欠抵押 DeFi 借贷开路

- **Agent 技能市场范式成型**：The Dojo（基于 Maiat Protocol）实现了 Agent 技能注册、链上质量评分、自动退款和 BAS 存证，让 Agent 通过市场发现互补能力并按次付费

- **x402 协议从理论走向应用**：Messier 将 x402 微支付协议用于 SaaS 订阅管理和 AI Agent API 按次计费，验证了 HTTP 402 + 链上结算的实际可行性

- **ERC-8004 / ERC-8183 标准复用**：The Dojo 使用 ERC-8004（身份注册）和 ERC-8183（托管会话）构建 Agent 商务协议栈，与此前 x402 生态形成技术复用

- **三个项目共享 $10K 奖金池**，并获得 BNB Miami 全额赞助参会资格

## 个人评注

三个获奖项目分别对应 Agent 经济基础设施的不同层面：身份与信用（Credence）、能力发现与交易（The Dojo）、支付与财务管理（Messier）。The Dojo 的 Agent 技能市场模型与 OpenClaw 的 Agent 协作方向有交叉——如果 Agent 能在市场上自主发现、评估并购买其他 Agent 的能力，将极大降低多 Agent 系统的编排复杂度。Messier 对 x402 的实际落地也进一步验证了 Agent 自主支付能力的可行性。
