**一句话摘要**：香港大学数据科学实验室开源 CLI-Anything，一条命令将任意软件转化为 AI Agent 原生工具，告别脆弱的 UI 自动化。

**关键洞察**

- 通过一条命令自动生成完整 CLI 接口，支持真实软件操作

- 彻底绕过 UI 自动化（如 Playwright）的不稳定性问题

- 目标：让所有有代码库的软件成为 Agent 原生工具

- 计划支持更多应用类别并与其他 Agent 框架集成

- 是 A2A 通信标准化的重要基础设施

**提取的概念**：CLI-Anything
