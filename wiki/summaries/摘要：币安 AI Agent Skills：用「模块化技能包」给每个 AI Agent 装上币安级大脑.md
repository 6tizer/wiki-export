## 一句话摘要

币安通过开源的 Skills Hub 将自己的市场数据和交易能力模块化，让任意 AI Agent 都能就少几行配置完成变币策略研究到交易执行的全链路。

## 关键洞察

- 首批 7 个技能覆盖：现货行情查询、下单执行、代币详情、叙事追踪、地址洞察、合约审计、交易信号

- **叙事发酵 → 地址监控 → 合约体检 → 现货交易**：全链路操作无需切换工具

- **币安的战略定位层级转变**：将自己定位为 AI Agent 基础设施层，而非仅仅是交易界面

- OKX 在币安发布后 4 小时内跟进上线，币安 Skills Hub 显示员已有开源工具基于全部 7 个 Skills API 构建第三方应用

- **风险提示**：API Key 需设置 IP 白名单、禁用提现权限；主网交易前先在 Testnet 跑通全流程
